BridgeBase #163 Article

　

本周的牌是由香港牌手Sam Wan表演的。Sam是香港多年来水平最高的牌手之一，他曾经两次作为香港队的队员参加世界奥林匹克桥牌队式赛。Sam是一位极富创造力的叫牌者和技巧高超的庄家。本周的这副牌并非特别引人入胜，但Sam的技术和牌感令人印象深刻。 
 

Sam手持： 

        S 532 
        H Q75 
        D J 
        C Q98543 

西家开叫1S，Sam的同伴作技术性加倍，东跳叫4S，Sam的同伴再次加倍4S，在他们的约定中，这要求同伴如果是均型牌就可以罚放，如果是非均型牌就应该叫牌。Sam选择叫5C，西家加倍结束了叫牌。5CX是最后的定约。 

West    North    East    South 
1S      X        4S      pass 
pass    X        pass    5C 
X       pass     pass    pass 

西家首攻SA，明手摊下如下一手牌： 

        S 10 
        H A10862 
        D AKQ4 
        C KJ10 

        S 532 
        H Q75 
        D J 
        C Q98543 

初看起来，定约的前景很好，只有黑心和CA两个输张。 

西家SA得到第一墩后，换攻D10，Sam用手中的DJ得到。Sam需要处理他手里的4个高花输张，通常的做法是用明手王吃一个黑心，用DAKQ垫掉其他三个。Sam接着王吃了一个黑心，并且兑现DA，垫掉另一个黑心，这一轮D上，西家跟D5。 

Sam从明手出CK调将，东家示缺，西家也忍让了这一墩。Sam知道，现在定约正是关键时刻：通过叫牌知道西家有5张黑心，再加上现在看到的4张草花，如果西是单张H，三张D的话，必须要求西是单张HK；如果西家红心缺门，4张方板，那么现在就必须兑现DAKQ才能完成定约；但Sam的牌感告诉他，西家一定是双张方板，要打他持有HKJ双张才能完成定约！ 

而事实正是如此！ 

四手牌如下： 

        S 10 
        H A10862 
        D AKQ4 
        C KJ10 

S AK974           S QJ86 
H KJ              H 943 
D 105             D 987632 
C A762            C -- 

        S 532 
        H Q75 
        D J 
        C Q98543 

Sam从明手继续出CJ，暗手用CQ盖过，西CA吃。西还回HK（顽强！）！明手上HA，Sam出小红心，暗手上HQ，满意的看到西家跌落了HJ，依靠明手的H10完成了这个困难的定约。 

Sam打的棒！感谢你提供给我们来自香港的视角和声音，香港是我们曾经去过的最迷人的城市。
