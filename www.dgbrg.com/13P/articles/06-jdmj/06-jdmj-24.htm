输张垫赢张

――选自纽约时报桥牌专栏


给初学者上的第一课应该就是介绍怎样在赢张上面垫输张。对于中级牌手，就要合理解释怎样在输张上面垫输张。但是为什么要在输张上面垫赢张？这种状况是怎样出现的呢？

这副牌来自本月纽约麦城Tristate地区的公开对式双人赛。南北方是两位种子选手，Mark Feldman和Bill Pollack。在这副牌里，他们的叫牌进程很是不爽。

双方无局，叫牌进程如下：

北       东        南         西       
1H       PASS      1NT        PASS
2C       PASS      3D         PASS
3NT      PASS      PASS       PASS

西首攻SK。

放在以前，南会用2D应叫1H，这种应叫新花表明有邀叫实力。在现代叫牌体系中，2D的应叫是应该保证有进局实力的，所以南不得不应叫1NT。这个逼叫真让北家大伤脑筋。因为他不能够再叫2S，那是表明强牌的逆叫。他又不能再叫2H，那将表明有6张H。所以他叫了2C，表明低限三张套。如果PD是1-4-4-4牌型，那么可以停在3-4配合的部分定约上。

北家应该PASS了，但是他跳叫3D邀叫，南家根据自己的低限牌力，根据防守方可能首攻S，而自己持有S10这个关键牌张判断，应叫3NT将是最合适的叫品。

西首攻SK，对于庄家来说，确实是个利好信息。他可以用SA吃进，而S套依然有坚固的防御能力。后面的进程颇为乐观，当南家向明手引DK时，西犯了一个致命的错误，那就是在南显示D套长而坚固的时候用DA盖吃。

西转攻CA，东用C9表示欢迎，西继续打C10，被明手CK盖吃，现在残局如下：

S：Q986                             S：73

                   C：J5 

南已经可以数到9个赢墩了，但他还想超墩，多得分。如果这时兑现D套赢墩，那么HJ得分的机会就很渺茫了。于是他引出S套的大牌SJ，垫掉一张C，这是标准的输张垫输张，这样做非常危险，如果西有三张C，那他将宕得很惨。

南因此尝试了一个惊人之举----输张垫赢张。他引出SJ，并垫上D2。这是一个用D套赢墩换取S套赢墩，这时他仍然只有9墩。这对西来说，真是一个考验。西输了，输在回引S9上。

回引S9看起来似乎很安全，其实不然。南用明手S10吃进，回小H到HK，连续兑现D套赢张。这就出现了著名的双挤：西要保留S止张，东要保留C的止张，没人能够保护H了。南的超墩终于如愿以偿。

如果西思路更加清晰，他可以在用SQ赢进之后回H，或者回D9投入。这都将有效阻挠Pollack的紧逼计划的实施。
