大小信号


大小信号，也称回声信号或欢迎信号。大小信号大概是防守人积极进行战斗所持有的武器中最为重要的一个武器。大小信号的常规的和经常性的用法，是在一个花色中作出大小信号时，就表示希望同伴继续攻出这个花色，或者希望以后同伴获得出牌权时再出这个花色。例如： 

       S Q85          
       H KQ82          
       D Q2          
       C 7643                 
S AK942        S 103       
H 103          H 64       
D 87           D A109653       
C J1093        C Q82       
       S J76          
       H AJ975        
       D KJ4          
       C AK          

南做4H定约，西首攻SK，明手跟出5，东跟出10，南扔出6。西接出SA，这时东跟出3。西看到东作出了大小信号，盼望继续出S，于是西再攻出第三轮S，东将吃。之后东又兑现DA，成为击败定约的宕墩。

在大小信号一开始是打出一张不寻常的大牌，例如上面一副牌例中的S10时，同伴当然是容易识别的。但是，在一般的情况下，一个牌手的双张中常常没有不寻常的大牌，这时只得被迫用一张小牌来开始作出大小信号。然而在这个花色的第二轮时，接着打出一张更小的牌，大小信号就会清楚无误了。下面就是一个例子，这是一次比赛中的牌例： 

       S K10765          
       H 52          
       D A84          
       C Q104                 
S 2          S 93       
H Q76        H J10984       
D J1092      D K763       
C AK875      C 32       
       S AQJ84          
       H AK3        
       D A5          
       C J94          

南做4S定约，西首攻CK，东跟出3，南暗手跟J。

西如果缺乏观察力，就会换攻DJ，从而让南得以做成定约。但是这里的西观察到C2还未出现。或者这张C2是在庄家暗手中，或者东是用C3开始作大小信号。因此西接着再打出CA，这时东跟出未出现的这张C2。于是东自然就将吃第三轮C，4S定约终于被击败。

从东的角度考虑，如果他的C例如说是82，当然要好得多，因为这样就可以用8开始作出一个很明显的大小信号。但是用32作出的大小信号也达到了同样的目的，这就要归功于西的敏锐的观察力。关于别的信号方法，参见单双数垫牌法（ODD－EVEN DISCARDS），倒大小信号（UPSIDE－DOWN SIGNAL）。

不幸的是，大小信号也和所有其他的约定方法一样，常常被随意乱用和误用，只是因为他是一个“传统的”的方法就随便作出。牌手常常忘记作出大小信号要同伴继续出一个花色，必须是防守人有一定的目的。下面一副牌例就是错用大小信号的一个例子。 

       S 53          
       H AK98          
       D Q1073          
       C A94                 
S 4          S J1098       
H J752       H Q103       
D AK94       D 82       
C 7652       C K1083       
       S AKQ762          
       H 64        
       D J65          
       C QJ          

南做4S定约，西首攻DK，东机械地跟出8。于是西接出DA，东再跟出2、之后西接出第三轮D，东将吃。从现在开始，庄家已经没有任何困难。他先叫完将牌，然后用明手的已打好的DQ垫掉暗手一张C，完成定约。

在西首攻DK，东应当跟出不鼓励的2，而不可跟出8。东将吃第三轮，难道会得到什么好处吗？甚么好处也没有，因为他本来就持有一个不会消失的将牌赢墩。假如东先跟出D2，西毫无疑问会换攻C。这样东的CK就会得到一个赢墩，庄家的定约就会失败。

还有另外一类约定性的情况，在这类情况下的大小信号并不是表示希望同伴继续攻出这个花色，而是向同伴显示这个花色自己所持的牌张数为偶数。这种约定方法在长度信号（LENGTH SIGNALS）这个条目中有所讨论和举例说明。这里只是顺便简略地予以介绍。

在防守方必须采用缓拿打法时，这种约定信号就是既简单又最为重要。这时这种大小信号的应用原理如下：

在明显看到庄意图打好明手的一个长套花色时（这时明手已别无其他花色的进张），第二家的同伴（或第二家自己）就会发生何时用自己的A得进才适合的问题，这时第二家（或第二家的同伴）便可以用大小信号来表示自己持有2或4张；第二家如果在这个花色持有3张牌（例如742），就可以在第一轮时跟出最小的一张牌（2），在第二轮时接着跟出次大的一张牌（4）。在后一情况下，同伴就会知道作信号的人在这个花色中准确持有3张牌，因为如果他持有2或4张牌，他是会作出大小信号的。下面一副牌例是这种大小信号实际应用的一个例子。 

        S 42          
        H 953          
        D KQJ95          
        C 842                 
S J9863       S K107       
H K108        H Q76       
D 76          D A82       
C J107        C 9653       
        S AQ5          
        H AJ42        
        D 1043          
        C AKQ          

南做3NT定约，西首攻S6，东盖K，庄家用A得进。接着庄家打出D10，西跟出7，北和东都跟出小D。庄家又打出D4，西跟出6，明手D9赢得，因为东正确地让掉了这墩D。现在庄家就肯定宕一墩，因为已无法做好并兑现明手的D。

西作出的大小信号（7，6）告诉自己持有2或4张D。但是在南打出第二轮D时，西已不可能持有4张D，否则这副牌中就有14张D。因此东知道西原先准确持有2张D，从而知道庄家在打出第二轮D之后，手中还有一张D。
