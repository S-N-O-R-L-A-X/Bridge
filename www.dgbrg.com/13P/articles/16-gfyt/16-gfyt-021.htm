迷人之局


原著：Anders Wirgren  翻译：玉龙狐(Guxw1234)  转自：桥牌世界

2000年春，在美国俄亥俄州的辛辛那提市，范德比尔特(Vanderbilt)杯的决赛在世界上最成功的两支桥牌队之间展开，一支是尼科尔队(Nick Nickell-Dick Freeman,Bob Hamman-Paul Soloway, Jeff Meckstroth-Eric Rodwell)，另一支是施瓦兹队(Richie Schwartz-Drew Casen, Bobby Levin-Steve Weinstein, Zia Mahmood-Michael Rosenberg)。两强相遇，打得难解难分，当比赛还剩16副时，施瓦兹队仍以113-88领先，但尼科尔队最终以161-143反败为胜。这是其中一副令人着迷的牌。

让我们来到开室，这里齐亚坐北，罗森博格坐南；哈曼坐东，索罗威坐西。南家罗森博格手持：

     S Q97
     H QJ9
     D A972
     C 1095

局况是双有，齐亚开叫1C (南北以弱无将为叫牌架构)，哈曼Pass，你如何应叫？

当然，你会在1D和1NT之间选择。这两个叫品都不是十全十美，但对同伴来说，描述性的，限制性的叫品往往比非限制性的令他舒服，而且，既然我对打D毫无兴趣，我会投1NT一票。在同伴开叫1C，其它花色可便宜地显示，1NT确保是一手平均型的牌。传统上，应叫1NT的牌要求比应叫1D的强一些，大至上为好的8个点到差的11点。齐亚与罗森博格这对使用弱无将开叫，因此齐亚开叫1C可能是15个大牌点以上的强无将类型牌，1NT应叫会使错误的一手牌坐庄吗？

也许会，也许不会。如果西家从J-8-x-x或J-10-x-x中首引D，而明手的是Q-x，或者西家首攻，而明手的是K-x，你将后悔叫了1NT；另一方面，如果西家的H是K-10-4而明手是3张带A，你会喜欢由西家引牌。而且，纵然你在S上有些有用的中间张，如果北家的S是3张带A或2张带A，你还是希望由西家而不是东家引这门花色。但还有一些有利于1NT叫品的论据，是什么？

即使这副牌属于敌方，应叫1NT可能关煞他们。对于1D，西家可轻易争叫1H或1S， 但他的牌可能较弱，不至于在2阶争叫，这对你方十分有利。桌上，罗森博格可不这么想，可能受他的同伴经常手持4-4低花时开叫1C的影响，他应叫1D。

西家是索罗威，他加倍；齐亚加叫2D，保证4张支持；现在轮到哈曼叫牌。让我们看看你手持哈曼的牌会做些什么：

     S J8643
     H K72
     D 104
     C 643

没有很多点，但有5张S和HK。你认为这足以参与叫牌吗？

我希望你会参与叫牌。即使是IMP制，为部分定约争斗依然非常重要。如果你方可打成2S或3S，与此同时，对方可打成2D或3D，想想看，本来可得+110分或+140分，结果却-90分或-110分，那意味着损失5或6个IMP。而且比赛中存在大量的部分定约，如果你不为此竭尽全力地斗争必然损失严重。即使你不能完成定约，也可能把敌方推到一个不能完成的定约。有所得总比有所失强。

所以哈曼叫了2S；现在又轮到罗森博格叫牌。

你会再参与叫牌吗？

不。你的确有9个大牌点，而且你方显示出平衡的实力，但你的平均牌型和在敌方花色上的实力是继续叫牌的警告信号。而且，如果你方的3D能打成，有同伴在那儿呢。如果他的牌合适，他会再叫一次。

但不是那么会事--索罗威加叫至4S，接下来两家Pass，你有什么要说吗？

没有。你的防御力量较好，看上去可能打宕4S，但由于敌方自愿叫到这个定约，你不能指望超过宕一。因而，加倍所得甚少，如果“加倍”向庄家提供了打成定约的线索，那就得不偿失了。所以，Pass最好。

叫牌过程：
西       北      东      南
Soloway  Zia     Hamman  Rosenberg 
         1C      Pass    1D
Dble     2D      2S      Pass 
4S       Pass    Pass    Pass 

    S Q97
    H QJ9
    D A972
    C 1095 

你首攻什么？

一张C。从3或4张小牌中向同伴最长花色引牌通常是很好的选择，你没有理由另辟蹊径。也许最有趣的问题是你引C10还是C5？这总是争论不休的话题，我投C5一票，这使同伴得到正确的张数信息。 这手牌明手是不会有3张的，所以积极首攻C10试图穿过明手的C K-x-x是一厢情意的想法。在首攻下，即使伟大的哈曼充其量只能得到9墩，整副牌是：
发牌：北；局况：双有
         S 5
         H 83
         D Q865
         C AKQ872
S AK102           S J8643
H A10654          H K72
D KJ3             D 104
C J               C 643
         S Q97
         H QJ9
         D A972
         C 1095

如你所见，(首攻C后)要么防守方打3轮C，缩减明手将牌后得到1墩将牌，要么得到2墩C(庄家猜对了将牌)；加上H、D各1墩，或者加上2墩D，宕一！但罗森博格没有首攻C，而是从DA之下低引一张小D，你对此有何想法？

这安全吗？如果明手摊出3或4张小D，或单张D，而庄家有K，你方唾手可得的4墩就会莫明其妙地变成3墩。为何在你方防守前景一片光明的时候冒不必要的风险呢？----这不是呼唤英雄的那种牌。好在明手摊下的牌中D组合正是罗森博格所期望的，哈曼明手放小----谁都会这样----这使得首攻看上去很美。

也许不。

在继续阅读之前，猜猜看这个首攻造成下述哪种结果：

　　1、防守方获得成功
　　2、庄家获得成功
　　3、不清楚

你一定猜防守方获胜，对吗？是的----噢，不！我们已经知道首攻并再攻C将使定约直截了当地宕1，如果首攻DA再换攻C也异曲同工。但是现在哈曼能够树立1墩D垫掉1个H失张，抱DA攻小不过是转移了防守方的赢张。相反，它为防守方指出了一条歧路。在DQ得到同伴的首攻后，北家齐亚兑现了CQ，此时罗森博格不得不表态。四明手看显然应鼓励齐亚续攻草花，但罗森博格看不到四明手，他看到的是潜在的危险，你只看南家和明手的牌能发现吗？ 

如果庄家哈曼2张H、4张C而不是现在这样的各3张，北家齐亚不得不攻到罗森博格的DA，否则庄家有机会完成定约----哈曼可将吃第2轮C，兑现1个将牌顶张，连打3轮H，第3轮手中将吃，吊将到明手顶张，用树立好的垫掉方块失张，轻而易举地赢到10墩----无需猜对将牌！

所以，罗森博格的首攻为自己制造了一个想象中的难题。第1墩真是奇特无比！实战中罗森博格果真打信号表示不欢迎Q，齐亚转攻方块到他的方块A，罗森博格换出C，明手将吃，兑现SA，现在哈曼有机会完成定约。

如果防守方所剩的2张将牌全在南家，为完成定约，要求南家有3张或者QJ双张。于是哈曼选择打敌方将牌均分，他兑现了SK，你会这样打吗？

我希望不会。为什么不在吊第2轮将牌前兑现A和K？当罗森博格在2轮上跟牌之后，你应该试着回答以下两个问题： 

　　1、齐亚手持如下面的牌，在他们约定的12-14大牌点的弱无将体系中，会开叫1NT吗？ 
    S Q5 H 83 D Q865 C AKQ72

　　2、罗森博格手持如下面的牌，在1C 或2S 后，他曾经支持过C吗？
    S 97 H QJ9 D A972 C 10985

如果你对上述两个问题都回答：“不”，你就打SK；否则，飞牌。你是否怀疑：我尝试对这些问题回答“是”。如果你按实际牌型打对了，事实上你期望防守方已经犯了错误。要知道防守是桥牌中最难的部分之一，即使是齐亚和罗森博格如此优秀的牌手，在防守时也绝非十全十美。所以，如果有证据表明防守方犯了错误，请抓住。
