两套效应 （关于牌题九的探讨）
　

队式赛制，双有局，东开叫
东第一家不叫，你坐南第二家开叫1D，西做技术性加倍，同伴应叫1S，东再次不叫，你持有以下的牌，该如何应叫？
S: Q 6 5
H: 3
D: A K Q 8 7 6
C: K 10 4

此时，如果南家PASS就显得有些离谱，北家一定以为南家会再叫的。南家可以基于方块套的长度再叫2D，也可以支持同伴的黑桃而再叫2S，可以基于方块套的坚固程度和黑桃的支持跳叫3D，基于红心单张且黑桃配合方块强套跳叫3H，以及干脆跳叫3S强烈支持同伴黑桃进局。在分析上述各种叫品之前，我们尝试着分析南家目前可以从叫牌进程中获取什么信息，以及能够得到什么有意的推断。

首先，西家在同伴PASS、南家开叫的情况下，于有局方做了技术性加倍。由于是队式赛，西家这样叫牌很可能会遭到北家再加倍的反击，因此西家应当有相当可靠的大牌点力和合适的牌型。西家在这种情况下的技术性加倍，不太可能象双人赛中，甚至象队式赛中局况有利时所做的那种很轻的加倍，否则南北家采用再加倍开始反击，最后惩罚东西家任何低阶定约宕三，对东西家都是致命的。据此推断，西家至少应当有13个大牌点和合适的牌型，如4-4-2-3、3-4-2-4和4-4-1-4。

北家在西家的技术性加倍（暗示可能有4张黑桃的情况下）仍然“强调”他手里黑桃套牌的价值。从北家的角度来看，南家很可能只有2张小黑桃。北家显然渴望南家在防御东西家可能的部分局定约时能够首攻黑桃，基于这个判断，北家的黑桃套一定是A领头的5张以上长套，黑桃套的强度和长度一定不劣于A J 10 x x或A J x x x x。因为此时是队式赛，所以北家的牌力一定不会象双人赛那样弱，也比队式赛中无局方持牌要强一些，他至少应当有8个有效的大牌点。那么北家的牌力范围就是8-13点，我们可以分两个牌力范围讨论北家的牌：低限范围（8点至差一些的10点）和高限范围（好一些的10点至13点）。

如果北家持有高限牌力，在牌型合适的情况下，他可以采用再加倍来反击西家。以下几副牌例就是北家可以再加倍的牌（牌例一至牌例四，且持有刚才申明的黑桃套）：

牌例一：
S：A J 10 x x
H：Q J x x
D：x
C：Q x x

牌例二：
S：A J 10 x x
H：Q J x
D：x x
C：Q x x

牌例三：
S：A J 10 x x
H：K J x x
D：x
C：Q x x

牌例四：
S：A J 10 x x
H：K J x
D：x x
C：Q x x

总之，北家如果再加倍，表示只有方块是短套（不配合），高限牌力且在其他花色上有分散的大牌，适合防守。现在北家没有再加倍，南家可以排除北家持有上述牌例所示一手牌的可能性。因此北家持牌情况就比较明显了：要么他持有低限牌力；要么他持有高限牌力但不适合防守的牌型，如北家可能持有5-5-2-1的牌型，或更加畸形的牌；亦或北家持有高限牌力但红心梅花中有一门花色绝无防守牌力也未可知。

如果北家持有低限牌力，他一定会考虑到万一南家黑桃单缺怎么办。除非北家的黑桃足够长，否则他方块一定有些支持，要么是3张小牌，要么是J x，甚至J x x都很可能。以下几副牌例就是北家持有低限牌力应叫1S时可能持有的牌（牌例五至牌例八）：

牌例五：
S：A J 10 x x
H：K x x
D：J x x
C：9 x

牌例六：
S：A J 10 x x x
H：x x
D：J x
C：Q x

牌例七：
S：A J 10 x x
H：x x x
D：J x
C：Q J x

牌例八：
S：A J 10 x x
H：x x x
D：J x x
C：Q x

从以上牌例我们可以明显看出，如果北家持有牌例五所示的牌，南北家很可能完成3NT定约；如果北家持有牌例六所示的牌，南北家很可能完成4S定约；如果北家持有牌例七和牌例八所示的牌，南北家很可能完成5D定约。因此现在问题很清楚，南北家离成局定约不远了。从得失比的情况看，在队式赛中，有局方处于成局边沿时，应当较无局方更加进取，所以南北家目前必须考虑的首要问题是选择什么样的成局定约。

南家的一手牌有三个主要特征：其一，坚固的方块6张套；其二，红心单张；其三，黑桃较好的支持。按照什么顺序向北家表示这些特征、如果不必要向北家表示所有特征，那么表示南家持牌特征中的哪些，就是南家目前面临的实际问题。下面我们结合分析南家目前可选叫品的利与弊，来回答上述问题。

2D：表示了南家手中方块套的长度，希望东西方竞叫2H，南家就可以扣叫3H来进一步表示手中持牌的强度。但是按照目前的情况看，东西家几乎不会再参与竞叫，而除非北家持有高限牌力，否则他会以为南家对黑桃不配合，从而PASS南家的2D叫品。

2S：表示了南家手中的黑桃支持。除非北家持有6张黑桃且有10点有效牌力，否则他不会再叫3S进行邀请。

3D：逼叫一轮，北家知道南家持有方块坚固长套。由于北家不知道南家有黑桃支持，所以他首先会评估南北方3NT定约的可行性。基于这一考虑，如果他持有如牌例五所示的一手牌时，会叫3H以示旁门档张所在，而南家将顺势叫3NT；如果北家没有红心档张而不能叫3H时，他如果持有如牌例六所示的一手牌时，会叫3S，南家则加叫4S；或者北家持有如牌例七和牌例八所示的一手牌时，会叫4D，南家则再叫4S让北家在4S和5D之间做最后的选择。

3H：这是mini-splinter，表示红心单张，黑桃高度配合，满贯试探。但是南家红心并非缺门，黑桃并非高度配合（至少需要Q x x x），梅花也没有A，所以同伴可能会产生满贯意图也不奇怪。如果北家持有如牌例五所示的一手牌时，4S定约可能要宕。

3S：与3H一样，忽略了南北方3NT和5D定约的任何可能性，但是他可以避免同伴的头脑过热。

当然，如果北家持有如下牌例所示的一手牌时，任何成局定约将都是灾难：

牌例九：
S：A J x x x x
H：Q x x
D：x x
C：Q x
