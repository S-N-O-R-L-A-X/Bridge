周末的意大利式宵夜


前两天讨论的牌例其实都不算难，只需稍加思索，便可避免犯错，也许有朋友会说，哈，宰相肚皮低估了我们的智慧，干吗不来点儿高难度的！今天是周末，给各位送上一份味道浓郁的意大利式宵夜怎么样？

故事发生在四十五年前，意大利蓝队在纽约击败了美国队，首次赢得世界冠军称号。这场比赛一开始争夺就很激烈，直至齐亚瑞迪主打了以下这付满贯后，两队之间的胜负才成定局。

北 S.QJ7 ............ 双方有局
.. H.A943 ........... 叫牌过程如下：
.. D.932 ............ 南 ...... 西 ...... 北 ...... 东
.. C.865 ............ 齐亚瑞迪 .......... 达阿里略
.. .................. 2C ...... Pass .... 2H ...... Pass
南 S.AK985 .......... 2S ...... Pass .... 3S ...... Pass
.. S.K8 ............. 4D ...... Pass .... 4S ...... Pass
.. D.AQ ............. 5C ...... Pass .... 5S ...... Pass
.. C.KQ102 .......... 6S ...... Pass .... Pass .... Pass

齐亚瑞迪和达阿里略使用逼叫的2C约定性开叫。2H应叫显示有HA，2S是自然叫，3S表示至少3张支持，带有一张大牌。4D和5C是扣叫，建议上满贯。但达阿里略由于没有余力，两次拒绝了同伴的邀请。

齐亚瑞迪的最后决定显得有点冒失，不过平心而论，达阿里略所持的牌力相对于他的叫品来说，确实是最低的了，要是他持有以Q带头的4张将牌，或者有5张H，或者甚至还有一张CJ，也不过是作同样的叫牌。

当齐亚瑞迪叫到6S时，现场评论员指出：这个满贯是绝对完不成的；即使齐亚瑞迪停在5S上，定约能否完成仍是个问题。

全副牌如下，西首攻HQ，庄家该如何打？

.. H.QJ10 .............. H.7652

.......... 南 S.AK985
............. S.K8

齐亚瑞迪用K赢得首攻，出S9到Q，再出一张C到K，然后出S8到J。S10于这一轮跌出，这种有远见的打法得到了报酬――庄家获得了以将牌7转入明手的一个额外进张。

当齐亚瑞迪从明手出第二张C时，东扑出A。要是他回打C，就能给西一个将吃。但由于西在跟出C时先出3再出4，东以为他的同伴有3张C而不是两张。

如果东回一张C，满贯就立刻被打宕；但如果他回一张D，齐亚瑞迪将有可能完成定约――如果他拒绝飞K而用A赢进；如果他以S7进入明手；如果他用C10飞牌；最后，如果他兑现所有赢张。这样，将在D和H上对西家实现紧逼。另一方面，如果东回打一张H，庄家将很难找到致胜路线……

东确实回了一张D，齐亚瑞迪似乎能透视牌张，续出如下：DA，S5到明手7，C到手中的10，SA和CQ。残局如下：

当SK打出时，西已无可避免地在H和D上受挤，从而让齐亚瑞迪漂亮地完成了他的满贯。

怎么样，够精彩吧？明晚奉请各位不要太早上床睡觉，还有精彩牌例，也是齐亚瑞迪的杰作。
