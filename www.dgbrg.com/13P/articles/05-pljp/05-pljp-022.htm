水乳交融看防守


当你在桥牌桌上坐定时，第一件要紧的事并不是洗牌发牌，更不是叫牌打牌，而是要牢牢记住：坐在你对面的那位便是你在这个战役中生死与共的伙伴，你俩肩负着共同的命运，能否和谐无间水乳交融，是决定牌局胜负的最基本因素。“Partnership”这个词在字典上也许只简单地注释为“合伙”，而在桥牌桌上，除了作为搭档之外，还蕴含着互信互谅同心协力等等更多意义，请看以下牌例：

.......... 北 3
............. 10952
............. K10862
............. KQ5
西 Q87 ............ 东 AJ10952
.. QJ6 ............... 874
.. 754 ............... J
.. 10764 ............. A93
.......... 南 K65
............. AK3
............. AQ93
............. J82

双方有局，东开叫，叫牌过程如下：

东 .... 南 .... 西 .... 北
1 ... 1NT ... Pass .. 3NT
Pass .. Pass .. Pass

这副牌例出现在一次重要的复式赛，开闭室两桌都以同样的叫法达到3NT，北家认为，手中的五张方块和坚强的梅花已经能够支持同伴正常争叫的1NT，因此，虽然敌方所叫的黑心十分脆弱，仍然跳叫成局，希望同伴一旦控制住敌方的首攻之后即可连下九城。

两桌的西家都同样首攻7，东家都同样跟9，庄家都同样以K吃住。接着两桌的庄家都同样连出方块长龙，然而不同的情况发生了：

闭室西家垫了两张梅花，庄家续打K，东家A取，考虑再三，耽心Q如在庄家手中，那么再出黑心，会让庄家白拣一墩，为稳妥计，回出红心，庄家大难不死，立即拿足十墩，防守方相对无言后悔不已。

开室则不然，西家仔细地看到东家的9已把庄家的K顶出，显然J10都在同伴的手中，为免除他的疑虑，于是在庄家的第四轮方块上立即垫掉手中的Q（精彩表演！），于是，哪怕东家再糊涂也肯定不会忘记在A上手之后让黑心套一咕脑儿出笼了。

当然，西家在同伴1开叫之后，自己持Q87配合和可能的H进手，首攻Q以协助同伴树立长套也是很好的选择。

此处西家的垫牌简单明确，不但免去了同伴的疑惑，增强了相互的信心，更有13imp的可观进账，因此，精诚合作或顾忌猜疑所造成不同的得失结果，就如人生启示一般，每每若是也！
