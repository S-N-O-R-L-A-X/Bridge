牌桌上的碰碰车


今天讲一个牌桌上的富有戏剧性的故事。

假如使掌握在对方手中的两张大牌在一墩中同时出现，可以称之为“撞车战术”，这种情况可以把必失的两墩牌减少为一墩，由于对方绝不会自动撞车（除非战略上有需要打通桥路之类的绝对必要才会故意扔掉大牌），因此这种战术基本上只有庄家才用得上，至于如何能够使对方大牌撞到一起，那就要看庄家心思慎密的火候了。

双方有局，南家开叫，叫牌过程如下：

南 .... 西 .... 北 .... 东
1 ... Pass .. 3 ... pass
4 ... Pass .. pass .. pass

西家首攻：J

叫牌过程是必然的，合约看起来也相当安全，但是由于西家的首攻击中了庄家的要害，于是乎风云变色，差点陷庄家于绝望的境地。

如果对方首攻不是J，庄家可以从容不迫地先逼出对方A，而后将自己的一张小方块垫在明手的梅花上，而如今对这要命的J，使得庄家只好用明手的Q盖上，东家以K再盖，自在意想之中，庄家不得不以A取得这一墩。

情况十分严峻，眼看必失两墩将牌以及一墩方块和一墩梅花不可，但是，在对方手中的将牌只有三张，其中包括A和K，那么，是否还有一线生机呢？

生机还是有的，假如东家手中是两张将牌而且其中带一张大牌，并且红心是Qxx的话，那就可能是个机会，既然如此，赶紧抓住这渺茫的希望吧。

第二墩牌庄家以K到明手，出10，手中J飞，果然赢得，前途已微露曙光，接着拔A，明手垫小方块，东家的Q束手就擒。希望更大了一点，接着再出9，明手继续垫去方块，东家以小将牌将吃。此时防守方无论打什么都没关系了，庄家再度上手之后开始吊将，果然东西两家的A和K双双现身，大将牌撞车，完成了庄家这个的构想。

这是想象力和撞车术的有机结合和运用，只有经验丰富处变不惊的庄家才能创造这样的戏剧性效果。
