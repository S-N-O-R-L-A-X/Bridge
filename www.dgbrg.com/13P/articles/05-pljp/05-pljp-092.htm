飞牌的故事


――周麒

旅居纽约时，John张经常邀我一起去打双人赛。张是我的一位朋友，年轻聪明，进步很快，但偶尔也会有我不理解的前卫想法。下面这副牌发生在1995年5月纽约昂纳桥牌俱乐部的一次双人赛中，全副牌是：

        S A1095
        H AKJ
        D 9753
        C 64
S 763            S --
H Q106           H 98432
D KQ4            D J108
C KQ75           C J10983
        S KQJ842
        H 75
        D A62
        C A2

南  西  北  东
1S  －  2C  －
2S  －  4S  －
4NT －  5H  －
6S  －  －  ＝

拿南家的牌，我开叫了1S。北家应叫2C，我虽有6张好黑桃，但大牌点少了一些，只叫了2S。北家一下子就跳了4S。按叫牌过程来讲，叫牌就该结束了。但是，也许是由于我未能跳叫3S，有些耿耿于怀，我多考虑了一下，按我们之间的叫牌约定，2C是5张梅花，跳叫4S至少有开叫实力。我认为我手中牌不错，2个A很好，6张黑桃套也不错。我的叫牌一向保守，不知道为什么这次冒进起来，我叫4NT问A，同伴回答2个A，现在，我义无反顾的叫6S。

西家首攻，他把牌面向下。北家把牌亮出，东西方是一对犹太兄弟，颇为精明的样子。看到明手只有两张草花，西家召请了裁判。裁判问我打的是什么叫牌体制，我回答说：“自然制”。她又问我2C是什么意思，我回答不出，望着John张。张也说不出。裁判说：“你们先继续打，需要我再找我”。我原是冲着北家拿5张草花才叫满贯的，就对西家说：“那就算我打4S好了。”西家见明手只有12个大牌点而且还是均型，我又只叫过2S，好不到那里去，想6S是做不成的，当然不肯，按规则也不允许，我只好还是打6S定约。

他把首攻亮出来，是一张将牌。桌上S9吃住，东家告缺。兑现HA，将牌回手，手上出红心用HJ飞过成功，兑现HK，扔掉暗手方块。手上兑现DA，送出小方块；西家赢进后再回将牌，桌上赢进后出第三轮方块将吃，结果方块33分布，我吐了一口气，至此已大功告成，我用第四轮将牌到明手，兑现方块，垫掉草花，做成了满贯定约。

要做成6S红心必须飞中，而且方块还必须33分布，只有18％的机会，如果是首攻草花，连这点机会都没有，实在是运气太好了。东西家也未再找裁判，自认晦气。这副牌后来在《纽约时报》上刊登，并评论说这是一副很典型的打牌技术应用的例子。
