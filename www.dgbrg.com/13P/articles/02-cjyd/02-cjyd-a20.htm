技术性加倍


技术加倍是防守叫牌中应用广泛、效果良好的武器，不重视技术加倍的作用和用法，不正确地使用技术加倍，是很容易吃亏的。

技术加倍的基本含义，是要同伴在未叫过的花色中做出一个选择，也就是说，叫技术加倍的人，他在未叫过的花色中一般都有好的支持张。本章主要介绍敌方开叫一阶花色时，我方立刻叫“加倍”的这一种。有些人喜欢用加倍一阶花色开叫来表示持有开叫花色的长套，如果让他们了解到技术加倍的好处（表示敌开叫花色对应手中的短套），相信他们会改变原来的做法。在了解这些好处以前，先看看不用加倍一阶花色开叫来表示持有该花色长套的理由：

1、这种机会很少；
2、仅依靠将牌长套来击宕一阶定约的可能性不大；
3、一阶定约很少会宕二三墩，如果真能宕二三墩，则防守方的联手方往往可做成3NT，在得分上与加倍宕二三墩差不多；
4、敌方如觉得不安全，他们常可在一阶上逃到另一个花色，或是逃到1NT，总之可逃到更好的定约；
5、做一阶定约时，要想得分的得话，无论防守方还是定约方，均要拿7墩才行，此外做庄还有一定的有利条件，既然如此，为什么不想办法争做一阶定约，而要惩罚敌方的一阶定约呢？

§3、1 适合叫技术加倍的牌

1、大牌点至少相当于开叫一阶花色的牌力，并且敌开叫花色是单张或缺门，其它花色有好的支持张， 一般是 4-4-4-1 牌型（5-4-4-0 较少能碰上）。

持 SKQ64 HQ752 D7 CAJ65，右敌开叫1D时，可叫加倍。

持 SQJ85 H8 DK1083 CAKJ4，右敌开叫1H时，可叫加倍。

这样，一次叫牌就可以表示有三个套和开叫牌力，何乐而不为呢？ 有时， 牌点较多或其它花色的支持张较好， 也可以用4-4-3-2 的牌来叫加倍。例如，持 SAQ105 HA1082 DKQJ C87，右敌开叫1C，可以叫加倍；

2、大牌不少于16～17点，敌叫花色为短套，另有一个好长套或两个不太差的长套。持 SKQ HAKJ1064  D1086 CA，右敌开叫1C／1D时，均可叫加倍，下轮叫出H套，表示大牌不少于16点的一手牌。加倍之后再叫出新花色，是有好牌的表示；如果牌力较低，不可以加倍之后再叫出自己的长套，应该采用直接争叫。持 SA10984 HQ DAK1083 CK8，右敌开叫1C或1H时，均可叫加倍。但右敌开叫1D或1S时，只能Pass，等待同伴的平衡叫牌；

3、大牌不少于19点的均型牌（有16～18点，并且敌叫花色有止张的均型牌可直接争叫1NT，详见第四章），等第二轮叫牌时再作强烈的表示。例如，持 SKQJ6 HAJ DKQ4 CKJ106，大牌20点，右敌开叫1S时，可叫加倍，下一轮叫2NT。持 SKJ83 HKJ4 DAQ CAK107，右敌开叫1C时，可叫加倍，下轮可跳加同伴可能叫出的S；若同伴应叫1C／1H时，可跳叫2NT。

从上述几条规定可看出，如果加倍人不能保证每门未叫过的花色都有好的支持张，则他应当拥有强大的牌力来维持下一轮叫牌。因此当牌力非常强大时，叫技术加倍不一定必须持敌开叫花色的短套。使用技术加倍必须注意条件，牌型和大牌点必须符合条件，否则容易把事情弄糟。

持SJ1053 HKQJ64 D―― CAJ53，右敌开叫1D时，应当叫加倍，有人喜欢叫1H，说是把有一个长套的信息告诉同伴，但是，用技术加倍告诉同伴你有三个长套不是更好吗？

持SAQ432 H64 DAJ3 CAJ3，右敌开叫1H，很多人会争叫1S，而叫加倍才是正确的。虽然低花的支持张不是非常理想，但这手牌已有16大牌点，如果直接争叫1S，同伴根本不会想到到你会有那么多的防守赢张和相当不错的低花支持张，如果他有一个较弱的低花五张套和双张S小牌，你直接争叫1S后，你们就不大可能叫到一个低花的不进局定约。

持SAQ84 H9 DKJ65 CA432，右敌开叫1S时，只好Pass，不能叫加倍，因为敌开叫花色不是短套，如果叫了加倍，同伴叫出的长套很可能是H，那时就不好办了（再叫2NT则是传递错误的信息）。

持S8432 HKQ752 D―― CQ965，右敌开叫1D，不能叫加倍。虽然牌型符合要求，但大牌太少，还是不要向同伴传递错误的信息为好。争叫1H倒可以考虑。

持SKQ4 H8653 DAJ7 CKQJ，右敌开叫1H时， 虽然牌点够争叫1NT， 但敌叫花色无止张，只好叫加倍。这手牌对未叫过的花色都有不错的支持张，大牌不少，叫加倍是正确的。

持SKQ4 HA8 DQ976 CK432， 右敌开叫1S时，争叫及加倍的条件均不符合，只好Pass，等待同伴的平衡叫牌。有14点牌并没有什么了不起，它仅比四手牌的平均值多一个A的牌力，远未达到“强大牌力”这一范围。如果这手牌再增加一个Q或两个J，就可争叫1NT了。

持SAKQJ H8542 DK432 C9，右敌开叫1C时，一定不要叫1S，而应当叫加倍，尽管H没有大牌，但它的张数有四张，完全符合要求，如果同伴有四张H，那么联手做H定约很合适。当你争叫1S时，即使同伴有五张H，他也不大可能叫得出来（条件不够）。恰如其分地叫牌，无论是对增进同伴间的信任，还是对提高叫牌技术都是有益的。

持SA1053 HKQ96 D987 CKQ，右敌开叫1C时，可以叫加倍，不要叫1H，虽然D只有三张，但两个高花均有很好的四张，足以补偿，如果同伴是4-3-3-3 且D是4张而叫出1D，联手打1D并不是坏定约。若右敌不是开叫1C而是开叫1D。 叫加倍仍比叫1H要正确， 理由是如果同伴持4-3-3-3 牌型，他一般应在一阶上选择叫三张的花色；如果他有7～8点大牌，且敌叫花色有止张，可叫1NT；万一他是5-3-3-2且C五张，叫了2C，这也不是太坏的定约。

持S1052 HAKJ9 DAJ C10762，右敌开叫1C时，应当叫1H，叫加倍不合适，S和D的支持张均较少，而且敌叫花色不短，牌力也很低。如果右敌是开叫1D，此牌处于叫加倍和叫1H的边缘，不过著者倾向叫加倍。若右敌是开叫1S，此牌只好Pass，不能叫加倍，因D支持张太少，且同伴已不可能在一阶上选择做花色定约；你争叫2H也是完全不符合条件的。

持SJ974 H―― DK864 CAKQ53，右敌开叫1H，绝对应当叫加倍，而不应当叫2C，尽管C套很好，但它仅有五张，更重要的还在于S和D均有四张，是技术加倍的有利条件，不必只盯住C这个好套，不管同伴有S四张还是D四张，都容易在低阶数上找到配合，如果你直接争叫2C后，同伴叫四张的S或D就是很大的冒险，也是不合理的叫牌；当你叫加倍后，同伴的S或D的四张套可以理直气壮地叫出，不必担惊受怕。

§3、2 同伴技术加倍

我怎样叫同伴的加倍是要求你叫出长套 （当然不包括敌方开叫的花色），不能因为手上只有二三点大牌就不叫，绝不可像一阶开叫的应叫那样，0～5点就Pass；相反，牌力越弱就越不能Pass，因为敌方往往可做成这个被加倍的一阶定约。

常用的应叫有：

1、平叫新花色：0～8点，一般是叫未叫过的花色中，张数最多的花色，有时无可奈何只好叫三张的花色。持 SK83 H8762 DJ97 CQ75， 同伴加倍1D， 你应当叫1H，而不是叫1S；

2、跳叫新花色：跳叫高花色至少是好的四张，低花至少五张，9～11点，如果牌型不好，有9点时并且高花只有四张也不一定要跳叫。持 SA8732 H763 DA CQ762，同伴加倍1H，你可跳叫2S；

3、叫1NT：7～10点，大致是平均型，敌叫花色有止张。例如，持 S1086 HKJ84 DK64 C842，同伴加倍1H，你可叫1NT。同伴加倍1H，应能在未叫过的花色中有牌力（或适当的张数），你不必担心未叫过的花色。

技术加倍是不提高叫牌阶数而传递多一些信息，而它的应叫却不同，一般要叫到联手牌恰如其分的定约（或阶数）上。同伴技术加倍，多数是第一种类型的牌，并且持13点左右大牌的机会居多。常用的应叫，就是以这为基础的。假如拿不定主意怎么应叫，那么应把逻辑推理建立在上面这个基础上，就比较容易做出正确的决定。

持 SQ852 H87642 D107 C93，同伴加倍1D时，不可以因为只有2点大牌就叫Pass，必须叫1H，不要叫1S，虽然S有大牌，但H比S多一张，同伴除D外均有支持，为什么要害怕做H定约呢？

持 SJ10 HK7653 D8654 CAJ，同伴加倍1D时，应当叫2H，把好消息告诉同伴，如果只叫1H，同伴很可能会认为你是无可奈何的叫牌。再说，同伴加倍1D，一般能保证其它花色有好的支持，且有一定牌力，联手不怕做2H定约。

持 SAQ642 H1073 DJ64 CQ5，同伴加倍1S，你可应叫1NT（你负责守S，同伴可守其它三门）。

持 SQ83 H1042 D87543 CQ4，同伴加倍1D时，应当叫1H，不能Pass，D套太差，不足以把技术加倍转成惩罚。不叫1S的的原因是，如果同伴是SAK5 HA9875 DJ CQ953，叫1S可不是好定约；若同伴是SAJ732 HAK5 DJ CQ953，在你的1H应叫后，他勉强够牌力来改叫1S，如果是前者，就不能改2H（因牌力不够）。

持SAQJ5 H5 D842 CA10943，同伴加倍1D时，可以跳叫2S，按牌情也够格跳3C，但选择高花配合是一般原则，如果同伴的S和C均有四张，联手仍以做4S进局为好。

持SQJ85 HJ53 D864 C1086，同伴加倍1S，只能叫2C，尽管有S止张仍不可以叫1NT，因为牌力不够。这手牌几乎没有进手张，打无将定约时所有的牌都得从同伴手里领出，是很容易吃亏的。当每门花色都是三张时，且牌力甚低，应该叫最“便宜”的花色，以避免叫牌阶数升得太高；若有5－5两套，且6～8点，应当先叫级别高的花色，牌力少时仍应叫最便宜的花色。

持 SK52 H864 DJ9 CAQ986，同伴加倍1H时，要跳叫3C，叫2C显示不出那么多牌力。联手肯定有二十多点牌力，做3C并无危险。

持 SJ876 H986 DQ5432 C10，同伴加倍1H时， 应该叫1S， 牌力太低，以尽量不提高叫牌阶数为好，同伴有4张S时，联手做1S不见得比做5－3配合的2D差。

不常用的应叫有：

1、跳叫2NT：10（11）～12（13）点大牌，带括号为质量不好的点，敌叫花色有止张，均型。例如：持SAJ7 HQJ108 DQ654 CJ10，同伴加倍1H，你可叫2NT；

2、跳叫3NT：13（14）～15点，均型牌，敌叫花色有止张，但有时不一定是靠大牌点，而是依赖一个好的低花长套可以拿好几墩才这样叫。例如，持 SKQ9 HAQ DJ842 CQ1064，或 SAQ H842 DQ4 CKQ10953，在同伴加倍1S之后均可跳叫3NT；

3、扣叫：表示有好牌，大牌一般不少于12点，足以保证联手再叫一次仍在安全范围内， 有少数时侯不一定必须具备很多大牌，只是用其它叫品不易表达清楚牌情才这样叫而已。持 SA1093 HKQ86 D7 CKJ65，同伴加倍1D，你可扣叫2D， 让同伴在未叫过的花色中去选择， 而不是想打D定约；

4、高阶数的跳叫：这种跳叫并不是表示有好的牌力，相反，是表示牌力不多，但拥有一个较长的套，又不适合防守的牌（防守赢张少）；

5、不叫：把技术加倍转成惩罚加倍，一般要有敌开叫花色好的六张，并保证能得三墩将牌，而且旁门的防守赢张还不能等于0，否则应当考虑叫牌。这个叫品和上一个叫品的使用率是很低很低的。

持 S65 HQ65 DKJ2 CKQ763，同伴加倍1D，你应当叫2NT，虽然叫3C也符合条件，但同伴并不知道你有D止张。为什么S是双张小牌也可以叫无将？同伴牌力不多时，他没有S牌力是难以想象的（否则他不该叫加倍）。如果同伴的S牌张甚少，大牌也少，那么他就有足够的牌力叫出他自己的花色长套（请参阅技术加倍的三种类型）。

持SK63 H76 D102 CAKQ642，同伴加倍1S，可叫3NT，有理由相信C可拿6墩（同伴有J3或三张小牌即可，对叫技术加倍的人来说，这个要求很低），只要同伴能拿2、5～3墩，3NT大有希望。

持SK63 H92 DKJ10764 C85，同伴加倍1D，可以叫Pass，把技术加倍转成惩罚，如果S的K换成小牌（即没有旁套防守赢张），以叫1S为好。

持SQ9852 HA9632 DA64 C――，同伴加倍1D，应当叫2D（扣叫），虽然只有10点大牌，但有 5-5两套足以补偿， 不知跳叫哪个高花才合适， 所以用扣叫让同伴自己去叫出高花，然后你可加叫到局。看来同伴在扣叫之后，叫3C可能性较大，那时只好再叫3D，希望同伴叫出三张的高花套，你仍加叫到局。

持SQJ109632 H2 DQJ63 C8，同伴加倍1C，这手牌应该叫4S。同伴可能与你有好的D配合，或是有好的H套，但是无论何者，只要同伴有单张S大牌或双张小牌，联手的进局定约看来还是在4S上，因为有两套配合时，以哪一套做将牌差别不大。

持S4 HQJ107653 DJ32 C42，同伴加倍1C／1D，可以跳叫3H，此牌根本没有防守力量，做庄倒有好几墩可稳拿，3H不是坏定约，而且敌方可能有S或低花的配合，跳叫3H可使他们发生困难。为什么不跳4H？因为输张太多，不能希望同伴有太多的AK，如果他真有较多的AK，他自会加叫到4H。代为叫出同伴的牌力是很冒险的，一旦他没有你希望的牌力，你们只好蒙受损失；如果叫4H时，同伴恰好有你希望的牌力，你的行动不值得赞赏，因为这个4H本应由正确的途径叫到，歪打正着过多对提高叫牌技术不利。

§3、3 加倍人的再叫

在叫出技术加倍之后，该采取怎样的行动，大多数人往往处理不好，一些有若干年“桥龄”并且打牌技术不错的爱好者也是如此。原因之一是沿用正常开叫及应叫的规定或类似的规定；其二是这种机会较少碰得上，经验不足，临时决定采取某种行动，又容易发生错误。为了处理好加倍人的再叫，加倍人应该这样提醒自己：“同伴的叫牌是在我的加倍‘压迫’下做出来的，他可能没有多少牌力，甚至可能只有三四点大牌。我必须好好分析联手的牌力才容易做出正确的决定。”

由于持低限牌力（11～15点）做技术加倍在所有叫加倍的牌中占的比重较大，有不少于40％的牌在同伴应叫之后应当叫Pass！

在同伴不跳叫应叫的花色中有四张支持且有两个大牌也不值得激动，因为他可能仅是4-3-3-3 牌型中的四张套，甚至还可能仅是三张（四张套是敌开叫花色时）！！当同伴不甘心敌方的低阶定约又叫出另一花色时，你仍不可大喜过望，他仅是希望你在这两套中选择其一而已。在同伴所应叫的花色中支持张不太好而改叫其它花色，也是一个最常见的错误，因为加倍之后又叫出新花色是表示有好牌，如果加倍人仅13点左右的牌力，在同伴的非迫叫之后，绝大多数应该叫Pass。虽然在一阶上的更改允许只有15点大牌和一个五张套，但仍以少做这种更改为好，过多的更改只能失去同伴的信任，叫牌技术也不会提高。桥牌有35％的运气因素，歪打正着也许可以解救不少次叫牌错误，但它终究不能永远保证你不受到灾难性的惩罚。

由于以上各种原因，加倍之后的基本再叫就是Pass。当你对上面这种不叫有了一点“感情”之后，才能掌握好其它的再叫：

1、加叫同伴的花色：当同伴的应叫不是跳叫，则这一加叫至少有16点牌力；若同伴的应叫是跳应叫，则持有不太低的牌力就可加叫，由于同伴跳应叫高花可以只有四张，所以有四张支持才加叫比较合适；

2、跳加同伴的花色：何种牌适合跳加同伴的花色是不难估计的，只要做庄技术比较熟练，以及对同伴应叫的低限牌力有所警惕，该不该跳叫就基本上心中有数。所以当同伴的应叫不是跳叫时，你的跳加叫没有18～19点（包括牌型点）、没有四张好的将牌支持做保证是不行的，因为同伴可能只有三四点大牌（甚至可能0点），而且所叫花色可能仅有四张（甚至可能是三张）。

3、叫新花色：如果同伴的应叫不是跳叫或扣叫，加倍人叫新花总是表示持有好牌，否则应当叫Pass。不可以持低限且对同伴应叫的花色中只有三张支持时，去改叫其它花色。加倍人若是跳叫新花色，则表示这是好的六张套，邀请同伴进局；

4、叫1NT：表示15～18大牌点，敌叫花色有止张，对同伴所叫花色没有四张支持，但在另两个未叫过的花色有好的支持（否则，技术加倍就用得不对）；

5、叫2NT：如果是跳叫的2NT，应有19～20大牌点，敌叫花色有止张；若同伴不是跳叫，你的2NT也仅是平叫，则低限牌力可以稍小，敌叫花色也有止张；若同伴是叫1NT，那么你的2NT就是邀叫，当然，同伴先叫无将，你自然不一定必须在敌叫花色中有止张，大牌应当约有16～17点左右（因为同伴是7～10点）；

6、叫3NT：这相当于2NT开叫的牌；如果同伴的应叫是1NT，则这3NT可以只有18点，如果牌点还要少于18点的话，应该有一 个好低花套做为保障；

7、扣叫：在叫技术加倍之后，又扣叫敌方的花色，是有好牌 的表示，有时为了让同伴进一步叫明他的牌情， 也可以使用扣叫。扣叫右敌开叫的花色并不是希望做这个定约，因为当你叫技术加倍时，已表示不准备做这个花色的定约；

8、重复加倍：加倍人在第二轮叫牌时还是叫加倍，表示持有一手好牌。如果同伴曾做过应叫，这个加倍是惩罚加倍；若同伴未做过应叫，这个加倍仍是技术加倍。

无论是加倍人还是加倍人的同伴，在技术加倍之后的叫牌，都不需要保留正常一阶开叫的应叫中那种 “一盖一” 和“二盖一”的概念，如果有的话，只能起妨碍你们叫牌的作用。因为在防守叫牌中，传递信息几乎完全是另一套办法，沿用正常开叫及应叫的规定是十分有害的。西持 SJ2 HKQ6 DA984 CKQ53，南开叫1S，西加倍，北不叫，东叫2H。如果按“二盖一”的概念，西再叫4H是合格的，但是，东真的有10点以上的大牌吗？请读者不要翻前面介绍应叫的规定，而试着从叫技术加倍的性质来寻找答案。如果还记得技术加倍是要求同伴在未叫过的花色中做出选择，那么，东的2H就不必具有10点大牌，甚至可以只有0点，因为他只是在做一个选择而已。请替东想一下， 他持 S763 HJ986 DQ85 CJ76，可以把对1S的加倍转成惩罚吗？能够拒绝叫2H而叫1NT吗（同伴加倍1S，表示S是短套，此牌没有S止张，不应叫出1NT）。当你能够替同伴着想时，说明叫牌指导思想已产生了一个飞跃。

持 SKQ64 HQ752 D7 CAJ65，右敌开叫1D，你叫加倍，左敌Pass，无论同伴叫1H／1S还是2C，你均应Pass，这手低限牌是不应再提高叫牌阶数的。

持 SQJ85 H8 DK1083 CAKJ4，右敌开叫1H，你叫加倍，左敌Pass，同伴叫1S，你仍不够加叫到2S的牌力。跳叫到3S是非常错误的，如果同伴只有三张S，连2S都不易完成。

持 SAQ105 HA1082 DKQJ C87，加倍右敌的1C之后，同伴如应叫1H／1S，此牌刚够加叫到二阶；如果同伴应叫2D，可以Pass，进局无望。

持 SKQ4 H8653 DAJ7 CKQJ，加倍右敌的1H之后，同伴应叫1S，你可不叫。牌型太平均，以不叫为好（进局可能性不大）；如果同伴应叫2S，你叫3H较好，

希望同伴有H止张时以3NT进局，或再叫五张以上的S，或在四阶叫五张的低花；如果同伴应叫1NT，你可叫2NT。

持 SAKQJ H8542 K432 C9，加倍右敌的1C后，同伴应叫1H，你的正确再叫是Pass，而不是1S或2H；如果同伴应叫2H／2S，仍Pass，输张太多，无论4H还是4S均有可能损失四墩；如果是有局方，同伴叫2H时，可考虑叫3H，如果同伴有H五张，4H还是有希望的。

持 SJ1053 HKQJ64 D―― CAJ53，加倍右敌的1D后，同伴应叫1NT，你必须叫Pass，而不要叫2H，因牌力不够。

持 SJ974 H―― DK864 CAKQ53，加倍右敌的1H后，同伴跳应叫2S，可以叫4S，因为C能拿5墩的可能性较大，H看来也可拿2～3墩，4S值得一试；若同伴应叫1S，你只好Pass，以防他的S只有三张，如果右敌又叫了牌，则你叫2S才心安理得（争夺不进局定约）。

持 S1052 HAKJ9 DAJ C10762，加倍右敌的1D后，无论同伴应叫1H／1S，还是2H／2S均要叫Pass，虽然同伴跳叫2H保证了H的好配合，但输张太多，进局希望不大。

§ 3、4 技术加倍的其它用法

一、敌方两人都叫牌之后的加倍

北     东     南     西
1H     Pass   2D     X

北     东     南     西
1C     Pass   1S     X

上面二例，西的加倍都是技术加倍，表示未叫过的花色有很好的支持（至少5－4两套），大牌不少于14点；如果长套的长度和质量都很好，则大牌减少3～4点都可以；如果未叫过的花色仅是4－4，那么大牌不应少于15～16点。

北     东     南     西
1H     Pass   2H     X

北     东     南     西
1D     Pass   1NT    X

这两例西的加倍可以当做直接技术加倍看待，由于敌方两人都叫了牌，叫牌阶数已较高，所以西的牌点不宜低于14点，牌型也不宜太勉强，如果牌型很好，大牌可减少1～2点。右边一例，虽然南在未叫过的花色中各有3～4张牌的可能性较大，但西的长套处于南的后面，危险性较小。

二、延迟的技术加倍

头一轮有叫牌机会时未叫加倍，当敌方在低阶上显示了两套花色之后，你才叫加倍，表示有未叫过的花色两套（一般是5-4）例如：

北     东     南     西
1D     Pass   1H     Pass
2S     X

北     东     南     西
1H     Pass   1NT    Pass
2C     X

  
北     东     南     西
1H     Pass   2D     Pass
2H     X

东的加倍就是延迟技术加倍，牌力和直接技术的要求一样。有少数情况下，延迟技术加倍是迈克尔扣叫的补充形式。应当学会区分什么是惩罚加倍，在叫牌中发生误解，极易导致灾难。

北     东     南     西
1S     Pass   1NT    Pass
2S     Pass   3S     X

北     东     南     西
1S     Pass   1NT    Pass
3S     X

  
北     东     南     西
1S     Pass   2C     Pass
3S     X

上面三例，东的加倍均为惩罚加倍，他大多是持“埋伏”的牌，并不希望你叫出长套或无将。中间一例的加倍还是指示西首攻北叫过的花色（S）。关于“埋伏”的牌，在第十章介绍。东如果是技术加倍，他本应在第一轮就叫出。右边一例不是技术加倍的理由是，叫牌阶数已很高，并且敌方的牌力很强，技术加倍不会有什么好处。如果右例的东想表示两套牌的话，可叫3NT，理由是在前面同伴都未叫过什么东西，联手绝不可能具有打3NT的牌力。

三、对阻击开叫使用

由于叫牌阶数较高，加倍人的牌力应比一阶技术加倍为高，不应少于16点，而且牌型还要求比较好才行。三阶上使用技术加倍比一阶上使用容易出问题，这是桥牌本身的复杂性引起的，初学者不必因此不敢在三阶上使用技术加倍。三阶上叫出加倍后，你的同伴认为可以打宕敌方的定约，或者这个定约必宕且比自己一方做定约得的分要多，可以不叫，把要求叫牌的加倍转变成惩罚加倍，否则，应当维持叫牌。
