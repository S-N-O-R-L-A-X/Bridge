重新探讨逼叫性1NT


By Eric Kokish and friends  翻译：刘京  蒋又新  

    Ira Rubin曾经说起过，从理论上讲，对高花开叫的逼叫性1NT应叫虽然十分有用，但实际上也带来了很多附加的叫牌压力。主要原因就是逼叫性1NT叫品本身所应用的牌太过宽泛，尽管应叫人很顺利地作了一次逼叫，但下一次叫牌时往往就缺乏足够的手段了。你常常会发现应叫人的第二次叫牌很难按照顺畅的表达自己的牌情，他要么遇到一定的尴尬 ，要么需要被迫作出一个过快的决断。这都是由于应叫人没有再次逼叫的足够手段。 
        例如：  1S      1NT 
                2C       ？ 
        应叫人持有： 
        （1） S Jxx     （2） S xx      （3） S J10     （4） S Ax 
              H xx            H Jxxxx         H Axxx          H Jxxxx 
              D Kxxx          D AJxx          D Axxx          D xxxx 
              C xxxx          C xx            C xxx           C KQ 
        拿这些牌你可能都愿意（或者被迫）叫2S 
    再看对低花的加叫，其牌力可以从很差的义务性加叫一直到几乎可以逼叫的加叫，我们仍以上面的进程为例，应叫人持有下面的牌都会加叫C： 
                1S      1NT 
                2C       ？ 
        （5） S xx      （6） S x       （7） S K       （8） S xx 
              H x             H Axxx          H x             H xx 
              D Jxxxx         D xxxx          D Axxxx         D AKx 
              C KQxxx         C KQJx          C Kxxxxx        C Q10xxxx 
    有如此多种类的加叫牌的一个前提即是2C保证3张，正是1NT逼叫，也使得1NT之后开叫人的再叫并不一定完全真实。 
    更进一步说，应叫人如何处理下面这类牌： 
                1S      1NT      S Jx 
                2C       ？      H KJxxx 
                                 D Axxx 
                                 C xx 
    叫2S可能埋没了5－3配合的H定约，而叫2H可能导致打5－1配合的将牌而放弃打更好的S定约，不仅如此，如果开叫人在2H之后进一步叫牌，应叫人怎么知道他是由于H极不配合而作的改叫还是高限牌力继续作出的进局试探？这里面的不确定因素实在太多，Rubin个人认为如何处理这类牌完全取决于搭档间的习惯。 
   Ira Rubin是个很有见地的人，他所说的其实很有价值，至少也值得一听，但很遗憾目前他所介绍的内容并没有正式的文稿。当今世界绝大多数牌手，特别是那些顶尖级牌手都非常清楚上面的问题，并且对叫牌手段作了相当的改善。实际上，正是那些缺乏经验而又不够细心的牌手总是对上述的那些牌例感到不快。Rubin也提出了一种并未广泛应用的方法，使得应叫人可以解决大多数牌的再叫问题。 
    同样不可否认，普通的体系构架也有相当广的适用范围，此时可供选择的叫品包括：（a）2NT；（b）3阶开叫高花；（c）3阶新低花（跳叫或非跳叫）；（d）最便宜的H叫品 。这基本上也可以覆盖很多种类的牌。 
    我们并不是说这种普通的体制不可以用，而是我们很清楚，逼叫性1NT作为二盖一进局体系的有力辅助工具，需要很明确地定义并准确交换信息。但逼叫性1NT本身就造成了很多叫品的模糊。无论是优秀的牌手还是差一些的牌手，都有自己处理这类牌的方式，但可能或多或少都有一些不足。 
    若干年以前，我和Bernie Chazen谈论了很多关于叫牌法的问题，从他那里听到的很多都让我感到很有兴趣，其中之一就是关于逼叫性1NT的后续处理。技术细节上，Bernie 以及他的搭档在下面这一序列后做了一定的研究： 
        开叫人          应叫人 
          1S             1NT 
        2C/2D             ? 
    改进的焦点在于应叫人最廉价的再叫，即2C后的2D，2D后的2H。将这一叫品作为傀儡转叫，应叫人可以很轻松地获得大量的再叫来描述各种很棘手的牌。当然明显地就是此时的一个自然品被损失了，也可能会有其它更多的损失，我们将留在后面继续讨论。 
    让我们看一下这种新的结构如何使用。首先我们统一使用标准的北美叫牌方式：5张高花叫，二盖一逼叫进局，弱跳应叫，简单加叫（1S－2S）显示7－10点，1NT逼叫上限为12HCP。 
    应叫人有各种牌都不会Pass1S，例如有某种配合，在H上有潜在的成局可能，寻求较好的改善部分定约的前景。这就使得逼叫性1NT的牌力范围很宽，大概可以是2－12HCP。 
    开叫人在逼叫性1NT应叫后，如果没有很好的叫品，将会再叫3张的低花（3－3时叫 2C）。他可能是一手平均牌型也可能是非均型，他手里的牌可能有够跳再叫的牌力，但是由于体系中对跳叫的限制（如长度，强度）无法立刻进行跳叫来显示，也可能牌力刚好不足以跳叫新花色，因为跳叫新花是进局逼叫。 
    开叫人          应叫人 
      1S             1NT 
      2C              ？ 
        2D：要求转移至2H，可能是弱H单套的牌，也可能是其它牌情需要进一步表述； 
        2H：特定地，双张S，5－6张H，不逼叫； 
        2S：双张S支持，8－10HCP； 
        2NT：自然叫，没有很好的C支持，可能隐瞒H套 
             （开叫人如果接受邀叫并且有3张H有义务叫3H）； 
        3C：自然叫，但仅仅是礼貌性加叫，不承诺太多大牌实力； 
        3D：自然叫，邀叫； 
        3H：自然叫，好的6张以上H，邀叫； 
        3S：坏的限定性加叫，10－11HCP，3张支持； 
        3NT：C极配，S上有很好的特征，不逼叫； 
        4C：C极配，S短，不逼叫； 
        4D：D＋C双套，进局逼叫； 
        4H：H＋C双套，进局逼叫； 
        4S：很强大的S附加一些大牌，例如： 
                S KQ10xxx 
                H Kxx 
                D xx 
                C xx 
        4NT：比叫3NT的牌还多1墩（C配合加S大牌）； 
        5C：比叫4C的牌多1墩，C极配，且S短。 
    注意：应叫人持有某些1－4－5－3的牌会有些麻烦；由于没有自然的2D可叫，他可以选择Pass 2C，冒险打3－3配合的将牌；这也意味着如果一些特定情况开叫人持有很好的S套准备做第三次叫牌的时候将有更大的打2C的定约的危险！ 
   2D后开叫人的再叫： 
    开叫人          应叫人 
      1S             1NT 
      2C              2D 
      ？ 
        2H：正常转移，这是最普通的叫品，除非开叫人持有很好的牌才可以越过2H作 
            再叫； 
        2S：很好的牌，6张以上S，这不是一个修正定约的叫品； 
        2NT：自然叫，大约17－18HCP，不逼叫； 
        3C：S＋C 5－5以上，建设性叫品； 
        3D：很好的牌，D短，典型的牌即是5－3－1－4；万一应叫人是D单套的弱牌可以 
            打3D定约，开叫人有足够的大牌实力弥补，而对于应叫人是H的单套牌有邀 
            叫意味； 
        3H：很好的牌，H短，典型的牌是5－1－3－4； 
        3S：很好的牌，S很强，不逼叫；前面的2C可能只是一个跳叫前的准备叫； 
        3NT：大约19－20点，但不适合跳叫（例如套很差）； 
        4C：S＋C 6－6以上，不逼叫； 
        4D：相当于4S再叫，但D短； 
        4H：相当于4S再叫，但H短； 
        4S：H和D都是短套，例如7－4型： 
                S KQ109xxx 
                H x 
                D x 
                C AKJx 
    接受转移后应叫人的再叫： 
    开叫人          应叫人 
      1S             1NT 
      2C              2D 
      2H              ？ 
        Pass：H单套的弱牌； 
        2S：弱示选，2－6HCP有3张支持或双张支持至多7HCP； 
        2NT：自然叫，且有C配合，开叫方可以在真正持有C套且为低限牌力时修正为3C； 
        3C：有进局诚意的严肃的C加叫，不逼叫； 
        3D：弱D单套的止叫； 
        3H：自然叫，至少6张以上，但是比2C后直接叫3H的套的质量要差，邀叫； 
        3S：区别于1S－3S和1S－1NT－2C－3S，这一叫品有两种可行的定义： 
           （a）   S支持，且有C配合的坏的限定性加叫； 
           （b）   特定的4－3－3－3牌型，4张支持； 
            你可以选择喜欢的一种使用； 
        3NT：12HCP，特定的3－3－3－4牌型，且两门红花色都不错； 
        4C/4D/4H：很不寻常的对S的配合显示叫，有真正的S配合，但是仍不反对打更长 
                  的花色，只是长套上没有足够的强度，例如： 
                S QJx 
                H -- 
                D QJ10xxxxx 
                C Kx 
                叫4D； 
        4S：S AKQxx，边花什么都没有。 
   下面一个部分，我们讨论开叫人再叫2D后的叫品，我们所用的手段与前面十分相似，但是叫牌的空间变得更少了，因此也有一些很明显的差异。 

    开叫人          应叫人 
     1S              1NT 
     2D               ？ 
        2H：要求转移至2S，现在不能打到2H，而应叫人通常都会持有弱示选的牌，如果 
            开叫方面对一手弱示选的牌准备继续叫牌的话，那么他就可以不接受转移而 
            叫其它叫品； 
        2S：双张S支持，建设性，8－10HCP； 
        2NT：自然邀叫，但通常没有D配合，可能隐瞒H套； 
        3C：自然叫，建设性，大致相当于二盖一不逼叫进局体系中的1S－2C－2X－3C进 
            程的牌； 
        3D：自然叫，礼貌性加叫； 
        3H：自然叫，建设性邀叫，但此时无法像前面介绍的进程那样区分套的质量； 
        3S：坏的建设性加叫，10－11HCP，3张支持； 
        3NT及更高的叫品：同开叫方再叫2C后的叫牌。 
         
    2H后开叫人的再叫： 
    开叫人          应叫人 
     1S              1NT 
     2D               2H 
     ？ 
        2S：正常转移，没有足够的额外特征； 
        3C：高限，5－3－4－1型，C单张，若应叫方为C弱单套时可以Pass； 
        3H：高限：5－1－4－3型，H单张，若应叫方为H弱单套时可以Pass； 
        其它叫品与开叫方再叫2C后相同。 

    接受转移后应叫人的再叫： 
    开叫人          应叫人 
     1S              1NT 
     2D               2H 
     2S              ？ 
        Pass：双张支持7HCP以下或3张支持2－6HCP，弱示选； 
        2NT：自然邀叫，同时D有配合，开叫方再叫3D不逼叫； 
        3C：C单套，几乎是止叫，弱于2D后直接叫3C的牌； 
        3D：好的D加叫，建设性邀叫； 
        3H：H单套弱牌，止叫，此时无法在二阶打到2H； 
        3S：限定性加叫，区别与1S－3S和1S－1NT－2D－3S，与前面介绍的相同，可以 
            选择两种方案之一； 
        3NT及更高的叫品与开叫方再叫2C后相同。 
    这种方案在解决最初提到的那些情况上有着很明确的价值，尽管你在有些情况不能打到2D或2H，你在尝试任何方法的时候总是不可避免会遇到一些问题和危险（很多有经验的牌手认为，为了维持叫牌的发展冒险打5－1配合的将牌比Pass2C或2D而冒险打3－3配合的将牌更糟糕）。同时，开叫人也要更加仔细一些地计划他的第二次叫牌，因为应叫人更有可能持有不太合适的牌而Pass。预见未来叫牌中的问题，并有计划的处理可以使得开叫人能够更好地组织不同的叫牌方式（例如持有好牌但是不同强度的S套可以选择不同的叫牌 手段）。 
    我和我的两个常规搭档打了几年这种方法，也取得了不俗的成绩。但是很奇怪的是，我和另外一些重要的搭档也打老式的标准逼叫1NT的处理工具，也没有遇到太多的很吃亏的情况。审视新方案的优劣或许还需要收集更多的资料，不过从百分比看来，还是有相当大比例的人已经在打了，所以很推荐你尝试一下。 
    很明显，通过转移，你可以区分直接叫牌和转移后再叫牌的各种含意。而且这并没有太多的影响，这种概念本身是很重要的。 
   大约15年以来，我和我的队友们一直在打一种约定，称为Mittelman附加叫。近期曾经在Bridge World上面由Josh Parker写到过这种叫法，如果你没有看到那篇文章，我在这里一并介绍给你。 

    在下面进程中： 
    开叫人          应叫人 
      1S             1NT 
    2C/2D            2NT 
      ？ 
        3阶原低花：不逼叫； 
        3阶新低花：逼叫，描述性的，一般是3张； 
        3H：逼叫，描述性的，一般是3张； 
        3S：逼叫； 
    这种方式很直接，也很好用。当开叫方持有双高花时，他还可以既能寻找H配合又能照顾其它情况。我们建议的使用方式是： 
    开叫人          应叫人 
      1S             1NT 
      2H             2NT 
      ？ 
        3C：转移到3D，之后再叫： 
                3H：逼叫，5－5(+)牌型； 
                3S：逼叫，6(+)－4牌型； 
                3NT：C上的特征，一般是5－4－1－3或5－4－0－4牌型，要求选局； 
        3D：逼叫，D有特征，一般是5－4－3－1或5－4－4－0牌型； 
        3H：不逼叫，5－5+牌型； 
        3S：不逼叫，6－4型，非高限，但是两套很有特征，输墩不多，例如： 
                S QJ1098x 
                H AKQx 
                D xx 
                C x 
    这种约定使得你可以在正确的花色上成局，并且在开叫方持有大量高花而应叫方持有大量低花时可以打到3NT。同时你也可以在某一个低花极配而另一个低花不配合时，避免打错误的3NT，而5阶低花甚至满贯都可以打成。 
    如果你们倾向于在低限牌持有5张S和6张H时开叫1S，还可以额外加一些定义。例如1S －1NT－2H－2NT之后，开叫方跳叫4H即表示这种5－6套的特定牌型，而跳叫4C/4D可以表示6－6套，低限牌力，所叫花色缺门。当然一切都取决与你们叫牌体系中对低限牌的一些叫牌原则和套的质量的要求。 
    2NT再叫： 
    1S－1NT－2NT这个序列通常是用来描述18－19HCP左右的平均型或准均型。你可以考虑在这个序列中使用转移叫，从而获得更大限度的叫牌弹性和不同程度的描述。具体的方式如下： 

    开叫人          应叫人 
      1S             1NT 
     2NT              ？ 
        3C：要求转移至3D，开叫方如果有特殊的配合可以进行超转移； 
        3D：要求转移至3H，与上述类似，转移之后的3NT再叫是要求选局； 
        3H：要求转移至3S，应叫方可能是一手很破的牌，如果转移之后应叫方再叫3NT 
            是显示双张带大牌的S支持，要求开叫方选局（应叫人通常某套有双张小）； 
        3S：显示C套，开叫方只有在对C有好配合（Axx，Kxx或更好）时才可以叫3NT， 
            否则应转移至4C；开叫方如果叫3NT，应叫方继续叫4C应当视为弱牌且几乎 
            为止叫。 
        3NT：自然叫； 
        4C/4D/4H：3张S支持的配合显示叫，例如下面的牌就可以叫4D： 
                S Kxx 
                H xx 
                D AQJxx 
                C xxx 
        4S：非寻常的进局加叫，如前面所提到的，类似： 
                S KQxxxx 
                H xx 
                D Axx 
                C xx 
    以3H转移至3S之后再叫4阶新花色可以定义为S配合以及如下特征：（a）该花色单缺，或（b）某种特殊的配合显示叫，类似： 
                S KQJ 
                H xx 
                D Axxxx 
                C xxx 
    转移至3S再加叫4S显示一手坏的限制性加叫，并且没有太好的特征。在这个体系中如果你要显示双低花，应叫人就要先转移至D再叫C套（逼叫）。 
    在1H－1NT－2NT后，转移叫的方法也依然有效，此时3H可以显示C套，而3S显示双低花。 
    你可以采用你喜欢的方式，而无需管其它的，如果实践中不太好用，也不必向人抱怨。
