不再漏掉一个局 (4)

Never miss a game again
作者:Zar Petkov(Canada) 翻译：monk
　

4)进取

    在桥牌中一定要进取或者保守吗？我希望你知道这个问题的答案--两者都是需要的。Karen McCallum曾经说过"我这辈子还没有Pass过一手有缺门的牌呢"，你想想，一手牌有了缺门就已经至少有14 Zars了。再给你这手牌两个A，加上12 Zars，这是一手有仅仅8HCP但是却有26 Zars的牌。你可以拿着8个标准大牌点进行开叫！

    这里有个例子，来说明在老式的HCP风格和Zar风格里面，"两手开叫牌可以打成局"的原则：

    首先，最常见的问题就是有24HCP是否可以打成局：

        W: S Qxxx       E: S KJ10xx
           H QJx           H Kxx
           D QJ            D Kx
           C KJxx          C Qxx

    标准叫牌法中，东有12点和5张好的高花套，西有12点和4张支持，两手都是可以开叫的牌，但是无论3NT还是4S都是没机会的。

    Zar中，东有12+3+3+8=26点，而西有12+1+2+8=23点，并不能进行开叫，即使SQ加上一点也还是不够。这仅可以加叫3S，成局需要联手有52 Zars。

    下面这个例子中将看到前面提及的"Karen风格"的牌，这很接近"开叫还是不开叫"的问题，看看标准叫牌法和Zars的区别：

         W: S KQxxx     E: S Axxx
            H KJxxx        H A10xxx
            D xxx          D xxxx
            C --           C --

    标准叫牌法中，东有8HCP，而西有9HCP，你拿两手牌都会Pass。

    而Zar中，东有8+4+9+5=26 Zars，西有9+2+10+5=26 Zars。两手都是可以开叫的牌，并且两手牌配在一起可以打成一个局，开叫的结果将是你们叫到很铁4H定约。

    我相信你已经注意到如果这两手牌中任何一手的C和D换一下，你竟可以打成大满贯！一副可能是All-Pass的牌居然会有大满贯！

    看起来很神奇吧，但是如果你还是用大牌点(HCP)来判断和评价一手牌，对牌型、控制、分布、进攻力量、花色支持等概念(这些在Zar Points的评价系统里都是能够计算的)仅仅有些模糊的感觉的话，那么这一切都是可能发生在你身上的。

    你只要能数到26，那就自信地开叫吧！

    在结束进取的开叫这个话题前，我们不得不展现一下什么是真正的进取。一些读者已经注意到了，在文章开头的那手7HCP的牌，我们也是建议开叫的。一定有人在嘟哝"这真够疯狂的"。

    现在我们稍微扩展一下--能开叫的牌中HCP的下限是什么?这里有几手牌能够给出答案:

        S Kxxxxx
        H Axxxx
        D xx
        C --
        7HCP，7+3+6+11=27 Zars

        S Kxxxxx
        H Kxxxxx
        D x
        C --
        6HCP，6+2+6+12=26 Zars

        S Kxxxxxx
        H Qxxxxx
        D --
        C --
        5HCP，5+1+7+13=26 Zars

        S 10xxxxxx
        H Axxxxx
        D --
        C --
        4HCP，4+2+7+13=26 Zars

    一手4HCP的牌也可以开叫！

    好了，如果你不喜欢这么刺激的话，大概会喜欢像4-4-3-2有8HCP这样"平静"的牌了，世界上任何一个人都会Pass的。

        S AJxx
        H Kxxx
        D xx
        C xxx

    就是这样，现在你知道我想说什么牌吗？

        8HCP            4HCP
        S AJxx          S 10xxxxxx
        H Kxxx          H Axxxxx
        D xx            D --
        C xxx           C --

    朋友，如果将牌是1-1的话，又是一个大满贯！
