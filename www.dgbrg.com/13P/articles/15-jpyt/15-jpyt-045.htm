二阶阻击叫后的Lebensohl


阻击叫后的叫牌对任何水平的牌手都是棘手的。在如下的叫牌后

北        东         南         西
2S        X          Pass      3D
Pass      ?

东很难判断西的实力。西可能毫无点力，但也可能拿了8-9点。为了解 决 这一难题，就设计了二价阻击叫后的Lebensohl ，以便更精确地描述牌情。 

二阶阻击叫后的Lebensohl约定

当对方开叫二价阻击叫，同伴技术性加倍，右手方 Pass，你的 2NT是Lebensohl。此2NT与无将叫牌无关，要求同伴再叫3C由你进 一 步描述牌情。 

以下以2H阻击为例说明其基本用法

北        东         南         西
2H        X          Pass       ？

1。弱牌 (0-6点)
1) S套：2S 
2) C或D套：2NT，同伴接力3C。

北        东         南         西
2H        X          Pass       2NT
Pass      3C         Pass       ?

C套：Pass
D套：3D

2。中等牌力 (7-10点)

1) C或D套：直接3C或3D。
2) S 套
A。四张：2NT，同伴接力3C，后3S：

北        东         南         西
2H        X          Pass       2NT
Pass      3C         Pass       3S

B。五张以上：直接3S：

北        东         南         西
2H        X          Pass       3S

籍此可区分S的长度。

3。11点以上的牌
Lebensohl 约定可以使你保留原来的叫牌含义。


加倍者强牌 (18点以上)

Lebensohl 2NT后，同伴持强牌有时可以不接力3C，而表达自己的牌。由于Lebensohl 2NT 者通常持一手0-6点的弱牌，只有当面对4-5点牌都有进局 前景时才考虑越过3C。

北        东         南         西
2H        X          Pass       2NT
Pass      ？

3D/3S/4C：D/S/C好套，至少18点，基本逼叫。
3H：极强牌，逼叫进局。
3NT：自然叫。常基于一强长套。(有些牌手表达其他含意)
4H：C/D双低套，至少5-5。

Lebensohl 的特殊用法

    利用Lebensohl 约定还可以区分一些不同类型的牌。不同牌手采用的约定不尽相同，以下介绍我个人较喜好的方式。

1。进局实力(至少12点)

区分止张及有无四张S
1) 无四张S，H有止张。直接3NT

北        东         南         西
2H        X          Pass       3NT

2) 无四张S，H无止张。直接3H扣叫

北        东         南         西
2H        X          Pass       3H

3) 四张S，H有止张。Lebensohl 2NT，然后3NT。

北        东         南         西
2H        X          Pass       2NT
Pass      3C         Pass      3NT

4) 四张S，H无止张。Lebensohl 2NT，然后3H扣叫。

北        东         南         西
2H        X          Pass       2NT
Pass      3C         Pass       3H


2。满贯兴趣

I。单套
A。 S套满贯兴趣
1) 5张以上S，仅成局兴趣。直接4S。

北        东         南         西
2H        X          Pass       4S

2) 5张以上S，略有满贯兴趣。Lebensohl 2NT，然后4S。

北        东         南         西
2H        X          Pass      2NT
Pass      3C         Pass      4S

3) 4张以上S，极强满贯兴趣。Lebensohl 2NT，然后3H，同伴暂时认为你仅成局兴趣。你接着再满贯试探，表示好牌。

北        东         南         西
2H        X          Pass      2NT
Pass      3C         Pass      3H

B。低花单套
扣叫3H，然后4阶出套。(当同伴改叫4阶低花时，有时会难以澄清牌情)

北        东         南         西
2H        X          Pass       3H
Pass      3NT        Pass       4C/4D


II。两套型满贯兴趣

1) 双低花(一般至少5-4)，H无控制：
Lebensohl 2NT，然后4H

北        东         南         西
2H        X          Pass      2NT
Pass      3C         Pass      4H

2) 双低花，H有控制：
直接4H

北        东         南         西
2H        X          Pass       4H

3) 四张高花带一低花
Lebensohl 2NT，然后4C或4D

北        东         南         西
2H        X         Pass       2NT
Pass      3C        Pass       4C/4D

注意：同伴加倍后，你直接4C或4D，仅表示C或D邀叫，大约12-13明手点。

北        东         南         西
2H        X         Pass      4C/4D (邀叫)

以上尽管以 2H 阻击叫同伴直接位置加倍为例，类似原理也可运 用于2D或2S阻击叫后，以及平衡位置加倍。

另外，Lebensohl 2NT为一约定叫，实战运用时必须提醒 (包括衍生的含义)。
            
最后，Lebensohl 2NT约定虽是一个有力的武器，但并不能解决阻击叫后所有 的难题。It’’s not a perfect world!
