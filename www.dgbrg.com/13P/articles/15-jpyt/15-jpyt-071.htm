不再漏掉一个局 (3)

Never miss a game again
作者:Zar Petkov(Canada) 翻译：monk
　

3)应叫
    现在的情形是你的同伴进行了开叫，这时候你们就处于一个进攻性叫牌过程中，而敌人则处于防守叫的位置。这会对你手上的牌值有何影响呢？

    你首先要进行初始评估(在第一部分已经讲过)，然后进行一定的调整，调整同伴所叫的花色和敌方所叫的花色。这里你能够应叫的最低限是16 Zars。

    －将牌上的大牌多计算一点(将牌10可以有1点，J为2点，类推A计为5点)，最多可以多计2点；
    －对于进行邀叫的第二套花色的大牌多计1点(方法同上)，最多为2点。

    这里最多可以多计2点是因为同时持有过多的大牌实际上是一种重复。现在你可以准备判断你们能够打的阶数了。这里是成局的计算：

    －52 Zars可以打四阶高花局(两手都是可以开叫的牌就可以进局)
    －57 Zars可以打五阶低花局
    －62 Zars可以打六阶小满贯

    通常是5点一阶，这些5点可以来自同伴花色的一个K(3HCP，1控制，同伴花色加1点)，或者边花的一个A(4HCP+2controls)等等。

    让我们来看叫牌中四个最常见的加叫序列：

    1.
       W: S Q10xx         E: S KJxxx
          H Ax               H Kxx
          D xx               D xxx
          C Kxxxx            C Ax

    东家有26 Zars=11HCP + 8Long(5+3) + 3Short(5-2) + 4Controls。西家有21 Zars=9HCP + 9Long(5+4) + 3Short(5-2) + 3Controls。东家开叫1S，西家的牌可以进行调整，SQ加一点，S10加一点，这样就有26点，足可以进局。

    叫牌可能是1S-4S(26-30Zars)。如果再加10 Zars：东家D换成AKx，那么你们就可以打成满贯。

    2.
       W: S Q10xx         E: S KJxxx
          H Qx               H Kxx
          D xx               D xxx
          C Kxxxx            C Ax

    东家有26 Zars=11HCP + 8Long(5+3) + 3Short(5-2) + 4Controls。西家有20 Zars=7HCP + 9Long(5+4) + 3Short(5-2) + 1Controls。东家开叫1S，西家的牌调整后可以加2点，为22点。

    叫牌应该是1S-3S(21-25 Zars)。

    3.
       W: S Q10xx         E: S KJxxx
          H xx               H Kxx
          D xx               D xxx
          C Kxxxx            C Ax

    东家有26 Zars=11HCP + 8Long(5+3) + 3Short(5-2) + 4Controls。西家有18 Zars=5HCP + 9Long(5+4) + 3Short(5-2) + 1Controls。东家开叫1S，西家的牌调整后为20点。

    这次只能是1S-2S(16-20 Zars)。

    4.
       W: S AQxx          E: S KJxxx
          H Ax               H Kxx
          D xx               D xxx
          C Kxxxx            C Ax

    东家有26 Zars=11HCP + 8Long(5+3) + 3Short(5-2) + 4Controls。西家有30 Zars=13HCP + 9Long(5+4) + 3Short(5-2) + 5Controls。东家开叫1S，西家的将牌A和Q可以各升值一点，总共有32点。

    叫牌应该从1S-2NT(32+ Zars)展开，这里2NT是Jacoby加叫。如果东家的D是Axx，那么联手就可以达到小满贯最少62 Zars的要求了。

    需要注明的是，在我们最初的研究中，应叫的最低条件是18 Zars而不是16，但是通过后来我们进行的额外实验，应叫标准被调整到16，这样也更符合5点一阶的计算方式。所以我们说，开叫方至少有26 Zars，开叫了1S，应叫方有16 Zars，就可以加叫2S。如果两手牌还能够各附加5点，那么你们就可以打成局了(达到52 Zars)。

    大牌的重新评价--你有多少实际资产

    当你估算初始牌值时，有个标准的方式就是对于短套上的大牌贬值1点计算。随着叫牌的发展，你就要对同伴和敌人所叫的花色进行修正。

    请记住评估你实际资产时著名的三个重要原则，这三个原则是与计算大牌点(HCP)部分的三条重要原则相匹配的。

    Rule 1)你的大牌位于同伴所叫花色--每张大牌加一点(包括10)，最多加2点(如果你有KQ10也只能加2点，而不是3点)；

    Rule 2)你的大牌位于敌人所叫花色--敌方所叫的花色中的大牌可能加1点也可能要减1点，这取决于叫该花色敌人的位置：AQ或者Kx在叫牌人的后面应该升值，而QJx分别要减值；相同的，AQ或者Kx在叫牌人的前面要减值。

    Rule 3)对于短套中你已经贬值过的大牌--如果该套是同伴所叫，那么这些大牌不但不要被贬值，反而是有额外价值的，而如果是在敌方所叫花色，这些大牌都不计点(译注：A除外，A是永远有价值的)。敌方花色的双张QJ完全没有价值，应该被减为0点，但是如果换作同伴的花色，就可以值4点，这里两张大牌只升值1点是因为这种双张大牌缺少某种打牌的灵活性，而且容易产生阻塞。

    在叫牌过程中，你不断了解同伴叫牌和敌方叫牌所带来的信息，进一步作牌值的调整。例如你右手的敌方叫了一门花色，这一套的AQx可以升值为AKx，而如果是你左手方叫的花色，这套的KJx就要减值1点。总之，你要不断动脑思考。

    配合的重新评价

    到现在为止，我们已经知道52 Zars可以成局--在大多数牌上都是如此。考虑一下这个问题，比如我有6张将牌(对于同伴加叫来说，他只认为是5张，6张将牌或许是个意外的惊喜)，而这已经是通过Zar Points分布点考虑的因素了，是不是这样呢？

    事实上，你有6张长套确实是在开始计算你手中的初始Zars时已经考虑进去了，但是没有考虑的因素是同伴在认为是5张的时候加叫了该花色，这就意味着我们有了额外的配合价值。

    除了你的"实际资产"以外，你的分布点也随着你和同伴牌的配置情况发生改变。在没有叫牌信息的时候，5431牌型的Zar Points就是13(分布点)，如果同伴的牌型是1345，那么你们两个人的牌值依然都是13点，不会发生变化。

    我们要能够计算出我们的花色中额外长度的价值，计算方法是：

    Rule 4)在将牌花色上，所承诺长度之外的每一张额外将牌都多计3个大牌点；

    Rule 5)在用来邀叫的第二套花色上，超过4张的每张额外长度多计3个大牌点；

    你知道我们的计算是基于不定方程组的，这里我们通过引入一个X_superfit的额外变元进行分析求解的，除了计算一系列方程的系数外，我们这种重新评价牌值的方法也是符合"总墩数定律"的。

    你可能会问"你所说的5 Zar Points一阶，一张额外长度3点的原则怎么符合总墩数定律呢？"(译注：按照总墩数定律，联手将牌长了一张，那么能负担的阶数就高一阶，按照Zar Points的理论这一阶应该多5 Zars，但是前面所讲的规则4中仅仅加了3 Zars，那么额外的2 Zars是怎么丢的呢？)这是个很合理的问题。答案是这样，你的6张长套在计算初始Zars时被考虑了2点，1点来自算式(a+b)，另1点来自(a-d)。但你或许会说"总墩数定律仅仅在敌人的大牌(HCP)相对平均分布时才适用"--没错。在我们很进取的叫牌理念中，大牌也是被分成两部分--牌型分布和位置成就了那些进取的成局定约，因此这也是无庸置疑的。

    第二个经常问到的问题是，"既然我们把极配加了不少点，我们是不是也应该调整成局52 Zars，满贯62 Zars的最低限制呢？"答案是"不"。如果那样你会经常叫到一些非常脆弱(或者说进取，如果你更喜欢这个词的话)的局或者满贯，当然不会失掉那些每天都叫的正常定约。

    当评价你们联手的牌配合得是不是完美时，有两种情况将是对你们十分有利的：

    1)你们有一个配合，但是是个极配，比如10张以上配合；

    2)你们没有极配花色，但是却有两套配合，比如一个8张以上配合，另外一个7张以上配合

    这里是重新评估联手配合时，如何计算额外Zar Points的准则：

        9张配合--3点
        10张配合--6点
        11张配合--9点

    对于"总墩数定律"的具体内容，请参阅Larry Cohen专门论述这部分的书籍。

    Zar将吃力量的计算--再看总墩数定律

    上面所说的额外将牌长度是一种盲目的价值，我们是否可以更好地来衡量额外将牌的价值呢？这些额外的将牌是仅仅贡献了一个额外的赢墩还是限制了敌方手中将牌的数量？

    看一下下面这两手牌，都是有5张将牌(将牌为黑桃)，并且你的同伴开叫了1S。A这手牌是5-4-4-0牌型，而B是5-3-3-2牌型。

                Hand A                  Hand B
                S A10974                S A10974
                H --                    H 876
                D 10865                 D K7
                C A963                  C K83

    诚然，这种区别在分布点的计算式(a+b)+(a-d)中有所反映，但是这只是在不知道将牌的前提下。两手牌都有25 Zar Points，4控制和5张将牌，但是这两张额外的将牌是不是价值相同呢？

    我估计你会比较喜欢A这手牌，虽然它只有8HCP。原因就在于A这手牌有更多的"Zar将吃力量"。

    上面我们说了对于8张配合之外的额外将牌长度，每长一张将牌可以加3点，现在我们需要做一定的调整了，只在某些情况下，才会给额外的将牌加上全部的3点，在另外的时候则会打些折扣。这里是计算额外将牌长度的"Zar将吃力量"的方法：

    对于额外的将牌长度，你需要给每张加上(3-d)点，这里d就是你最短套的花色长度。如果你有一手4-3-3-3牌型，你的额外将牌除了在初始计算时已经考虑过的长度外，就不具有任何额外的价值了--(3-d)=3-3=0。

    如果你有个单张，那么每张额外的将牌都可以加上(3-1)=2点。

    稍等！我们是不是可以找到一个更简便的形式呢？实际上可以归结为以下几条：

    Rule 6)如果你的短套是缺门，额外的将牌长度(超过8张配合的)每一张计3点；

    Rule 7)如果你的短套是单张，额外的将牌长度每张计2点；

    Rule 8)如果你的短套是双张，额外的将牌长度每张计1点。

    似曾相识？

    没错，这就是Goren计点法里牌型点算法(双张1点，单张2点，缺门3点)，但是这里仅仅在持有额外将牌长度的时候才适用，也就是将牌配合要超过8张。因此，对于前面的两手牌来说，都有两张额外的将牌，A的两张额外将牌值6 Zar Points，而B的额外将牌只有2 Zar Points的价值。

    这里有一个计算Zar将吃力量时需要注意的问题。如果你们的将牌仅是常规配合(8张配合而不是极配)，而你的将牌长度和短套长度差至少达到2张的时候，你也可以计算一次额外将牌的价值。所以如果你有4张将牌和边花一个单张，你加叫了同伴的4张花色(比如一盖一花色)，你也可以计算2点；相同的，你用3张将牌加叫同伴的5张高花，单张也可以计2点。但是，3张将牌的时候你就不能计算双张的价值了，因为它们的长度差仅仅为1。这一点我们已经进行了集中测试。这种调整算法可以照顾很多竞争性叫牌的情况，你需要判断该怎样做的时候也许会有所帮助。

    了解了以上部分，你应该认识到这给总墩数定律带来一些新的补充，无需盲目进行联手将牌张数的计算，而可以合理的调整你的行动。

    关于总墩数定律的额外发展，请参考本文的第二部分最后两节"The Finer Arts of Zar Points"。

    (注:Never miss a game again是第一部分，全文的开始为Annotation，后面还有The Finer arts of Zar Points和The Research on Zar Points，Bermuda Bowl - Zar Points Perspective，Critics and Opinions regarding the Zar Points等几部分，Never miss a game again是介绍Zar Points的核心概念的)

    你的6张长套有多少价值

    我们已经讨论过了双套配合和极配的价值，让我们来看下面这手很普通的有5张高花套的牌：

        S Jxx
        H KQJxx
        D AQx
        C xx

    你开叫1H，同伴加叫2H，敌方都Pass。你会叫什么？

    现在如果你的红心换成6张套，你的决定是否会有所改变？

    我们有13HCP，3控制，总共16大牌点，加上11个(3+8)分布点，共有27 Zar Points。一手开叫牌，并且我们已经做了开叫，但是没有太多额外的东西了--因此我们Pass。

    挪一张边花的牌张到红心上，变成红心是6张套。情况有所改变吗？

    这取决于第6张红心是从哪个花色移过来的。如果你从双张花色(即C)挪一张过来，你就变成6-3-3-1牌型，Zar分布点从11变到14，并且第6张将牌可以加3点，总共有33 Zar Points--足可以进行成局试探了，因为你独自就可以承受3阶。因此你可以叫2S，寻求同伴在该花色上的帮助。

    如果你是从Jxx花色中移一张到红心上，你的牌型就是6-3-2-2，分布点从11变为13，加上第6张将牌的3点，Jx贬值1点，那么你的牌值就是31 Zar Points--最可能的依然是Pass，除非你在一个竞叫进程中。

    现在读到这里，你可以看到Zar将吃力量计算的意义了。

    同样，如果第6张将牌来自AQx--你需要扣除AQ双张1点(双张大牌需要贬值1点，因为其打牌中的不灵活性，容易发生阻塞，前面有所提及)，加上第6张将牌的3点，总牌值也是31 Zar Points--Pass，除非是竞叫。

    一个容易掌握的原则就是--如果你能数到32 Zar Points就可以试局，如果达到36 Zar Points就可以直接进局。你也能够知道如果移动边花的2张牌，使之变为6-4-2-1或者7-3-2-1牌型，你所采取的行动将会有何改变。

    再看一个6张高花的牌例：

        S Jxxxxx
        H xx
        D AQx
        C AQ

    这次你开叫1S，同伴叫2S。这手牌来自Jeff Mechstroth的《Win the Bermuda Bowl with me》一书--如果你的钱很有限只能买一本桥牌书的话，这本书应该是你的选择。

    Jeff的观点是这手牌的价值仅可以进行成局试探。让我们看看计算器告诉我们的结果。我们有13HCP和4控制，加上9+4=13分布点(持有黑桃套加1点的原则仅在你决定是不是开叫时候才计算)，总共有30 Zar Points。

    如果你很懒，想使用每张额外将牌加3点的规则，你就加上第6张将牌的3点，这样就达到了33点--足以进行成局试探。如我们所讨论过的，直接跳叫进局需要36 Zar Points以上。如果你用Zar将吃力量计算，那么因为短套是双张，你就只能加上1点了，可以有31点--确实就是在成局试探的边缘。

    看来计算结果表明Jeff这次又对了。
