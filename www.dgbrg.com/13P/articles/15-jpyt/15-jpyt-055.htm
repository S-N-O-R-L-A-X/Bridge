Drury约定叫的后续探讨


这牌叫错在哪里？
-------从一副错误的定约看Drury约定叫的后续叫牌

前天晚上，和几个朋友在中游玩牌，我的PD是“傻样儿”，OPPS是“卖花布”和他的朋友。下面这副涉及Drury约定叫的牌，引起大家的激烈的讨论。

一．持牌
北（“傻样儿”）：          南（“山上摇铃”）：
S：J8x                   S：AKQ10xx
H：J10xx                H：KQxx 
D：x                    D：KQ10x
C：AK9xx                C：――

东家和西家因为没有参与竞叫，所以持牌省略。

二．叫牌过程：
             N        E       S         W
            pass     pass    1S         pass
            2C①     pass    2D②       pass
            2S③     pass    4C④       pass
            4S⑤     pass    6S⑥       all  pass

注①：2C为Drury（目前国内中文翻译多为“德鲁里”，过去常译为“朱瑞” ）约定叫。它针对第三家（偶尔也包括第四家）经常允许轻开叫1副高花（9-12P，且所叫花色可能仅好的4张），你作为PASS过方恰好持一副有配合的好牌（10-12P，3张以上将牌支持，如S：K10X；H：Q9XX；D：AJXX；C：XX。你PASS后，同伴开叫1H 或1S），你不应该采用跳叫3副邀叫，而叫2C，问同伴是否是正常开叫。这种约定性应叫仅运用于高花开叫情况，它可以避免开叫方牌力欠缺时的冒叫，以及可能错过成局定约的烦恼。

对Drury约定叫的答叫有两种类型：

（1）早期的简单Drury答叫：开叫方如果是轻开叫或者牌力欠缺，就叫2D，意为持低限牌力，或是没有正常开叫牌力，对成局无兴趣。如果开叫方属于正常开叫，可以示第二套花色，或叫回原始高花示套，或再叫无将示一手好的平均型牌。

（2）后来改进的逆Drury答叫：开叫方如果是轻开叫或者牌力欠缺，就简单在2阶回到原来将牌花色上，意为持低限牌力，或是没有正常开叫牌力，对成局无兴趣。如果开叫方属于正常开叫，可以示第二套花色，或跳叫原始高花示6张套，或再叫无将示一手好的平均型牌。

还有，现代流行所谓的双路Drury，即2C和2D均为Duruy约定叫，二者的区别仅在于对同伴开叫高花的张数的差异。其中2C表示对同伴开叫的高花正好有3张支持，而2D表示对同伴开叫的高花正好有4张或4张以上的支持。

注②：2D为逆Drury答叫。显示开叫方属于正常开叫，且D为第二花色套。

注③：2S为Drury约定性应叫的低限。

注④：4C为splinter，显示所叫花色为单张或缺门。表示满贯兴趣。

注⑤：4S表示C上有浪费点力，不宜试探满贯。

注⑥：6S认为即使同伴C上有浪费点力 ，但假设同伴持有一红花色A还是完全可能的，所以6S应该是一个合理的定约。 

三．打牌过程：

西家首攻单张H，东家HA拿，再回H，西家将吃，拔DA，定约2下。

四．有关叫牌过程各叫品的讨论

（1）2C：当时，南家因为一时冲动，叫上个臭不可闻的小满贯，恼羞成怒，将一股无名火发向应叫方的2C，狡辩说应叫方的牌叫2C太勉强，还是简单加叫2S（8―10P-）更好些。南家的无理行径当时就受到局中人和旁观者的一致谴责。好在南家还算是个谦虚好学之老学生，1分半中后就接受了2C――Drury是最合适叫品的观点。同时才有了后文，有了本贴，呵呵。

（2） 2D：对于开叫方的2D再叫，在场的大约有一半表示赞同；而以北家为代表的另一半则认为叫2H表示还有4张另一门高花更必要。

现在仔细想想，持5-4-4-0型牌（5张是S套，缺门的是C套），在同伴2C ――Drury问叫时，一定是叫2D更好些，除非D是全小的4张，而H是一不错的4张套。因为如果应叫方还有4张H，正好顺势在2D后以最经济的2H叫品呼出。4-4配合的H是丢不了的，而且在2阶就可以找到。

（3） 2S：如果上述第（2）点的理由成立的话，应叫方的再叫2S显然不如2H好。

（4） 4C：对于南家的4C=splinter，显示所叫花色为单缺，示满贯兴趣。疑义不大。

（5）4S：对于北家4S表示C上有浪费点力，不宜试探满贯，当时所有在场牌手均没提出疑义。 

现在再仔细想想，还是有可商榷之处：即也许5C是一更积极的叫品。因为 4C=splinter，显示所叫花色为单缺――有可能是缺门，更有可能是单张。AK结构的价值虽然对于同伴的缺门是个灾难，但是对于同伴的更有可能的单张的话，要比KQ结构的价值强一百倍！  

（6） 6S：对于南家的6S，大家异口同声：傻蛋！呆瓜！脑子进水了！同伴既然在4C后没扣出4D 或4H，而示弱回到4S，他能有DA或HA？你当他真是“傻样儿”？？？！！！  /cry

五．假设地、较为合理地叫牌过程

（1）        N        E      S        W
            pass     pass    1S      pass
            2C       pass    2D      pass
            2S       pass    4C      pass
            5C       pass    5S      all  pass
说明：虽然因为H是4-1分布，在西家首攻单张H的情况下，5S定约1下，但是平心而论，5S应该是个不错的定约。

（2）   N        E      S      W
       pass     pass    1S     pass
       2C       pass    2D     pass
       2H       pass    4C     pass
      4H/4S     all  pass
说明：上述进程中，南的1S-2D-4C三个叫品清楚的表明他持5-4-4-0牌型，北家没有DA，且知道自己的CACK实在是生错地方了，所以坦然地把定约丢在使北家幽怨、忧闷地4阶上。

六．关于持C套情况下Drury约定叫的补充说明

由于Drury约定叫将2C占用，所以pass  hand 在持9-11hcp、对同伴所叫高花缺乏配合、且有6张比较好的梅花套时，必须设计出2C以外的叫品。

目前常用的叫品有3种：

（1） 在局况和牌型合适的条件下，用6张较好的C套开叫3C；

（2） 在PD一阶高花开叫后，直接叫3C（这种叫品通常是6-3-3-1型，PD的开叫高花为单张）。

由于上述6-3-3-1型牌更倾向于（1）中的直接开叫3C，所以现在的PASS---1H/1S---3C 多用来表示“混合加叫”：梅花为不错的5张以上套, 且对同伴开叫花色有4张支持。

（3） 在PD一阶高花开叫后，先叫2C，假装是Drury问叫；再同伴答叫后，再叫3C。这个叫品应该和（2）中的“混合加叫”搭配使用。

2C勉强了一点，但不算非常过分，如果是4张S则没有任何问题，现在短一个，
使得单张的价值变小了。

2D之后，出2H没有任何意义，在已经知道S配合的情况下，2H应该是值得介绍的
套而不是J10xx这样的东西，叫2H远不如叫3C。北家第一口2C叫得有点强，这时候再叫3C恐怕担心把牌叫的过高，2S应该是非常中规中矩的叫品。

2S过来，试探满惯的最佳叫法应该是5C，排除C的罗马关键张问叫。splinter这时候并不适用，因为只关心同伴在红花色上的A。而且，同伴在持有CA的情况下，虽然知道CA无法增值，但A永远都不会减值，他并不清楚你是空门。

同伴4S之后，直接蹦到6是严重违反叫牌纪律的叫法，而且非常伤害搭档之间的感情，因为明显是对同伴判断的怀疑。

1M-2C后，开叫人：
1、直接回2M：正常的低限开叫，即使应叫人是一般的有限加叫牌，仍然没进局兴趣。但不反对应叫人持有更好的有限加叫牌时做进一步试探，如果应叫人做进一步试探他应该有4张将牌或有不错的边花套，也就是说应叫人应该有一手具备相当弹性或清晰前景牌；
2、平叫2D以外的新花：实叫，最少值得长套邀局的牌（第二非D套的逼局强牌被归入此）；
A、应叫人回3M为示弱；
B、如果应叫人有值得进局的牌，则再应叫平叫NT描述一手缺乏控制的牌，或者再应叫新花描述大牌位置（不一定是A）。
3、2D：D套长套邀局或着轻开叫，应叫人必须再应叫2M。此后开叫人：
A、2NT：4张D套的长套邀局；
B、3D：14-15P、5-5+；
D、PASS：轻开。
4、2NT：D套，5-5+，邀局实力；
5、3D：逼局强牌，16P+、5-5+或18P+、5-4；
6、跳叫新花：自身爆裂叫。

对于本例，俺个人倾向于（应该说肯定会）第一应叫2S――如果非要说应叫2C更适合于本例后续叫牌的展开的话，那么，如果本例中南家处于第一或第二家时，是否应该另起一贴讨论？

贴中最后所指1M-3C为“混合加叫”似乎有误。
混合加叫应该是指1M-3C/3D=7-9P、4支持/9-11P、4支持，贴中所指应该是“配合显示叫”，也可以说是所罗威新花跳应叫的变形――因为应叫人已经实际上不可能持有单套强牌了。
混合加叫和配合显示叫也可以联合使用，有些搭档间是这样约定的：没有任何干扰（也就是OPP开叫的开叫也没有争叫）时，采用混合加叫；反之，采用配合显示叫。
