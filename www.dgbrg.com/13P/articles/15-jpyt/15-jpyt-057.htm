对抗1NT开叫的44个约定叫


一、阿斯普罗 ASPRO
    2C = H+X，2D = S+m  应叫人使用接力
    3C/3D/3H = 罗马跳争叫 6-5/6-6 两套
    2NT = S+C

二、阿斯普特罗 ASPTRO
    2D = S+X，2C = H+X  应叫人使用接力

三、阿斯特罗 ASTRO
    2D = S+X，2C = H+m  应叫人使用接力

四、贝克 Becker
    2C = 双低花，2D = 双高花

五、伯根 Bergen
    同 DONT (Marty Bergen 是 DONT 的发明者)

六、蓝梅花 Blue Club
    2C/D/H/S = 转移
    平衡位置加倍 = 高花单套 或 双低花

七、布罗泽尔 Brozel
    加倍 = 惩罚，有好的单套(5-6张)
    2C = C+H，2D = D+H
    2H = H+S，2S = S+m
    2NT = 双低花
    3X = 短套，三套牌

八、肯斯诺 CANSINO
    2C = 三套牌，C有长度(5-4-4-0，4-4-4-1，5-4-3-1)
    2D = 双高花
    加倍 = 惩罚(弱无将)；单套高花 或 双低花(强无将)

九、卡佩莱蒂 Cappelletti
    2C = 任一单套，2D = 双高花，2M = M+m

十、卡纳佩转移 Canape Transfers
    转移4张套，保证有另外更长的套

十一、克瑞希 CRASH
      2C = 同色两套 Color (C+S/D+H)
      2D = 同级两套 RAnk  (C+D/H+S)
     2NT = 同形两套 SHape (C+H/D+S)
      强无将：加倍代替2NT，显示 odd 套

      Crash 源由 KitWoolsey 针对大梅花体系的约定 发展而来

十二、CRO (同CRASH)

十三、DONT
      加倍 = 单套
      2C = C+X
      2D = D+M
      2H = 双高花
      2S = 单套S(较弱)

十四、超级约定 Super Convention (Exclusion bids)
      2X = 自然，或 排除该花色的其他三套牌
      同伴首先按自然对待如果该花色少于3张，否则依靠判断选择一套叫牌。

十五、盖特斯附属叫 Gates Adjunct
      平衡加倍方法，等等。(见 Amalya Kearse 桥牌竞叫约定)

十六、格兰诺-阿斯特罗 Grano - Astro
      加倍 = S+X
      2C/2D = C/D+H
      由 Matt Granovetter(美国)发展

十七、汉密尔顿 Hamilton
      见 Cappelletti

十八、哈罗 HELLO
      2C = D套 或 M+m， 2D = H套
      2H = 双高花， 2S = S套
      2NT = C套， 3C = 双低花， 3D = 双高花强牌
      加倍 = 惩罚

      由美国的 Helms 和 Cowman 发明，在一些州很流行

十九、凯尔西 Kelsey
      2C = 允许打C定约，红花短套
      2D = 允许打D定约，黑花短套
      同伴(第四家)需要猜测，如果自己有一套或第三家叫牌通常可以解决

二十、兰迪 Landy
      2C = 双高花
      如果持等长的高花可以用2D来让同伴再选一次
      Landy 有许多版本
      一些牌手在第四家持双高花叫加倍，2C为自然叫 

二十一、莱昂内尔 Lionel
        加倍 = 11+HCP，至少4S+4X
        2C/2D = 11+HCP，至少4H+4C/D
        2H = 11+HCP，至少4H+4S
        2S = 自然
        平衡位置同样处理
        第二家加倍后，同伴可以叫pass罚放(需要10+HCP)；
            或叫2C(示选)，或2D(D和H示选)；
            在加倍后直接叫2H为自然，不逼叫。
            直接跳叫3S为阻击，如果持S套有成局兴趣，可以先叫2NT，
            显示无短套，或在3阶叫短套。

        由 Lionel Wright 发展，新西兰

二十二、修正精确阿斯特罗 Modified Pinpoint ASTRO
        2C = C+H， 2D = D+H
        2M = 自然
        加倍 = S+X

二十三、Pottage
        由英国的 Julian Pottage 发展，见 Cappelletti

二十四、Power
        Power 体系将 Brozel 与 RCO 方法揉合在一起。

二十五、RCO
        强无将和第四家：
            2C = 同级两套 Rank
            2D = 同色两套 Color
            X  = 同型两套 Odd
            2M = 自然
        弱无将(除第四家)：
            X  = 惩罚
           2NT = 同型两套 Odd

二十六、RIPSTRA
        2m = 双高花，m较长或较好(可以是5-4-2-2)

二十七、罗斯 Roth
        加倍 = 双高花
        2C = C+S， 2D = D+S
        2M = 自然
       2NT = 4H+6m
        3C = C+H， 3D = D+H

二十八、罗斯--斯通 Roth-Stone
        2C = C+S， 2D = D+S
        3C = C+H， 3D = D+H
        加倍强无将 = 双高花
        加倍弱无将 = 惩罚

二十九、萨哈拉 SAHARA
        2M/3m = 自然
        加倍 = H+m
        2C = S+X
        2D = 一些三套牌
        2NT = 双低花

        美国托莱多市 Gene Vincent 发表于桥牌世界杂志

三十、SCORRCHIO
      加倍 = 单套(非D套)
      2X = 5张套 + 4张对应花色(高花和低花)

      由悉尼的 Trev 和 Eric Kokish 发展

三十一、SHARPLES
        有三种不同的版本，以下是 Terence Reese 版：
        2C = C+另两套
        2D = D+双高花
        2M = 自然

三十二、SOAP
        2C = 5张H， 2D = 5张S
        加倍 = 16+
        2M = 4M+6m
        3X = 中等牌力(通常11-15)

        SOAP 原是由 Paul Marston 设计来针对人工阻击叫和Ferts，
             有些牌手借用来防守1NT开叫。

三十三、SPLASH
        套 Suit 加 Plus 另一 A 高级套 Higher
        (同 DONT 但加倍=惩罚，而不是单套)

三十四、SUCTION
        花色 = 上一级花色 或 另两套花色：
        2C = D 或 H+S
        2D = H 或 C+S
        2H = S 或 C+D
        2S = C 或 D+H

三十五、转移 Transfers
        从1970后就基本上无人使用了。

三十六、TIZI
        加倍 = 单套
        2C = 至少 5-4 低花
        2D = 至少 5-4 高花
        2M = 5M+4m

        由 NSWBA 桥牌教师 Tina Zines 发展
        (用以训练新手升至中级牌手熟悉约定叫的过渡约定)

三十七、TOSS
        加倍 = 单套
        2X = 5张套 和 另一4+张套

        由墨尔本的 Willie Jago 发展

三十八、UNI-Club
        2C = C+X， 2D = D+M
        2M = 自然
        2NT = 三套牌，有S套
        3C  = 三套牌，短套S

三十九、VROOM
        加倍 = H单套，或 H+m
        2C = S单套，或 S+m
        2M = 4M+6m
        2NT = 双低花

四十、 WIND(Westwood 1NT Defence)
       加倍 = 选择性，至少13HCP，可能基于一个好的5张套如KQJ9x
              同伴可以pass罚放，或傀儡至2C
       2C = 双高花 (2D 询问哪门更好)
       2H/2S/3C/3D = 自然，但合适加倍(即间张套)
       2NT = 双低花
       2D = M+m，好的低花套，4张或弱的5张高花
            由于2D是逼叫，也适用于强M+m套

       由悉尼的 Bill Westwood 发展
四十一、WINDA(WIND的另版)
        加倍 = 双高花，2NT = 双低花
        2C = C+M， 2D = D+M
        2H/2S/3C/3D = 自然

四十二、WONT(Wreck Opponents NoTrump)
        加倍 = 单套
        2C = C+M， 2D = D+H
        2H = 双高花，2S = D+S
        2NT = 5-5以上双低(其他的双套可能是5-4)

        由悉尼的 Rakesh Kumar 发展

四十三、无名1
        由 Kirmse-Marsal 使用，在1993年英国杂志发表：
        2C = 单套C或D；S+m；双高花

四十四、无名2
        2C = D套；或 M+m，低花不会比高花长
        2H = 双高花
        2S = S套
        2NT = C套
        3C = 双低花
        3D = 进局逼叫，双套牌(非双低)
        3M = 阻击
        3NT = 进局逼叫，双低花
