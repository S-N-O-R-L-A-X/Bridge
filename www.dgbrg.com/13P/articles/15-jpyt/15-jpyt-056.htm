对抗强1C开叫的21个约定叫


 1、最古老的就是 Mathe 约定：
    Dbl = 双高
    1NT = 双低

 2、Dbl-1S = 转移
    1NT = 双低
    2m = m+H 5-5
    2M = 弱二
    2NT = 阻击
    3X = 要打

 3、2C = H+any
    2D = S+m
    1NT = 双低
    Dbl = 竞叫，2C逼叫

 4、1NT = S+any
    2C  = H+C
    2D  = H+D
    2NT = C+D

 5、Dbl = C+M
    1NT = 无C两套
    2NT = 双低

 6、1NT = 双低
     2C = m+M
     2D = 双高
    Dbl = 三套(可能5431)

 7、Dbl = C
    1NT = m+M 或 4441
     2C = 双低
     2D = 其他两套

 8、Dbl = H
     1D = S
     1H = H+C or S+D(同顶)
     1S = S+C or H+D(同色)
    1NT = H+S or C+D(同级)

 9、SUCTION
    Dbl = D or H+S
     1D = H or S+C
     1H = S or C+D
     1S = C or D+H
    1NT = C+H or D+S
    (规律：叫品转移单套和该单套下两级花色双套)
            1NT 显示非贴邻两套

 10、Dbl = C or D+H
     2C = D or H+S
     2D = H or S+C
     2H = S or C+D
     2S = S+D
    2NT = 强双套

11、Dbl = 4-4M 或 强牌
     1D = H or S+C
     1H = S or C+D
     1S = C or D+H
    1NT = S+D or H+C
     2C = D or H+S
     2D = 6H or S+C(55+以下同)
     2H = S or C+D
     2S = C or D+H
    2NT = S+D or H+C

12、 1D = 双高
    1NT = 双低
     2D = 单套高花
     1M = 4M

13、Dbl = H+S or C+D(同级)
     1D = H+D or S+C(同色)
    1NT = 6C or 5D+5M
     2C = 6D or 5C+5D
     2D = 6H or 5S+5m
     2H = 5H+5S
    2NT = 两套

14、Dbl = 双高
    1NT = 双低
     1D = D+H
     1H = S+C
     1S = H+C or S+D

15、CRASH
    Dbl = S+C or H+D(同色)
     1D = H+S or C+D(同级)
    1NT = S+D or H+C(同形)
    1C-1D后用Dbl/1NT/2C

16、Dbl = 好牌
    1NT = 5C+4M
     2D = 5M+5other
    2NT = 5C+5D
   
17、Dbl = 双高
    1NT = 双低
     1D = D or S
     1H = H or C
     1S = 3套牌

18、Yeslek
    Dbl = D or H+S
     1D = H or S+C
     1H = S or C+D
     1S = C or D+H
    1NT = C+H or D+S(对波兰1C=15-18NT)
     2C = D or H+S
     2D = H or S+C
     2H = S or C+D
     2S = C or D+H
    2NT = C+H or D+S
     3X = AQJ10xx
    强牌先pass(实际上就是两轮的SUCTION)

19、Dbl = D or H+S
     1D = H
     1H = S or C+D
     1S = 4+S-5+m
    1NT = 6X
     2C = 4+H-5+C
     2D = 4+H-5+D

20、Dbl = 套/首攻
    1NT = 双高
     2X = SUCTION

21、Dbl = 双高 或 好牌
    1NT = S+D
     2D = D+H
     2H = H+S
     2S = S+C
    2NT = C+D
     3C = C+H
