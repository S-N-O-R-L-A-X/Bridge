§2、迫伴赌倍后之发展

　　在上节我们已稍微提到作迫伴赌倍，必须对其他未叫花色有好的支持，这里我们给你个更明确的定义：你对所有未叫花色，每门至少要有3张支持方可作迫伴赌倍。即是说：当你赌倍敌方的1H开叫，你黑桃、方块和梅花三门不可有缺门、单张或双张的情形。（而且，通常你应该要有4张而非只是3张黑桃的。因为你这个迫伴赌倍是有很强烈要求同伴叫出未叫的那门高花的；而且，由我们下面所述，你会发现，你的同伴将会最优先选择这样作的。）

　　当然，你不可能是只有牌型而毫无牌点就可作迫伴赌倍的。由于你是预计同伴最少会有一门牌组和你配合的－－通常也应该是这样的；因此，这时你计算点力可将分配点也算进去。分配点之计算法：缺门加5点，单张加3点，双张加1点。当然这个短门只以敌叫组为限。但是，如你有敌叫组的Q、J这种点力也是不应该算进去的。那到底要有多少点才可作迫伴赌倍呢？假如被你迫叫的同伴能够在一阶回答时，只要11点你就可作迫伴赌倍。即是说，只有11点时，你只能赌倍敌方1H以下的开叫。但当你要迫同伴在二阶回答时（你赌倍1S或更高的合约），最少必须再多2个点－－13点才可。以后每抬高一阶必须再多加2个点。

　　迫伴赌倍是个迫叫的叫品，假如你的上家并不叫牌，不管你是再弱的牌，你都不可以派司这个赌倍的。你力量愈少，敌方这个被赌倍的合约得到的分数愈多。你必须明了：你的同伴是很可能没有防御能力的；他敌叫组愈短，他是愈像会赌倍的。

　　当然，假如同伴是赌倍敌方的开叫1H，而你持有：
　　S8　HKQJ1082　DK82　C965
你可以将这个赌倍派司过去当作处罚，此谓之罚放。但是，一定要是像这个例子那样好的牌才可以罚放－－你的王牌必须要坚强到可以由你来抽敌方王牌的程度。假如你是持了：
　　SJ87　HQ10852　D963　C84
绝不要考虑罚放同伴赌倍的1H。虽然你有5张红心，而且也无4张的牌组可以回答；但是，你的红心是那么破碎，力量又是那么少。你应该回答1S－－最便宜的3张花色。

　　由于你纯粹是被迫要回答同伴的迫伴赌倍的，你这个被迫的答叫并不保证有任何力量的。比方你持：
　　S9865　H876　D963　C432
对着同伴的迫伴赌倍，你也是要回答1S的。因此，赌倍的人必须明了，你的同伴这种最经济的回答可能就是持类似上面这付牌的。所谓最经济的回答是这种：
　　开叫人　　同伴　　答叫人　　你
　　1C　　　　X　　　　－－　　1D、1H、1S
非跳叫的回答，同样是称为消极性答叫－－就像同伴开叫1C，你回答1D一样，你是可以什么都没有的。通常当你是8点以下时，你必须作消极答叫；而有9点以上时，则需作积极答叫－－跳叫。当力量更多－－有12点以上时，你们是很可能有一局的，那就必须示叫敌叫组，像这样：
　　开叫人　　同伴　　答叫人　　你
　　1H　　　　X　　　　－－　　2H
这种示叫敌组并不表示在敌方这门花色有什么玩意；你并不保证这门有第一或第二甚或第三圈的控制。你只是在告诉同伴你们可能在那儿有一局而已。比方像上面这样的叫牌过程，你持了：
　　SKQ87　H985　DAQ1084　C7
这种价值15点的牌（单张加3点，长门第5张起各加1点），你不能只叫个不迫叫的2S或3D。你先示叫敌组迫叫一圈，如同伴叫出2S，你可直接跳4S；如同伴叫出3C，你可再叫3D。必须注意的是：这个3D仍旧是迫叫的（这个在本节最后会详细说明）。

　　持了下列诸牌，你应该怎么叫？假定叫牌过程都是这样：
　　开叫人　　　同伴　　　答叫人　　　你
　　1D　　　　　X　　　　　－－　　　　？

    1. SQ863　HKQ94　DQ7　C852

    2. SK7　HJ9643　D87　CA865

    3. S10642　H852　D8　CQJ843

    4. S8　HQJ65　DJ84　CK10653

    5. S97　H64　D108543　CJ842

    6. SKQJ4　HAJ84 D86 C654

    7. SQJ108752 H6　D943　CJ4

     

    1. 叫1S；你有7个点（敌叫组的Q应不算点），当作是消极答叫中的高限。当然叫1H也未尝不可；但是你准备在可能的情形下再抢叫2H，比方叫牌如发展成：
      　　1D　　X　　 －　　1S
      　　2C　　－　　2D　　2H
      这是可让同伴在两门高花中作一选择。假如第一次你是叫1H，这是就非得叫2S了，而同伴欲选择红心时就被逼上三阶了。

    2. 跳叫2H；你全手牌价值10个点--8个大牌点，加上长门红心及短门黑桃各加一点。必须作积极答叫。

    3. 叫1S；假如你们有一局，或是可以抢到一个部分合约，通常比较像是较高的花色。甚至就是有6张的低花，对着同伴的迫伴赌倍，都宁愿叫只有4张的高花的。

    4. 跳叫2H；由于牌型的极度配合，这付牌比表面上看起来要强壮的多。加上分配点，你的牌有10个点的价值，所以必须跳叫。而且不要为了只有4张红心而不敢跳叫；同伴最少有3张而大致是有4张支持的。事实上，是你支持他的红心而不是要他支持你的；而有4 张支持是不用害羞的，尽管跳无妨。

    5. 叫2C；只有这么少的力量和这么烂的方块，绝不可派司同伴的赌倍或是叫1无王。虽然只有3点，但你并没有选择的余地。叫2C仍是消极答叫，同伴不会把你当作有多少东西的。

    6. 叫2D；有12个点，只跳叫的话是太强壮了；所以必须示叫敌组才可。

    7. 跳叫3S；这是比较特殊的情形，你并没有多少大牌点，但有一长串黑桃。虽不够跳叫2S的力量，但只叫1S又不太合适。像这种牌可以多跳一阶叫3S。

　　到目前为止，我们都是假定在你同伴赌倍后，开叫人的同伴都是派司的。假如不是这样，答叫人他再赌倍，叫一门新花或是支持他的同伴时，你对这个赌倍的答叫就完全不同了。最少，你不是再被迫非叫牌不可的；而且，他这一叫说不定已用掉了你本来可以自然答叫的空间。

　　假如叫牌过程是这样子的：
　　开叫人　　同伴　　答叫人　　　你
　　1C　　　　X　　　1D、1H、XX　　？

　　这时，你假如是持很弱的牌，你便可以派司；但假如你有6个点以上或是有个长门时，尽量设法叫牌。例如你持：
　　SKJ86　H83　DQ942　C853
　　SQJ1063　H92　D8643　C96
在上面那种过程中，你都可以叫1S。

　　但是当你持了9－11个点时，你仍旧必须跳叫，一个简单的自由叫牌并不能满足有这么多的力量。假定你拿：　　SA10854　H96　DK1052　C85
你就必须这样叫：
　　开叫人　　同伴　　答叫人　　你
　　1C　　　　X　　　　1H　　　2S
假如没有DK，你就应该叫1S。

　　在同伴赌倍之后，这个答叫人是很像会叫的很高来剥夺你们的叫牌空间，引起你们一点麻烦。比方这样：
　　开叫人　　同伴　　答叫人　　你
　　1C　　　　X　　　　2C　　　2S
这种情况就分不清楚到底是你本来就是要跳叫2S的牌或是仅要叫1S的牌；这种过程，你持下面二牌同样都是这样叫的：
　　（一）SKJ10863　H86　D954　C75
　　（二）SK1084 H93 DKQ84 C853

　　因此，我们有个很简单的原则来处理这种情况：你必须有多余2个点的力量才可以在比你应该被迫叫牌那阶高一阶中叫牌。例如：一阶自由答叫需要有5－8点，所以当你要在二阶作非跳叫的自由叫牌，你就必须有7－10点。同样的，你必须有9－11点方可跳叫至二阶，所以当敌方的叫牌逼迫你非跳至三阶不可时，你必须有11－13个点的力量。例如：
　　开叫人　　同伴　　答叫人　　你
　　1H　　　　X　　　　3H　　　　？
这种过程，你拿：
　　SKJ86　H86　D963　CQJ85
你就必须派司，假如敌方只是叫到2H，你就可以很舒服地叫2S，但仅有7点是不够在三阶自由答叫的，而拿：
　　SKJ86　H84　D963　CKQJ8
就应该叫3S；假如没有插叫，你是要跳叫的。但仅有11点还够跳至四阶，但尽可在三阶自由叫牌。同样的过程，你持：
　　SKJ86　H84　DK96　CKQJ8
就应该叫4S。

　　我们说过，迫伴赌倍主要的凭藉就是牌型的分配。你的同伴作迫伴赌倍时，就是希望听到你能叫出一门牌组来，绝不希望听到你回答的是无王。因此，对着同伴的赌倍，你绝不要因为持了弱牌且无牌组可叫就回答无王。假如同伴赌倍敌方的1S，你拿SQ1082　H865　D954　C1054，叫2C，绝不是1NT。无王合约是最容易被敌方处罚的－－敌方只要凭大牌力量而不需要有什么特别的门组就可以处罚你的。因此，持弱牌时绝不要回答无王；对着同伴的迫伴赌倍，你回答无王乃是强叫。通常应是9点以上的平均牌型，而对敌叫组有两次挡张。当同伴赌倍敌方的1D，持这手牌：SQ85　H965　DKJ106　CQJ5就可以回答1NT，表示有9－11个点。

　　当有12－14点时，可以跳叫2NT；如力量更多时，则可跳叫3NT或示叫敌组。

　　有时虽非平均牌型，但长门是低花而敌叫组有挡时亦可叫无王。例如同伴赌倍敌方的1H，你拿：
　　S86　HK96　D863　CKQ1084
可以叫1NT，而拿着：
　　S86　HK96　DQ84　CAQ1084
就应该跳叫2NT。必须注意的是：你叫无王并不保证每门都有挡张，但敌叫组是一定要有挡张的。可是，当有4张高花牌组时，是应该优先选择高花而非无王的。

　　由于同伴对你的迫伴赌倍的回答是非常的明确，因此你的再叫是很容易的。假如成局是绝不可能的，你就派司；有可能但不确定时，你就再叫；一定有一局时，你就直接把它叫掉。

　　假如同伴是消极答叫，他最多只有8点。当你少于16点时，你们是不可能有一局的，当你有17点以上时，并不否定有一局可能，你应该再叫，17－19点时，加一支持，20－22点时跳叫支持。叫牌过程是这样的：
　　开叫人　　你　　答叫人　　同伴
　　1D　　　　X　　　－　　　　1S
　　－　　　　？
持下列诸牌，你应叫什么？

  1. SAQ65　HK54　D86　CAJ95
    派司；同伴是少于9点的，而你只有15点，你们不可能有一局的。就算开叫人再叫2D，你也应该派司。

  2. SKQ65　HA107　D8　CKQ1064
    2S；你有17点，所以一局是有可能的。但是，假如你的同伴再以3S邀请时，你还是应该派司的，同伴一定就是6－7点－－假如有8点，他应该直接叫4S的。你是最低限的17点，当然不能接受邀请。

  3. SKQ85　HAK7　D8　CAQ1064
    跳叫3S；当然假如同伴是什么玩意都没有，3S很可能都完成不了的。但是，只要他有一点力量，诸如5张黑桃，CK或有个单张之类，他是会叫上而且完成一局的。

  4. SAQ8　HKQ1064　D84　CAQ6
    叫2H；你的牌如第一圈只盖叫1H是嫌太强了。这种牌都必须先经赌倍再将牌组叫出。

  5. SAQ8　HAKJ5　D86　CAK108
    再叫2D；虽有这么强的力量，但方块无挡不能叫无王；黑桃又只有3张，不能支持同伴；将1S派司过去又可能会丢掉一局。因此，再叫2D迫同伴再叫一次牌。这时，你的同伴应优先叫出有5张黑桃。

  6. SA8　HAKQ1086　D7　CKQ104
    跳叫3H；这牌当然只要同伴有一点力量支持就有一局的，你不能只叫2H；先赌倍后再跳叫就表示是这种牌。当然，这个3H不是迫叫，同伴真的没有东西还是可以派司的。所以，你要是持了比这个更好的牌，应该直接跳成局或是以2D再迫叫一圈。

　　当你的同伴对你的迫伴赌倍是积极性的答叫－－跳叫，他是有9－11点的。假如你是少于13点，便应该派司。假如是14或15点，你们就可能但非一定有一局，你就可以邀请。假如你有16点以上，你们便一定有一局了；如已找到王牌配合便可直跳成局，否则你可叫出自己的牌组。这个在积极答叫以后，赌倍的人再叫出新花仍是迫叫的。

　　你赌倍敌方开叫的1D，同伴跳叫2H，持下列诸牌，你应该再叫什么？

  1. SAJ85　HKQ94　D84　CJ106
    派司；同伴最多只有11点，你也仅有12点，不会有一局的。

  2. SKQ5　HK1084　D7　CK10865
    3H；加上分配点你有14个点，假如同伴是高限的11点，你们是可能有一局的。

  3. SAQ963　HKQ4　D4　CK1064
    2S：你知道你们是已经足够一局了，但只有3张红心，不可支持可能仅有4张的同伴。叫出自己的5张牌组，看同伴能否配合或是叫出有5张红心。

  4. SAK864　HKQ93　D8　CQ95
    4H；既然红心已有很好的配合，你就不用再叫出你的5张黑桃了。所以应直接跳至4H。

　　假如对着你的迫伴赌倍时，同伴是以示叫敌组表示有12点以上，反逼来迫叫你一圈；这时，你必须叫出你5张的牌组，或是叫出最经济的4张花色。同伴之任何再叫，除了是支持你的牌组外，都是迫叫。例如：
　　开叫人　　你　　答叫人　　同伴
　　1H　　　　X　　　－　　　　2H
　　－　　　　2S　　　－　　　 3S　
你的同伴有12或13点；只跳叫嫌太强，但是在你的2S之后还不够直跳4S的力量。所以，除非你是最坏的11点，否则都应该叫上4S。
