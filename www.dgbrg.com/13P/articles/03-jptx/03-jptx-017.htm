示攻性加倍

　

在打牌中，一个准确的首攻是对方定约打宕；一个错误的首攻使不可能作成的定约打成。这些事情不胜枚举。在你拿了使对方定约被宕的关键张时，使同伴不至于攻错，这时首功指示性加倍就能发挥奇效。

1、对满贯定约的加倍

对方叫到满贯定约，通常都有30点左右。你们如果打宕对方定约，可以赢十几点。如果把这时的加倍当作惩罚性加倍，无非只能增加1-2点，并且可能指示对方打牌，得不偿失。因此这时的加倍肯定是指示首攻的加倍。
对于对方的无将满贯定约，通常要求同伴首攻明手叫过的第一门花色。
对于有将定约，你通常可以根据自己手上的牌型来判断。
如果对方在将牌之外只叫过一门花色，就出这一门。
如果你发现牌型成奇型，那一般同伴是缺门。要求将吃。
如果同伴不是缺门，他可能持有一门旁门花色的A。

2、虚叫的加倍

对于对方的一切虚叫的加倍，都是有惩罚性的首攻指示性加倍。例如STAYMAN，JACOBY TRANSFER，BLACKWOOD，GERBER等。牌例如下。

N      E      S      W
1NT    /      2C     X
表示W有强C套，可开叫实力，如对方定约要求首攻C

N      E      S      W
1H     /      3H     /
4NT    /      5D     X
表示W有较强的D，一定实力，如对方定约要求首攻C。

N      E      S      W
1C     /      1D     X
表示W有强D套，开叫实力，如对方定约要求首攻D。

N      E      S      W
1S     /      2D     /
2H     /      3C     X
这时S的3C为第四花色，表示如N的C有止叫3NT，W叫X可能持有C；KQJXX，旁门有1A，如对方叫3NT，要求同伴首攻C。

3、无将的加倍
N      E      S      W
1C     /      1S     /
2NT    /      /      X
这时W可能持有S：AQJXX 开叫以上实力，要求同伴首攻S。

N      E      S      W
1NT    /      3NT    X
W至少持有一门KQJT10X，另一门A。

N      E      S      W
1C     /      1S     /
1NT    /      3NT    X
要求同伴首攻S。

N      E      S      W
1C     /      1H     1S
1NT    /      3NT    X
W要求同伴首攻自己叫过的花色S。

N      E      S      W
1C     1S     1NT    2D
2NT    /      3NT    X
W要求同伴首攻自己叫过的花色S 小，W手中有大牌，不超过3张支持。

当然，还有一些情况同伴之间约定好了，不能误解。
