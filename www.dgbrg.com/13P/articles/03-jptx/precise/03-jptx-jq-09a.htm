精确体系摘要速查

第九章  2C开叫后的叫牌

2C 开 叫

精确法的2C开叫表示11―15点，C为5张以上的花色。如果C花色只有5张，则必须还要附带有一个4张的高级花色，才能开叫2C。下面举一些例子来说明2C开叫的要求条件：

S6  HA764  DK104  CAQJ96，开叫2C。此牌有14点，一个好的C花色，并另有一4张的高级花色。

SA85  H7  D976  CAKJ743，开叫2C。此牌的C花色为一极好的6张套，完全足以开叫2C。

SA73  H62  DK83  CAQJ84，开叫1NT。持5-3-3-2牌型及13-15点，且5张套是低级花色时，以开叫1NT更为合适，因为这样能把持牌情况显示得更为清楚。

S87  HA6  DQ1032  CAK642，开叫1D。此牌只有5张C，又无4张的高级花色，最好留出更多叫牌时空，以便探查出最佳的定约。

SK7  HAQ76  D62  CQ8642，不叫。此牌的C花色太弱，不够开叫2C的要求，而开叫1H又需要至少1张。

对2C开叫的非逼叫性应叫

当你的同伴开叫2C时，如果你只有0-7点，则通常你都应该不叫。成局已经无望，为了想找一个较好的不成局定约而去叫牌，很容易造成危险，而受到巨大损失。

在持8点以上的牌力时，你可以有下列各种非逼叫性的应叫：

1)   2S或2H＝8-10大牌点，所叫花色为5张以上的长套。开叫人在持低限牌（11-13大牌点）时，若对此花色尚略有支持就应停叫，只当在以此花色作为将牌会有极大危险的情况下，他才可以再叫3C。开叫人若持高限牌（14-15大牌点），则对此花色有3张以上的支持时，他就可以加1副作为进局邀叫；否则就叫2NT来作为进局邀叫。

2)   3C＝8-10大牌点，对C花色有好的3张或更佳的支持，但没有5张以上的高级花色。这时开叫人若持低限牌，就应当不叫；若持高限牌，则叫3NT或另叫出一新花色。

3)   2NT＝11-12大牌点，平均牌型，并且没有4张以上的高级花色。这时开叫人若持均型低限牌，就应当不叫；若持非均型低限牌，且C为6张以上长套时，就叫回3C，作为止叫。开叫人若持高限牌，则叫3NT或另叫出一新花色。

关于2C开叫后的上述几种应叫，下面用一些例子来加以说明：

SQ8643  HQ42  DJ32  C86，不叫。持这类弱牌时，切勿为了想找一个较好的不成局的定约而去叫牌。

S872  HAQ1062  DKJ3  C74，叫2H。表示8-10大牌点H为5张以上长套。如果同伴作出3C示弱止叫，你就停叫；但当同伴作出任何进局邀叫（如3H或2NT再叫）时，你均予以接受而加叫成局（因此牌为高限10点）。

SAQJ874  H743  D10874  C―，叫2S。你只有7点，但你的S花色长套极好，而C花色缺门又十分危险，故宜于叫成2S。这时无论同伴做出任何进局邀叫，均应予以拒绝。

S64  HAJ8  DK763  CJ1065，加叫成3C。表示8-10点，并有好的C支持。

SK98  HK107  DAJ86  C432，叫2NT。表示11-12大牌点，平均牌型，并且没有4张以上的高级花色。

对2C开叫的强应叫

应叫人跳叫新花色

在同伴开叫2C时，你如果持有14以上的大牌点，并有一个好的5张以上的新花色，则可跳叫此新花色（即时同伴只有低限11点，也完全符合叫到成局的牌力要求）。例如，在同伴开叫2C时，下列各牌的应叫分别为：

S76  HAKJ64  DAK104  CQ2，跳叫3H。

SAKQ1085  H72  DAJ4  C65，跳叫3S。

在应叫人跳叫新花色之后，开叫人若对此花色的支持差，就应当叫3NT。如果对此花色有3张以上的支持时，开叫人若持低牌限（11-13大牌点）就应当加1副，若持高限牌（14-15大牌点）就可以在最低的叫牌水平上扣叫出自已的A。

2D约定性应叫

在2C开叫后的2D应叫，是一种约定性应叫，且为逼叫。2D应叫表示11以上大牌点，并要求同伴进一步叫明其持牌情况。（当应叫人对C花色有较好配合时，大牌点数的要求可略为降低。）这时开叫人按下述方式进行答叫：

1)  2H或2S＝4张以上花色，11-13大牌点。

2)  2NT＝C为6张的长套，并在其余三门花色中两门花色有止张，11-13大牌点。

3)  3C＝C为6张的长套，并在其余三门花色中只有一门有止张，11-13大牌点。

4)  2D＝低级花色两色套，至少6张C和5张D。

5)  3H或3S＝4张以上花色，14-15大牌点。

6)  3NT＝C为坚固的6张套，14-15大牌点。

7)  4C＝C为很长的间张套，14-15大牌点。

通过上述各种再叫，一般能够把开叫人的持牌情况显示得极为清楚，因而应叫人在再次叫牌时就能够方便地抉择出最后定约。但有时候，应叫人有可能还需要进一步了解更多的情况：

1)   若开叫人的再叫为2NT，则应叫人可叫3D（约定叫），用以询问开叫人在哪两门副牌花色中有止张。开叫人按以下方式答叫：

3H＝H和D花色中有止张。
3S＝S和D花色中有止张。
3NT＝S和H花色中有止张。

2)   若开叫人的再叫为3C，表示只有一门副牌花色中有止张，则应叫人可叫3D（约定叫），用以询问是哪一门花色。开叫人按以下方式答问：

3H＝H花色有止张。
3S＝S花色有止张。
3NT＝D花色有止张。

下面举一些例子来说明2D约定性应叫的实际应用：

开叫人: SA85  H7  D976  CAKJ743            2C / 3C / 3S / Pass

应叫人: S632  HAJ92  DAQ8  CQ62            2D / 3D / 3NT

在此例中于2C开叫之后，应叫人需要了解更多的情况，才能据以判断在H、D或NT中何者为最佳的定约。于是他叫出2D约定性应叫来询问开叫人的牌情。开叫人答叫3C，表示C为6张套，在一门副牌花色中有止张，且为低限牌。应叫人由于在S花色中没有止张，所以又叫出3D来询问开叫人在哪一门花色中有止张。在知道了开叫人的S花色中有止张时，应叫人立即叫成3NT进局。（若开叫人是在一门红花色中有止张时，应叫人就只能叫回4C。）

开叫人: SK7  HAJ83  D8  CKJ10864            2C / 2H / Pass

应叫人: SAQ83  HKQ65  DK94  C95            2D / 4H

在此例中，2D约定性应叫使应叫人得以找到H花色的4-4配合；即使在开叫人已经显示低限牌力的情况下，应叫人还是有足够的牌力直接加叫到成局。

开叫人: SKJ3  HKJ5  D2  CKJ10874            2C / 2NT / 3NT

应叫人: S652  H1042  DAKJ3  CAQ6            2D / 3D / Pass

在此例中，开叫人的2NT再叫表示C为6张，在两门副牌花色中有止张；而应叫人则对究竞在哪两门花色中有止张极为关心。若是应叫人叫出3D来作进一步询问，开叫人答叫3NT表示在S和H花色中有止张，这正是应叫人所希望的。

开叫人: SKJ3  HKJ3  D2  CKJ10874            2C / 2NT / 3NT / Pass

应叫人: SAQ4  H1053  D763  CAQ65            2D / 3D  / 4C

在此例中，应叫人在叫出3D之后，希望开叫人能够答叫出3H，以表示在两门红花色中有止张。但未能如愿，开叫人实际答叫出3NT，这时应叫人明智地看到D花色在做无将定约时会造成灾难，因而适时地叫回4C。

开叫人: S53  HAKJ3  D62  CAQ864            2C / 3H / Pass

应叫人: SQJ104  H98  DAJ9  CK732            2D / 3NT

在此例中，应叫人只有11点，所以他准备在开叫人持低限牌时停止于不进局的定约上。但是开叫人却跳叫3H，表示有一4张以上的H花色，且为高限牌，于是应叫人正确地叫成3NT。

当对方干扰时

如果在2C开叫之后，对方跟着叫出加倍，则这时应叫人叫再加倍表示11以上大牌点，且在其它三门未叫过的花色中均有牌力。同时是强烈地向开叫人建议，如果对方逃叫时予以惩罚性加倍。其它各种应叫，包括2D约定性应叫，均保持原有的意义。

如果在2C开叫之后，对方跟着进行争叫，则这时应叫人在3副水平以上叫出的加倍为惩罚性加倍；而在2副水平上，则是所谓示弱加倍，希望开叫人能够叫出一4张以上的高级花色。如果对方是在2副水平上作出的争叫，则这时应叫人的2NT应叫就代替2D约定性应叫。这时开叫人只要是持低限牌力（11-13大牌点），均答叫3C；开叫人若持高限牌力，则仍按原来方式答叫，但3副水平上扣叫对方所叫花色时，只表示在这个花色中有止张，而并不一定是4张的花色。


本 章 摘 要

2C开叫＝11-15大牌点，C至少为好的5张套，如果只有5张C，则必需还有一个4张的高级花色。

1)   应叫人不叫＝0-7大牌点。

2)   应叫人应叫2D＝约定性应叫，逼叫。至少11大牌点；要求开叫人进一步叫明牌情。对2D约定性应叫，开叫人的再叫如下：

2H或2S＝4张以上花色，11-13大牌点。

2NT＝C为6张套，在另外三门副牌花色中有两门花色有止张，11-13大牌点。这时应叫人可叫3D，用以询问在哪两门花色中有止张。开叫人再按下述方式答叫：

3H＝H和D花色有止张。
3S＝S和D花色有止张。
3NT＝S或H花色有止张。

3C＝C为6张套，在一门副牌花色中有止张，11-13大牌点。这时应叫人可叫3D，用以询问在哪一门花色中有止张。开叫人再按下述方式答叫：

3H＝H花色有止张。
3S＝S花色有止张。
3NT＝D花色有止张。
3D＝低级花色两色套（至少为C6张、D5张）。
3H或3S＝4张以上花色，14-15大牌点。
3NT＝C为坚固6张套，14-15大牌点。
4C＝C为很长的间张套，14-15大牌点。

3)   应叫人应叫2H或2S＝8-10大牌点，5张以上的花色，非逼叫。这时开叫人可有以下各种再叫：

不叫。
3C＝止叫。不宜以应叫人的花色作为将牌。
加1副＝进局邀叫，至少3张以上将牌支持。
2NT＝进局邀叫，对应叫人的花色支持差。

4)   应叫人应叫2NT＝11-12大牌点，平均牌型，没有4张以上的高级花色，非逼叫。这时开叫人可以有以下各种再叫：

不叫。
3C＝止叫。不宜做无将定约。
新花色＝高限牌，逼叫。
3NT＝高限牌，止叫。

5)   应叫人应叫3C＝8-10大牌点，好的3张或更佳的支持。没有5张以上的高级花色。非逼叫。

6)   应叫人跳叫新花色＝14以上大牌点，好的5张以上的花色，逼叫。这时开叫人可以有以下各种再叫：

3NT＝对应叫人花色支持差（0-2张）的任何牌。
加1副＝对应叫人的花色有3张以上的支持，低限牌。
新花色＝高限牌，对应叫人的花色有3张以上支持，所叫花色有A。

复习测验题

第一部分

在下列各题中你如何开叫？

 1)  SAQ86  H1063  D7  CKQJ95，开叫2C。你此牌符合2C开叫的所有要求条件：12大牌点，C为好的5张套，并另有一4张的高级花色。

 2)  SKJ74  HQ3  DQ4  CK8643，不叫。C花色太弱，不够在2副水平上开叫的要求条件。此外，两个红花色的大牌都没有护张，还是不叫更为稳妥。

 3)  SK3  HQ103  DK83  CAQJ87，开叫1NT。当牌型为5-3-3-2，牌力13-15大牌点，且5张套为低级花色时，最好的选择就是开叫1NT。

 4)  SKQ532  H7  DA  CKJ8643，开叫2C。在持有两色套牌时，开叫较长的一个花色。

 5)  SAK  H87  DAJ86  CQ10752，开叫1C。此例不开叫2C有两个理由：C花色太弱，并且C花色只有5张而又没有一4张的高级花色。

第二部分

在下列各题中同伴均开叫2C，你如何应叫？

 6)  SA64  H72  DK832  CQ1084，叫3C。表示8-10大牌点，并对C有好的支持。

 7)  SAQ865  H7  DAQ3  C532，叫2D。若叫2S，则只能表示8-10大牌点；若叫3S，则又表示14以上大牌点。

 8)  SKQ10862  H7  DA53  C642，叫2S。表示8-10点，S为5张以上长套。

 9)  SA98  HAKQ92  DAQ3  C62，叫3H。表示至少14大牌点，H至少是好的5张套。

10)  SQ63  HK10874  D862  C95，不叫。持很弱的牌时最好不要自找麻烦。

11)  SKQ82  HA973  D862  CK3，叫2D。你需要了解更多的情况。例如，开叫人是否有一个4张的高级花色，他的牌力是低限还是高限。

12)  SK83  HAQ7  DJ1086  CQ32，叫2NT。表示11-12大牌点，平均牌型，没有4张以上的高级花色。

第三部分

在下列各题中你均开叫2C，同伴应叫2D。你如何再叫？

13)  S2  HKQ3  DA107  CQ109832，叫2NT。表示低限牌，C为6张套，并且在两门副牌花色中有止张。如果这时同伴又叫3D，你就答叫3H，以显示在H和D花色中有止张。

14)  SQ1042  HK83  D7  CAQJ62，叫2S。表示低限牌，S为4张套。

15)  SAKJ3  H6  D52  CAQ10865，叫3S。表示高限牌，S为4张套。
