精确体系摘要速查

第八章  1D开叫后的叫牌

1D 开 叫

精确法的1D开叫表示11-15点的，以及一个4张以上的D花色。例如：

SK863  HK1087  DAQ62  C3，开叫1D。开叫高级花色时，所叫花色必须至少5张。

S3  HAJ1097  DAQ9864  C5，开叫1D。在1副水平上开叫最长的花色。

S4  HAK3  DAJ105  CQ7643，开叫1D。自然实叫性的C花色开叫必须叫2C，所以在C花色较弱时，一般总是用其他可行的开叫办法来代替2C开叫。

0-15点的牌对1D开叫的应叫

应叫人不叫

在前一章中已经看到，在精确法的限制性开叫之后，应叫人若持弱牌，是没有必要努力设法应叫的。如果同伴开叫1D，则你在持0-7点时，一般均应当不应叫。例如：

S8632  H7542  D32  CQ83，不叫。

SQ8632  HQ32  D865  CQ10，不叫。成局根本无望，如果仅仅为了想找到一个较好一些的定约而去叫牌的话，很可能造成严重的损失，所以还是不叫为宜。

高级花色应叫

低级花色成局定约要求赢得11墩牌，高级花色成局定约只要求赢得10墩牌，而无将成局定约则只要求赢得9墩牌。因此，在同伴开叫高级花色时，你若有足够的将牌支持（以及足够的牌点）时，应尽快地支持同伴的高级花色；反之，在同伴开叫低级花色1D时，对D花色的支持就需格外审慎。

最好的办法，就是如果你有一好的高级花色的话，尽先把这个高级花色应叫出来。在精确法中，在1D开叫后的1H或1S应叫，表示8-15点，所以花色至少4张，且为逼叫。例如，当同伴开叫1D时，下列各牌的应叫分别为：

SQ1073  H7  DKQ62  CJ842，叫1S。

SK1076  HQ1075  DA3  CA74，叫1H。当有两个4张的高级花色时，应当叫1H，留出叫牌时空，以便同伴有可能在1副水平上叫出S。

SAQ1075  HAJ763  D6  C32，叫1S。当有两个5张（或两个6张）的高级花色时，应当先叫S。这样你在下轮叫出H之后，同伴如果对H不配合而愿选择S的话，他就可以在最低副数水平上叫回H。

SKQ86  H72  D63  CAQJ63，叫1S。在精确法中，对1D开叫的2C应叫，通常表示没有4张以上的高级花色。

在1D开叫和1副高级花色应叫之后，继续的叫牌均为直接了当的自然实叫。如果你是开叫人，并持低限1D开叫牌（11-13大牌点），则你必须按照下述方式进行再叫：

1）对同伴的高级花色有4张以上的支持时，将同伴的高级花色加1副。

2）当同伴的应叫为1H时，你若有4张以上的S，你就叫1S。

3）再叫自己的6张以上的D花色。

4）叫1NT，表示比较平均的牌型。

5）叫2C，表示C为4张以上的长套，并且是不平均牌型（至少5-4-3-1）。

所有以上各种限制心再叫，均系非逼叫。如果应叫人在已经知道你是低限开叫牌之后，仍希望逼叫进局时，他就必须或者由他自己直接叫到成局（封局止叫），或者跳叫一新花色作为逼叫。应叫人的其他再叫，均系非逼叫（但有一例外情况，就是在开叫人的1NT再叫之后，应叫人跳叫回到3D乃是逼叫）。

如果你持高限1D开叫牌（14-15点），则在应叫人的高级花色应叫之后，你可按下述方式进行再叫：

1）跳加叫（加2副）同伴的高级花色，需持有至少4张支持。

2）在超过2D的水平上叫出一好的4张以上的新花色。

3）跳再叫自己的好的6张以上的D花色。

4）跳叫3C，表示是低级花色两色套牌（至少5-5）。

5）跳叫2NT。

在开叫人的高限再叫之后，应叫人叫出任何新花色都是逼叫一轮，跳叫出任何新花色都是进局逼叫，而直接叫到任何成局定约则是封局止叫。其他再叫则均非逼叫。

这里也需要补充说明一点，对于上述高限1D开叫牌，在高级花色应叫之后的各种再叫规定，也不应当机械地执行，而应当根据具体情况适当灵活应用。大致说来，当为高限强牌时，照上述各种规定再叫是比较好的；但在牌力不很强的一般高限牌时，以作低限牌再叫为宜，以免因同伴持低限牌力而联手的牌又不配合时，造成重大损失。应当记住，同伴若持11点以上的牌力，则即使在你的低限再叫之后，他也会再作一次叫牌，因而成局机会是不致轻易错失掉的。

下面看看几个例子：

开叫人（你）1D，应叫人1H，你？

SK863   HK1087  DAQ62  C3，加叫成2H，表示低限开叫牌（11-13大牌点），且至少有4张H支持。

SKJ63  H87  DAK8632  C5，叫1S。叫出一个牌力尚可的新的高级花色，比再叫6张的D花色为好。

SQ103  H1082  DAQ1092  CK2，叫1NT。表示低限开叫牌，且对H没有4张支持。

S863  HA  DKQ10843  CQJ2，叫2D。表示低限开叫牌，D为6张以上的花色。

S743  H2  DAQ986  CAK104，叫2C。表示低限开叫牌，但牌型至少为5-4-3-1，即其中D和C两花色为5-4，D或C任一花色为5张均可。

SK8  HKQ87  DAQ543  C74，跳叫加成3H，表示高限开叫牌（14-15大牌点），且至少4张H支持。

SKJ102  HQ32  DAKJ75  C2，跳叫2S。表示高限开叫牌，且S为好的4张套。

SAJ3  H7  DKQJ63  CKJ32，跳叫2NT。表示高限开叫牌，对H没有支持。

S8  HK62  DAQJ976  CKJ4，跳叫3H。表示高限开叫牌，且D为6张以上的花色。

SK3  H6  DAQJ54  CAJ642，叫3C。表示高限开叫牌，且D和C两个花色至少为5―5。

低级花色应叫

在同伴的1D开叫之后，如果你无法应叫高级花色，则其次的最好办法就是加叫同伴的D花色。在精确法中，跳加叫成3D是示弱止叫，表示弱牌但至少有5张D支持，加1副成2D，表示11―15大牌点，至少4张D支持， 并且是逼叫一轮。例如，当同伴开叫1D时，下列各牌的应叫分别为：

S―  H732  DKJ654  CJ8632，加叫成3D。表示示弱止叫。这一关煞性的加叫，看来是比较安全的，并且给对方参与叫牌造成极大困难。

S64  HA73  DAQ86  CJ1042，加叫成2D。你有11大牌点和4张D支持，但没有4张以上的高级花色。

S6  HA63  DKJ875  CAQ42，加叫成2D。加1副是逼叫一轮，因此你有时间来保证叫到成局。

在应叫人加叫成2D之后，开叫人应协助探查联手有无以3NT成局的可能。办法就是按下述方式进行再叫：

1）持低限1D开叫牌（11―13大牌点）时：

2H＝有H止张，无S止张。
2S＝有H止张，无H止张。
2NT＝两个高级花色均有止张。
3D＝两个高级花色均无止张。

2）持高限1D开叫牌（14―15大牌点）时：

3H＝有H止张，无S止张。
3S＝有S止张，无H止张。
3NT＝两个高级花色均有止张。
3C＝两个高级花色均无止张。

下面看看这类再叫的实例应用例子：

开叫人: SK83  H72  DAQ865  CK62            1D / 2S / Pass

应叫人: SAQ5  H64  DKJ74  CQ1053           2D / 3D

开叫人再叫2S表示低限，有S止张，但没有H止张。在得到这样精确的信息之后，应叫人便可毫无困难地叫出3D来作为止叫。

开叫人: SK83  H72  DAQ865  CK62            1D / 2S / 3NT

应叫人: S64  H AJ6  DKJ74  C AQ54          2D / 3S / Pass

此例中，开叫人的S止张正是应叫人所需要的。为了保证3NT成局定约由位置适合的一人来做，所以应叫人又叫出3S。这肯定不是表示应叫人有意于做S定约，因为应叫人的2D应叫已表示没有4张以上的高级花色，而是要求开叫人叫成3NT，因此，开叫人遵命叫成一个肯定做成的3NT定约。

开叫人: S652  H7  DAKJ73  CAQ103            1D / 3C / Pass

应叫人: SAK8  HAJ10  D10642  C764           2D / 3NT

此例开叫人的3C再叫，表示高限开叫牌，但两个高级花色均无止张。这样应叫人就可以算出联手至少有26大牌点，且对于高级花色也很放心，因而很有信心地叫成一个极好的3NT定约。

开叫人: S652  H7  DAKJ73  CAQ103            1D / 3C / 5D

应叫人: SAK8  H952  DQ1064  CK85            2D / 4D / Pass

在此例中，应叫人正确地料到H花色有危险，因而选择以D花色作进局试探。开叫人由于在D花色中有一宝贵的第五张王牌，所以有足够理由接受同伴的邀请而叫成5D进局。

如果在同伴的1D开叫之后，你既无法应叫高级花色，又无法加叫D，则你有可能应叫2C。如通常一样，二盖一应叫为逼叫，且牌力至少11大牌点：

S83  HA62  D643  CAQJ72，应叫2C。

S10765  H6  DA3  CAK9864，应叫2C。此牌的4张S花色太弱，因而不能机械地按规定应叫1S，而应当把好的6张套C花色叫出。

在2C应叫之后，开叫人应当按照2D逼叫性应叫之后的规定方式，叫明自己的牌力和高级花色止张情况：只有一点改变，就是当开叫人为低限开叫牌，且两个高级花色均无止张时，改用2D再叫（而不用3D）来表示。

无将应叫

在1D开叫后的1NT应叫，表示8-10点大牌点，没有4张以上的高级花色，并且为平均牌型。1NT应叫为非逼叫。3NT应叫也表示平均牌型，但牌力为14-15大牌点。例如，在同伴开叫1D时，下列各牌的应叫分别为：

SK103  HQ83  DQ63  CJ1042，应叫1NT。

SAQ7  HKJ2  D1097  CKQ52，应叫3NT。

SKJ3  HAQ10  DKJ4  CJ8643，应叫3NT。不可应叫弱的C花色。

在这些实叫性的无将应叫之后，开叫人一般均应停叫。只有在牌型极不平均，不适宜做无将定约时，方才可以在1NT应叫之后再叫出一新花色；或者在持有15大牌点时，如牌型平均，可在1NT应叫之后加叫成2NT，邀请同伴进局。

16大牌点以上的牌对1D开叫的应叫

当应叫人持有16大牌点以上的牌力时，他对1D开叫的应叫，基本上与前一章中所讨论的对高级花色开叫的应叫完全相同。亦即：在持有一好的5张以上的花色时，跳叫此花色；或者在牌型比较平均时，跳叫2NT。（注：1D开叫之后，应叫人跳叫新花色这种应叫，同样可以作为A问叫。关于A问叫参见第十二章。）例如，在同伴开叫1D时，下列各牌的应叫分别为：

SAKJ97  HAK6  DQ83  C73，跳叫2S。

SKQ107  HAK3  DKQ85  CK7，跳叫2NT。

以上各种应叫都是进局逼叫，并且是强的满贯邀叫。开叫人的再叫与前一章所讲述的再叫基本相似。亦即：在跳叫新花色这种应叫之后，开叫人如有3张以上支持，他就应当加1副，以便肯定用这个花色作为将牌；支持差时，开叫人就叫3NT以表示平均牌型的低限开叫牌，如为非均型牌时则叫出一好的4张以上的新花色或再叫3D。在2NT应叫之后，开叫人叫3C表示非均型低限牌，叫3NT表示均型低限牌，叫3D表示均型高限牌，叫新花色表示非均型高限牌。

当对方干扰时

在同伴开叫1D之后，对方参与叫牌时，应叫人的大多数应叫均保持原来意义。在对方的技术性加倍之后，应叫人叫再加倍表示至少11大牌点，并表示希望在对方逃叫时予以惩罚性加倍。在对方的争叫之后，应叫人在2副水平上叫出的加倍是所谓示弱加倍，表示至少有一个4张的高级花色；但在3副或更高副水平上叫出的加倍，则是惩罚性加倍。扣叫对方所叫的花色，是一种逼叫，要求开叫人能够叫出一好的4张以上的新花色。


本 章 摘 要

1D开叫＝11-15点，4张以上的D花色。

1)  应叫人不叫＝0-7点。

2)  应叫人应叫高级花色１H或１S＝8-15点，4张以上花色。逼叫。这时开叫人的再叫办法如下：

持低限1D开叫牌（11-13大牌点）时：

对应叫人的高级花色加1副，需持有4张以上的将牌支持。
对应叫为1H时叫1S，这表示S是4张以上的花色。
再叫自已的6张以上的D花色。
叫1NT，表示均型牌。
叫2C，表示C为4张以上的花色，且为不平均牌型（至少5-4-3-1，其中D和C为5-4，但何者为5张不限）。

持高限开叫牌（14-15大牌点）时：

跳加叫（加2副）同伴的高级花色，需持有4张以上的将牌支持。
在超过2D的水平叫出一好的4张以上的新花色。
跳再叫自已的6张以上的D花色。
跳叫成3C，表示低级花色两色套，但必须至少5―5。
跳叫成2NT，表示平均牌型。

3)  应叫人应叫1NT＝8-10点，没有4张以上的高级花色，平均牌型。非逼叫。这时开叫人的再叫办法如下：

一般不叫。
叫2NT＝15点高限牌，邀叫3NT。
叫新花色＝牌型极不平均，不宜做无将定约。

4)  应叫人应叫2D＝11-15点，没有4张以上的高级花色， 至少有4张D支持。逼叫。这时开叫人的再叫办法如下：

持低限1D开叫牌（11-13大牌点）时：

叫2H＝H有止张，S无止张。
叫2S＝S有止张，H无止张。
叫2NT＝两个高级花色均有止张。
叫3D＝两个高级花色均无止张。

持高限1D开叫牌（14-15大牌点）时：

叫3H＝H有止张，S无止张。
叫3S＝S有止张，H无止张。
叫3NT＝两个高级花色均有止张。
叫3C＝两个高级花色均无止张。

5)  应叫人应叫2C＝11-15点，（通常）没有4张以上的高级花色，C为好的4张以上的花色。（有可能持有好的6张以上的C花色和一个弱4张套高级花色。）逼叫。这时开叫人的再叫，几乎完全与2D应叫后的再叫相同，只是用2D代替3D来表示两个高级花色均无止张的低限牌。

6)  应叫人应叫3D＝示弱止叫，表示弱牌，但有5张以上的D支持。

7)  应叫人应叫3NT＝14-15大牌点，没有4张以上的高级花色，平均牌型。非逼叫。这时开叫人一般不叫。

8)  应叫人跳叫新花色＝16以上大牌点，所以花色为好的5张以上的长套，进局逼叫。（这种应叫可用作A问叫。）这时开叫人的再叫办法如下：

有3张以上的将牌支持时，加1副。
支持差时，叫3NT表示均型低限牌；叫一好的4张以上的新花色；或再叫3D。

9)  应叫人应叫2NT＝16以上大牌点，没有好的5张以上的花色。进局逼叫。这时开叫人的再叫办法如下；

叫3NT＝均型低限牌。
叫3C＝非均型低限牌。
叫3D＝均型高限牌。
叫新花色＝非均型高限牌。

第三部分

在你开叫1D之后，同伴的应叫分别如下列各题所示。你持下列各题的牌时如何再叫？

15)  同伴应叫1S，你持S7  HK863  DKJ872  CA103，叫1NT。若叫2H则表示高限开叫牌。

16)  同伴应叫2C，你持SAQ8  H743  DAJ10652  C9，叫2S。正如在2D应叫之后那样显示止张。

17)  同伴应叫1H，你持SK863  HJ2  DAK432  CQ8，叫1S。表示低限开叫牌，S为4张以上的花色。

18)  同伴应叫1S，你持S732  HA6  DAQ10872  CQ7，叫2D。表示低限开叫牌，D为6张以上的花色，并且对S没有4张支持。

19)  同伴应叫2D，你持SAQ8  HAJ72  DKJ753  C6，叫3NT。表示高限开叫牌。并且在两个高级花色中均有止张。

20)  同伴应叫1H，你持SAK74  H8654  DK1065  CA，叫3H。跳加叫以显示高限开叫牌。并且对H有4张以上的支持。

21)  同伴应叫2D，你持S872  HKQ3  DAKQ643  C3，叫3H。表示高限开叫牌。在H中有一止张。

22)  同伴应叫1H，你持SAJ107  H62  DKQJ85  CK2，叫2S。跳叫以显示高限开叫牌。

23)  同伴应叫2D，你持S63  H72  DAKQ652  CAQ8，叫3C。表示高限开叫牌。在两个高级花色中均无止张。

24)  同伴应叫1S，你持SK8  HQ7  DKJ1065  CK1082，叫1NT。应叫2C，则表示非均型的低限牌。

25)  同伴应叫2C，你持SK83  HKJ6  DA98765  C2，叫2NT。表示低限开叫牌。在两个高级花色中均有止张。
