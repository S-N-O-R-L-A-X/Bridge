附：罗马关键张问叫

Edwin B. Kantar修正後的RKCB问A/K法
 
一. 基本内容：
将牌配合后，问叫人用4NT询问答叫人4个A+将牌K(5个A)以及将牌Q+旁门K(4个k)的持牌情况：
    问 A
    4NT - 5C = 1 或 4 A
          5D = 0 或 3 A
          5H = 2 或 5 A without 王牌Q
          5S = 2 或 5 A with    王牌Q
基本上，应该不会有答叫人有五张 A的情况，所以答叫人答叫5H/5S通常就是两张A，由于5C/5D出现了"或"的回答，通常询问人可以藉由先前的叫牌
来判断出来。
 
    有缺门时的回答
          5NT= 0 或 2 A，有一缺门
          6C = 1 或 3 A，梅花缺门
          6D = 1 或 3 A，方块缺门
          6H = 1 或 3 A，红心缺门
          6线王牌 = 1 或 3 A，比同意王牌高的某一花色缺门
  

二. 有缺门时的罗马关键张(RKB)：又称“排除罗马关键张”(EKB) 
  RKB的询问者有缺门：由于RKB的询问者只希望听到此缺门以外的关键张数，因此关键张由5个减为4个。 
  方法是：RKB的询问者在将牌明确后，跳叫5阶缺门花色即为(EKB)。
答叫：按序级答叫关键张数：第1级：0；第2级：1；第3级：2；
 

三. 提前问张(Kick Back):这是专门为低花满贯试探而设计的
基本概念：同伴间在三阶上同意了低花将牌后立即跳叫最经济的未叫花色=提前问关键张。 
如：1s--2c--3c--4d#.4d=提前问关键张。 
答叫：与 罗马关键张(RKB)(一) 相同(加级) 
 
提前问张(Kick Back)与示短跳加叫(Splinter)的区分三原则：
1）提前问张必须紧跟三阶同意低花后马上叫出如：1d--2c--3c--4h#
2）同伴在三阶上作了示短跳加叫(Splinter)后，你再叫4阶同意低花=提前问张. 
如：1d--2d--2h--3s(示短跳加叫)--4d(提前问张). 
3）如果4个花色都已叫出过，明确低花将牌后跳叫第4花色=提前问张。

如：1d--2c--2h--2s(第4花色)--2nt--3d(明确低花将牌)--3h--4s(跳叫第4花色)=提前问张,而不是封局。

  
 
四. 5NT问K
    按照Edwin B. Kantar 的用法，在问完A之後，询问人再叫5NT为问K 的数目(不含王牌K)，此时的回答是依照级数，6C表示0K，6D表示1K， 
6H表示2K，6S表示3K。但在黑桃为王牌时，则须回答那些花色有K(因为黑桃为最高花色，所有的回答都不会超过安全线位的6S)， 其回答法为∶
        6C  = 有梅花K，还可能有一张红花K
        6D  = 有方块K，绝对没有梅花K，还可能有红心K
        6H  = 只有红心K
        6S  = 什么旁门K都没有
        6NT = 旁门三张K都有 
  

五. 询问王牌 Q
5.1 王牌为红心
    王牌为红心，当回答人对4NT回答5H或5S时， 询问人便可知道同伴是否有王牌Q；
若回答人回答的是5C，询问人叫∶
        5D  = 询问王牌 Q
        5H  = 束叫(因为通常1或4 A都分得出来)
        5S  = 对黑桃的直接询问
        5NT = 询问旁门 K的数目
 
若回答人回答的是5D，询问人叫∶
        5H  = 若回答人为0 A便Pass，若回答人为3 A便是询问王牌 Q
        5S  = 若回答人为0 A便是询问王牌 Q，否则为对黑桃的直接询问
        5NT = 询问旁门 K的数目
 
     对於询问王牌(红心) Q的回答，Edwin B. Kantar是采用∶
        5H  = (以5D询问)没有王牌 Q
        6H  = (以5H或5S询问)没有王牌 Q
        6H  = (以5D询问)有王牌 Q与黑桃 K
        5NT = 有王牌 Q，没有旁门 K
        6m  = 有王牌 Q与该门 K
        6S  = (以5H或5S询问)有王牌 Q与黑桃 K
 
5.2 王牌为黑桃
    王牌为黑桃，当回答人对4NT回答5H或5S时， 询问人便可知道同伴是否有王牌Q；
若回答人回答的是5C，询问人叫∶
        5D  = 询问王牌 Q
        5H  = 对红心的直接询问
        5S  = 束叫(因为通常1或4 A都分得出来)
        5NT = 旁门 K的位置
 
若回答人回答的是5D，询问人叫∶
        5H  = 若回答人为0 A便是询问王牌 Q，否则为对红心的直接询问
        5S  = 若回答人为0 A便Pass，若回答人为3 A便是询问王牌 Q
        5NT = 询问旁门K的位置
 
     对於询问王牌(黑桃) Q的回答，Edwin B. Kantar是采用∶
        5S  = (以5D或5H询问)没有王牌 Q
        6S  = (以5S询问)没有王牌 Q
        5NT = 有王牌 Q，没有旁门 K
        6m/H= 有王牌 Q与该门 K
 
5.3 王牌为方块
    王牌为方块，当回答人对4NT回答5H或5S时， 询问人便可知道同伴是否有王牌Q；
若回答人回答的是5C，询问人叫∶
        5D  = 若回答人为1 A便Pass，若回答人为4 A便是询问王牌Q
        5H  = 询问王牌 Q
        5S  = 对黑桃的直接询问
        5NT = 询问旁门 K的数目
 
若回答人回答的是5D，询问人叫∶
        5H  = 询问王牌 Q
        5S  = 对黑桃的直接询问
        5NT = 询问旁门 K的数目
 
     对於询问王牌(方块) Q的回答，Edwin B. Kantar是采用∶
        6D  = 没有王牌 Q
        5H  = (以5D询问)有王牌 Q与红心K
        5S  = 有王牌 Q与黑桃 K
        5NT = 有王牌 Q，没有旁门 K
        6m/H= 有王牌 Q与该门 K
 
5.4 王牌为梅花
    王牌为梅花，当回答人对4NT回答5H或5S时， 询问人便可知道同伴是否有王牌Q；
若回答人回答的是5C，询问人叫∶
        5D  = 若回答人为1 A便叫5NT，若回答人为4 A便是询问王牌Q
        5H  = 询问王牌 Q
        5S  = 请同伴叫5NT或直接询问
        5NT = 询问旁门 K的数目
 
若回答人回答的是5D，询问人叫∶
        5H  = 询问王牌 Q
        5S  = 请同伴叫5NT或直接询问
        5NT = 询问旁门 K的数目
 
     对於询问王牌(方块) Q的回答，Edwin B. Kantar是采用∶
        6C  = 没有王牌 Q
        5H  = (以5D询问)有王牌 Q与红心 K
        5S  = 有王牌 Q与黑桃 K
        5NT = 有王牌 Q，没有旁门 K
        6D  = 有王牌 Q与方块 K
        6H  = (以5H询问)有王牌 Q与红心 K
 
5.5 询问王牌 Q小结
    总之，当回答人不打算Pass询问的人的问王牌Q时，其回答为∶

    最便宜回意王牌
    跳叫至六线王牌
    五六线简单叫出新花
    5NT 
    跳叫至六线新花
    跳叫6NT

= 没有王牌 Q
= 有王牌Q与较高花色的 K
= 有王牌Q与该门花色的 K
= 有王牌Q，旁门无 K
= 有将牌Q和两个旁门K，跳叫的花色无K
= 有将牌Q，三K全有

 
       
  
六. 直接询问
    询问人在问A之後，在六线王牌之下叫出一门新花，或是在五线叫出非询问王牌Q的叫品(如果约定此叫品不是迫同伴叫5NT)，便是直接询问。
欲使用直接询问必须确定五个关键张与王牌 Q都齐了。若被询问花色与六线王牌之间至少有一级牌可叫，其回答为∶

    加高一级
    加高两级
    更高级
    6NT
    七线较王牌低的花色
    (非被询问花色)
    七线王牌
    被询问花色的七线

= 被询问花色无第二控制或第三控制
= 被询问花色有第三控制(两小或有Q)
= 被询问花色有第二控制(单张或有K)
= 被询问花色有Kxx、Kxxx、或Kxxxx
= 被询问花色为 Kx

= 被询问花色为单张
= 被询问花色有 KQ

 
 
若被询问花色与六线王牌之间已无叫牌空间，其回答为∶

    加高一级(即王牌)
    6NT
    七线王牌
    被询问花色的七线

= 被询问花色无第二控制
= 被询问花色有Kxx、Kxxx、或Kxxxx
= 被询问花色为单张或Kx
= 被询问花色有有 KQ

 
 
从这里我们可以看出∶直接询问是用来询问一特定花色的第二与第三控制之用。
 
但若王牌为黑桃，则在询问完关键张後叫6H为询问第三控制(稍後会解释)，其回答法同延後询问。
  

七. 延後询问
    延後询问乃是在询问王牌Q或以 5NT询问K之後，询问一门的控制情况。也就是在询问王牌Q或问K之後，询问人又在六线王牌之下叫出一门
新花，便是延後询问。欲使用延後询问必须确定五个关键张与王牌 Q都齐了。
 
若被延後询问花色与六线王牌之间尚有叫牌空间，其回答为与直接询问相同。
 
若被延後询问花色与六线王牌之间已无叫牌空间，其回答为∶

    加高一级(即王牌)
    6NT
    七线王牌
    被询问花色的七线

= 被询问花色无第三控制
= 被询问花色有Q
= 被询问花色为双张
= 被询问花色有有 QJ

 
　
很明显的，无叫牌空间的延後询问是用来询问一门花色的第三控制。但在王牌为黑桃时，无叫牌空间的延後询问红心是用来询问红心第二控制。
 
    这里我们解释一下为何当黑桃为王牌时，在问 A之後，直接叫6H的询问为询问红心的第三控制，而延後询问红心为询问红心第二控制。
首先看第一种状况∶
        4NT - 5C
        5H/6H
因为5D是询问王牌Q，所以5H的直接询问在明显有空间的情况下，当然是询问红心的第二三控制。所以跳叫6H应该表示询问人只想问红心的第三控制。
第二种情况则是∶
        4NT - 5x
        5NT
因为我们规定当王牌为黑桃时，5NT问K乃是回答旁门K的位置，所以若被询问回答出6C或6D表示出梅花K或方块K时，还是可能有红心K，所以此时
我们希望把延後询问的6H作为询问红心第二控制之用。第三种情况则是∶
        4NT - 5D
        5H/6H
因为5H是用来询问王牌Q，所有要直接询问红心就必须跳叫6H，既然第二种情况已经定义了延後询问的6H为询问红心第二控制，则这种状况就应该
是询问红心第三控制。 
 

八．选择哪一门花色作王牌的原则
罗马关键张最大缺点是，若不能完全弄清哪一门花色为已同意的王牌时，就不易判断哪一个是K关键张.但若能按以下5条原则就基本可克服这
个弱点：
1）若仅有一门花色被支持过，则此花色的K就是关键张； 
2）若有两门花色被支持过，则罗马关键张问叫人所支持花色的K是关键张，
   如：1h--2c--3c--3h--4nt(罗马关键张问叫):CK是关键张。
3）人为强开叫人(如精确1c,自然强2c开叫等)所叫套的K是关键张，
   如1c(精确)--1s--2h--3d--3h--3s--4nt(罗马关键张问叫):HK是关键张。
4）若答叫人对开叫作过强跳叫旁门，则强跳叫花色的K是关键张，
   如1s--3c(强跳叫)--3s--4nt(罗马关键张问叫):CK是关键张。
5）若上叙4种情况都没用上，则最后叫出花色的K是关键张，

如：1s--2c--2s--4nt（罗马关键张问叫):SK是关键张。 

  

九. 第二门花色
    在前面所说的直接询问与延後询问，询问人都是叫出一门未表示过为牌组的花色来询问。但若是询问人在可以叫出直接询问或延後询问的时
机，叫出一门曾经叫过的花色，其意义为∶
    1.询问人在RKCB之後，叫出他曾叫过的花色∶这是自然叫，表示该门花色坚  强，请同伴选择合约。
    2.询问人叫出自己或同伴第一个表示的牌组∶自然叫，请同伴选择合约。
    3.询问人询问王牌 Q，同伴表示没有之後，询问人叫出自己或同伴第一个表示的牌组∶自然叫，请同伴选择合约。
    4.询问人询问王牌 Q，同伴表示有之後，询问人叫出自己或同伴第一个表示的牌组∶延後询问。
    5.询问人在六线叫出自己第二个表示的牌组∶自然叫，请同伴选择合约。除非RKCB花色被跳叫支持过，则此时为询问叫。
  
  
十. 为何要使用14(5C)30(5D)
    主要的原因在於以红心为王牌时，便於询问王牌 Q。我们看看下面的例子∶
      开叫人  答叫人
        1S      2C
        2H      3H
        4H      4NT
        5D
 
答叫人手持∶ S x
            H KTxx
            D Ax
            C AKJxxx
如果使用原本的03(5C)14(5D)的RKCB，开叫人的5D很明显的表示一张A，此时红心小满贯的有无便系於开叫人是否有红心Q。此时答叫人再叫5H
为束叫，只能叫5S询问，此时不论开叫人是否有红心Q，合约已经超越5H了(至少是5NT) 。在开叫人只有红心Axxx时，5NT通常不会是好合约，
6H则只能希望有个防家的王牌是QJ双。所以在以往的RKCB中，在询问王牌(红心)Q时， 必须保证所有的A(包含王牌K)都已齐全。
 
但如果使用14(5C)30(5D)的答叫呢? 在上述的状况中，开叫人对 4NT的回答将是5C，答叫人可以加高一级叫5D询问开叫人是否王牌Q，开叫人若
没有王牌Q，便可以舒服而安全的叫出5H。这样在询问王牌(红心)Q时，就可以允许缺一张A。
 
相对的，14(5C)30(5D)的答叫在以较少的A回答出较高的叫品，也有其问题∶
    (1).王牌为梅花，当回答5D者为0张A时，合约可能已经过头。但在缺三张 A时，不论回答的是5C或5D都可能已经过头。所以当低花为王牌时，
问A者应至少有两张真正的A；或是以其它的提前询问法问A。
    (2).同意王牌为方块，当回答5D时，不能确定回答者是0或3A时，不知道要Pass或继续问下去。其实在使用03(5C)14(5D)法，而王牌为梅花，
对4NT回答5C时 ，亦有此问题。解决的方法就是以提前询问法问A。
    (3).同意王牌为红心时，当回答5D时，不能确定回答者是0或3 A时，询问人可以加高一级叫5H，当回答人为0 A时，Pass 5H；3 A便回答是
否有王牌Q。
    (4).同意王牌为黑桃时，当回答5D时，不能确定回答者是0或3 A时，询问人可以加高一级叫5H，当回答人为0 A时，便叫5S；3 A便回答是否
有王牌Q。
