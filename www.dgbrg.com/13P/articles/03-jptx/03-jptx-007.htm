莱本索尔约定叫

――强烈推荐使用，特别是使用精确制的牌手
　

莱本索尔特约的原名lebensohl，它的第一个字母往往被人误作大写。分明是姓氏为什么偏偏要违背书写规则不作大写呢？原来有这个典故：据该特约设计人员George Boehm在《桥牌世界》1970年11月号撰文讲，约一年前，他参加纽约一次瑞士队式赛，一个高瘦粗犷的青年，指着约定卡上划有红线详列叫法的“莱本索尔”字样问道：“这是什么？”Boehm解释说“这是麻省理工学院学生坎・莱本索尔想出的特约，我同伴开叫1NT，你盖叫2S，我拿着六张套，约5―6大牌点，但不能竟叫3D，因那是成局逼叫，不叫又不甘心。怎么办呢？我可以叫莱本索尔2NT，同伴依约定须再叫3C，我再叫3D，他知道是这样的牌便会不再叫。这有点象费林特约。”那青年再看一下约定卡，露出鄙夷的表情。Boehm光火了，不客气地说：“坎・莱本索尔是位有点名气的好牌手，年纪虽轻，却是颇有天份的理论家。你是哪里来的年轻人，敢对他的思想不尊重？”那青年冷静地回答：“我就是坎・莱本索尔。虽然你没把我的姓氏拼错，我还是不稀罕你那玩意儿，不想同它沾边。”Boehm认为，莱本索尔先生既然宣告弃权，这特约便不宜再写作他的姓氏了。

这特约虽出自莱本索尔的构思，却有Boehm父子加工臻于完善。大概坎・莱本索尔看到同他的原版不一样，所以不愿意掠美吧。Boehm设计并为大众接受的叫法是：开叫1NT，对于盖叫二阶花色，应叫家可酌情用以下叫品：  

    （1）加倍，是惩罚性；
    （2）竟叫二阶花色，非逼叫；
    （3）叫三阶花色，成局逼叫；
    （4）虚叫2NT，开叫家必须再叫3C，应叫叫如持弱牌长套，便可不再叫，应叫家如再叫三阶花色，其等级较对手的花色低，是弱叫，较对手花色高的，则是成局邀叫（invitational）；
    （5）直接扣叫，是要求开叫家叫出他的四张套高花，并表示应叫家没有对手花色的止张，这是成局逼叫。但开叫家如无四张高花套，又无对手花色的止张，要再四阶再叫其较好的低花套时，应叫家也可酌情不再叫。
    （6）直接跳叫3NT，是支持成局牌力，但无四张高花套，也无对手花色的止张。
    （7）先叫2NT，以后扣叫，是有四张高花套，并有一个对方花色的止张。开叫家应再叫四张高花或3NT。
    （8）先叫2NT，再叫3NT，是有对手花色的止张。

    倘对手作三阶花色盖叫，应叫家叫法是：
    （1）加倍，不是惩罚，是逼叫性的；
    （2）竟叫新花，是成局逼叫；
    （3）其他叫品均为自然叫。
    倘对手作四阶花色盖叫，应叫家叫加倍是惩罚性的，叫新花是自然的，叫4NT是标准黑木问叫（也可经约定作为自然叫），叫5NT是满贯邀叫。

阻击叫后的叫牌对任何水平的牌手都是棘手的。在如下的叫牌后

北        东         南         西
2S        X          Pass      3D
Pass      ?

东很难判断西的实力。西可能毫无点力，但也可能拿了8-9点。为了解决这一难题，就设计了二价阻击叫后的lebensohl ，以便更精确地描述牌情。

二阶阻击叫后的Lebensohl约定：

当对方开叫二价阻击叫，同伴技术性加倍，右手方Pass，你的2NT是lebensohl。

例二：NT与无将叫牌无关，要求同伴再叫3C由你进 一 步描述牌情。

以下以2H阻击为例说明其基本用法

北        东         南         西
2H        X          Pass       ？

1。弱牌 (0-6点)
1) S套：2S
2) C或D套：2NT，同伴接力3C。

北        东         南         西
2H        X          Pass       2NT
Pass      3C         Pass       ?

C套：Pass
D套：3D
2。中等牌力 (7-10点)
1) C或D套：直接3C或3D。
2) S套
A。四张：2NT，同伴接力3C，后3S：

北        东         南         西
2H        X          Pass       2NT
Pass      3C         Pass       3S

B。五张以上：直接3S：

北        东         南         西
2H        X          Pass       3S

籍此可区分S的长度。

11点以上的牌
lebensohl约定可以使你保留原来的叫牌含义。

加倍者强牌 (18点以上)
lebensohl 2NT后，同伴持强牌有时可以不接力3C，而表达自己的牌。由于lebensohl 2NT者通常持一手0-6点的弱牌，只有当面对4-5点牌都有进局前景时才考虑越过3C。

北        东         南         西
2H        X          Pass       2NT
Pass      ？

3D/3S/4C：D/S/C好套，至少18点，基本逼叫。
3H：极强牌，逼叫进局。
3NT：自然叫。常基于一强长套。(有些牌手表达其他含意)
4H：C/D双低套，至少5-5。

lebensohl 的特殊用法
利用Lebensohl 约定还可以区分一些不同类型的牌。不同牌手采用的约定不尽相同，以下介绍我个人较喜好的方式。

1。进局实力(至少12点) 区分止张及有无四张S

1) 无四张S，H有止张。直接3NT

北        东         南         西
2H        X          Pass       3NT

2) 无四张S，H无止张。直接3H扣叫

北        东         南         西
2H        X          Pass       3H

3) 四张S，H有止张。Lebensohl 2NT，然后3NT。

北        东         南         西
2H        X          Pass       2NT
Pass      3C         Pass      3NT

4) 四张S，H无止张。Lebensohl 2NT，然后3H扣叫。

北        东         南         西
2H        X          Pass       2NT
Pass      3C         Pass       3H

2。满贯兴趣
I。单套
A。 S套满贯兴趣

1) 5张以上S，仅成局兴趣。直接4S。

北        东         南         西
2H        X          Pass       4S

2) 5张以上S，略有满贯兴趣。lebensohl 2NT，然后4S。

北        东         南         西
2H        X          Pass      2NT
Pass      3C         Pass      4S

3) 4张以上S，极强满贯兴趣。Lebensohl 2NT，然后3H，同伴暂时认为你仅成局兴趣。你接着再满贯试探，表示好牌。

北        东         南         西
2H        X          Pass      2NT
Pass      3C         Pass      3H

B。低花单套
扣叫3H，然后4阶出套。(当同伴改叫4阶低花时，有时会难以澄清牌情)

北        东         南         西
2H        X          Pass       3H
Pass      3NT        Pass     4C/4D

II。两套型满贯兴趣

1) 双低花(一般至少5-4)，H无控制：lebensohl 2NT，然后4H

北        东         南         西
2H        X          Pass      2NT
Pass      3C         Pass      4H

2) 双低花，H有控制：直接4H

北        东         南         西
2H        X          Pass       4H

3) 四张高花带一低花 lebensohl 2NT，然后4C或4D

北        东         南         西
2H        X         Pass       2NT
Pass      3C        Pass      4C/4D

注意：同伴加倍后，你直接4C或4D，仅表示C或D邀叫，大约12-13明手点。

北        东         南         西
2H        X         Pass      4C/4D (邀叫)

以上尽管以 2H 阻击叫同伴直接位置加倍为例，类似原理也可运 用于2D或2S阻击叫后，以及平衡位置加倍。

另外，lebensohl 2NT为一约定叫，实战运用时必须提醒 (包括衍生的含义)。

最后，lebensohl 2NT约定虽是一个有力的武器，但并不能解决阻击叫后所有 的难题。

一、1NT后的莱本：

对方在1NT后的高花争叫，特别是2S,经常是第3家伤脑筋。比如：
S     W     N     E 
1NT   2S    ? 
N-S74
HKJ9765
DJ8
CKJ4
N想叫3H,但同伴会怎么理解呢？也许他会认为你有13p和5张H，用3H逼叫。莱本解决了这个问题，约定所有3水平以上的叫品都是逼叫性的。2NT是转到3C的转移叫，之后应叫者就可以pass（拿C套）或叫另一花色示停了。比如：
S     W     N     E 
1NT   2H    2NT   - 
3C    -     3D 
N-S952
H6
DQJ8542
CAJ7
这时的3D不是邀叫，同伴必须pass。
S     W     N     E 
1NT   2H    2NT   - 
3C    -     3S 
N-SKJ9652
HJ4
DA1094
C3
这里，N本来可以叫2S示停。现在在莱本后叫3S,所以是邀叫。直接跳3S才是逼叫。
S     W     N     E 
1NT   2S    ? 
N-S96
HKQ74
DQ3
CAQ1062
N叫3C就是逼叫。如果H配上，前途就光明了。
对以下的叫牌又怎么理解呢？
   S     W     N     E
a. 1NT   2S    3NT
b. 1NT   2S    2NT   -
   3C    -     3NT
c. 1NT   2S    3S
d. 1NT   2S    2NT   -
   3C    -     3S（S换成H也一样）
a、表示要打3n且S有止。
b、表示要打3n且S没有止。
c、Stm，保证另一高花有4张，但S无止张。
d、表示S有止，其他同C。
那么使用莱本就失去了自然的2NT这个叫品，那你那下面的牌怎么办？
S     W     N     E 
1NT   2S    ? 
N-S952
HAQ8
DK1095
CQ93
检验叫X，表示11点左右的均型牌，对S没什么许诺，同伴认为不适合可以改叫。
要N拿SQ1085 HA42 DJ105 C1073 ，就pass,获得未X的罚分。当然。也有约定这时的x表示惩罚性的，看自己的理解了。

二、弱二之后的莱本：

同样的对于2H/S X后的叫牌有着类似的问题，可以使用莱本。有个微小的区别就是这时的扣叫不再是Stm,而是逼局且有温和的满贯兴趣，但可能没有好套。

三、修正的莱本：

也有人认为点高的人做庄家更为有利，于是1n-2H/S后应叫：a、2NT 转移到3C。b、3C 转移到3D。C、3D 转移到3H/S。
比如：  S     W     N     E
   1、 1NT    2S    3D    -
       3H     -     -
   2、 1NT    2S    2N    -
       3C     -     3H
   3、 1NT    2S    3D    -
       3H     -    
再叫分别表示：1、弱牌。2、邀局的牌力。3、逼叫的牌力。
