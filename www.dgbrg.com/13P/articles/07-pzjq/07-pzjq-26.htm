作者：舒服斯基(_wushangjie)

第3篇：东流奇遇

    用一帆风顺来概括东瓜皮的桥牌之路是非常贴切的，一出道就有一个名震中华的好老师，北京队。国青队。国家队一步一登天。没辙呀，人要有命那运气挡都挡不住，好象他的父母能掐会算，给他起的名字都有预见--他的大名叫福中！

    福中生在文革后期的北京，在社会大动荡大变革中长大，王朔就是写这一茬人出的名。我们到工体的足球看台，那几万个齐声高喊"傻b"。"牛b"的中青年就是这一代人小小的缩影。我不认为那是脏话，要表达充满胸中的爱憎，还有比这更淋漓更简洁有力的口号吗?舆论曾担心他们类似英国流氓，其实那是绝对多虑，英国流氓打架不要命，但这些貌似疯狂的京痞子，即使在衔头上百人和几个山东大汉叫板，也就是躲在街角扔几块砖头，伸着脖子乱喊几声哄哄罢了！

    写到这儿，您可别误会，大名鼎鼎的福中可不是痞子，相反，他是牌圈中北京人里少有的绅士之一，这一点从他平时永远衣着整洁，发型完美就不容置疑。可闲的你写那些干吗?别急，朋友！

    但凡北京人都有一个鲜明的特点，那就是能侃善骗，京嘴子闻名天下那可不是谣传。笔者在这里奉上一句真言：千万别和北京人斗嘴或讲理。不服?哪天你和我队友石正钧或周青宏待上几分钟，你就明白嘴巴有比吃饭更重要的功能。这里我先放下那两位牛货不表，看一看最老实的福中吧--他自称东流，我们都叫他东瓜皮。

    和普通的牌友一样，成名的人也是常换搭档，福中先后和王晓静。鞠传成分手后。。。。。。

    那一天，在沙家浜的小河边，蹲着一位汉子，一副愁眉苦脸的样子目送流水，我恰巧路过，一看这不是我心中的大白偶--福中的新搭天津杰克赵吗！拍马搭关的好机会呀，赶忙上前去套近乎，看那赵杰郁郁的样子，其实我心说：真有事想不开，就跳下去算了。但嘴上得关怀呀，只听赵杰说：老卷呢，没法和你们北京人混，有一个算一个，真***不是东西！

    我正色道：兄弟，你这是打哪说起，是不是那东瓜皮说咱牌面，不和咱搭了?

    赵杰道：那到不是，我的牌就好比飞茌天上的雄鹰，他们只能仰视(啊，天)！可他们欺负我这外地人，嘴上没真话呀，你看刚才我玩双扣，背了好久了，好不容易抓了一把牛牌，除了炸弹还是炸弹，正想痛快一把，可那福中从外面进来，喊我服务台接长途，你想，我媳妇快生了，如有事我能不去吗?扔了牌飞奔下楼，哎，有个屁电话，气死我也。

    信北京人的话，气死活该。我在赵杰耳边一番交待，他疑惑的瞪我：这招能行?

    我说：你去试试。

    就见赵杰又是飞奔上楼，咣的一揣门，怒指正坐在他位子上玩牌的福中：你小子以后再敢骗我，我抽你丫挺的！

    就见噌的一下，人影一晃，福中没了。

    从此，他们队中再也没人和赵大侠逗贫了，老夫我则品尝了半截天津大麻花。。。。。。

    岁月倒流，那一年，北京著名的地坫庙会重新恢复，17岁的福中和一位小伙伴来此游玩，品了很多美味小吃之后，来到一座大院，见门上的招牌纸上写着：体育擂台，内设中国象棋，围祺和桥牌。

    福中和他的小伙伴都是地道的扑克迷，但他们从没听说过还有什么桥牌，俩人好奇心起，进院就奔了贴着桥牌室的小屋。见里面10几个人围在一个坐着4人的桌旁，静静的看他们打牌，福中人小体瘦，很容易的挤了进去，就这样第一次接触了这项使他成名的职业。

    略去一干枝节，单表看了半天方才略知玩法的福中，心想，这简单，不就是有将的时候象升级，没将的时俟同拱猪谁大谁出吗。他把伙伴拉出来，对他说：我看懂了，他们先在纸上写什么定约，1c就是以草花当将1+6拿7圈，s花色最大，无将最高，谁叫的高谁玩，你去花3毛钱买张票，我手扛，只要抓着黑桃，赢面就大，万一他们抓着黑桃，咱们就用无将抢啊。

    小伙伴疑惑的看着他，但一想到平时玩扑克，只要有福中，别人就没的赢，竟真的去买了张票，过了一会儿，两个其实对桥牌还一窍不通的小青皮坐上了擂台。

    在当年桥牌还没什么人会玩，这擂台是几个先行者自费搞的活动，他在庙会只生存了一届，当时甚至牌套都不用，就一副牌，现发现打，福中的手就和他的名字一样，在福字之中啊，第一手牌，就拿到了一大把黑挑：  

                      s  ak986543
                      h  a98
                      d  7
                      c  7

    看了自己的牌，他又愤怒又得意，得意手壮，愤怒的是这牌要在憋7中拿到，那得赢多少钱呀，他在纸上写了1s，并成为最终定约，对方首功hx，全手牌如下：

                     s  qj
                     h  qj10x
                     d  aqxxx
                     c  qj
            s 10                  s  72
            h kxxx                h  xx
            d jxx                 d  kxxx
            c axxxx    福中       c  kxxxx
                     s  ak986543
                     h  a98
                     d  7
                     c  7

    你也许看不懂，这牌怎么就打1s?各位看官，这俩小哥们甚至连局分都不知道就来现世了，得正分不是叫的越低越好得吗?那写这牌有什么劲，定约实在没劲，可里面的故事却会让所有桥牌爱好者们感叹左右人生命运的有时就是机遇！

    话说福中明手放hq拿到了这一墩，他突然天真的问：憋7最后出7分数能翻，这桥牌有什么讲究，怎么能翻?

    两位擂主一个高大威猛，他明白今天碰到了一位棒槌，他说：能翻啊，象高花打成4个6个或7个有很多的奖分。

    另一个面秀口气幽默：这桥牌最后一张出7没用，倒数第2张出7分数大涨。

    说者无心，听者有意，那福中精神大振，他出da，将吃一轮d，王牌下桥，再将吃一轮，王牌又下，再将一轮d，做大了第13张d，但没桥下去了。可扑克游戏中的技巧是一门艺术，当你有一大长套的时候，奔吃他就会让你柳暗花明，连打s形势如下：

                      h  j
                      d  j
                      c  qj
              h  kx              h  x
              c  ax    福        c  kxx
                      s  x
                      h  ax
                      c  7

    出最后张s，下家必须弃掉ca，否则将被投入，明手垫d。最后家垫小c。福中拨了ha，他欢快的出c7，就这样梦家qj双张c竞无法阻挡的取得第12墩。他得意的叫着：我倒数第2张出7且打成6个，给我加分吧！

    规则是贴点，当高大威猛的汉子对他说，由于你联手26点，打成6s=230分=6点-6个贴点，此牌不得分，福中急了：赖呀，你们在这骗钱呢。。。。。。

    后来大汉从自己兜里掏了3毛钱退了他们，并把奖品一本《《桥牌入门》》，送给了福中，并在上面写了一个地址：东四8条东华桥牌俱乐部。

    福中不知道，当他离开的时候，这位后来成为他恩师也是一代人心中永远屹立不倒的偶象口中曾念念自语：天才啊天才！
