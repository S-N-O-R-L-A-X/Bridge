舒服系列

　

作者：舒服斯基(_wushangjie)

第1篇：无悔的漂泊


    兜里揣着一副牌--逮谁跟谁来！这是时下扑克迷的形象写照。

    牌戏让人上瘾，但当你一学会桥牌，那其它的玩法你就顿觉索然无味。搞别的一般都需挂点彩头，才能刺激兴趣，只有打桥牌不用赌钱，也能酣战不休。想当初，刚学的时候，几个朋友凑到一起玩个通宵那是常事。至于旷工逃学到处自费打比赛，这些都不在话下。可今天桥牌终于成为混饭吃的职业了，却突然发现自己成了其它什么也不会做的废人。由此证明桥牌是一项毁人不倦的项目一点也不夸大。

    老爷们喜好什么就沉迷什么，我以为这是天经地义，也可说是男人本色！

    但女孩呢?女孩手捧扑克，沉思苦想，反正我是感觉不舒服甚至有点可笑，有那工夫，去跳跳赏心悦目的健美操，花前月下的谈谈多角恋那多浪漫，总之，在牌桌上辜负这大好的舂光--可惜呀！

    在古老的南京城，有一位娆窕淑女，名叫黄艳，在大学里学会了桥牌，从此踏上了这条不归之路。她还算幸运，和另一位才女周新蕾一出道，就入选了仅仅搞了一茬的国青女队，有机遇碰到高人的点拨，很快就在稀缺的女子战场中有了一席之地，还没毕业就被火车头招至旗下，成为最早的半职业牌手。可当周妹妹随夫去了美丽坚，形单影支的黄艳一度离开牌圈。她独闯北京在一家大公司谋了一份不错的职业，正根的小白领是多少女孩的追求，但桥牌是勾魂的魔，进来了就别想走开，当牌圈再次召唤，难说犹豫，黄妹妹又一无反顾的杀了回来！

    哎，搞不懂这么一项挣钱不多无法养老又是经常痛苦且前路茫茫的游戏，我们为她付出自己的青舂--真的值得吗?

    这是前年全国混双决赛中的一副牌，黄妹妹坐西拿着：

                      s  axx
                      h  a7x
                      d  akxxx
                      c  jx

    东西单方有局北开叫：

             n        e       s       w
             1d       /      1h       /
             1nt      /      2h       /
              /      //

    对方是自然体系，黄艳拿这么捧的牌，居然无法发言，假如南仗着单无局没点乱搞，已方即使有局牌也会被他憋死，但黄妹妹不怕，因为她对面坐着一位肥男叫累神，在平衡位置他手里只要有点货就不会老实，一定跳出来捣乱且时常被人家送上西天，要不怎么人称纯瓜一郎呢！

    这回纯瓜没动，黄艳首拨da，看见了明手牌：

                         s  kxx
                         h  xx
                         d  q10xx
                         c  akxx

    同伴出dj。假如这是单张，该续打dk，同伴将吃后下次再上手让同伴的小将牌消灭掉明手的dq！这是显而易见的防守，不论怎样，再拿已经暴露的dk不会有何不妥，但黄艳回了一张小s，全手牌如下：

                 s   kxx
                 h   xx
                 d   q10xx
                 c   akxx
        s  axx       n      s q109x
        h  a7x              h j8x
        d  akxxx w      E   d jx
        c  jx               c 109xx

                       S
                 s   jxx
                 h   kq109x
                 d   xx
                 c   qxx

    庄家不敢跳k，因为k一但被打死就有迅速输掉3墩s的危险，想了一下，还是放小，东sq得，出第二张d到dk， 现在黄妹妹已数出5墩牌，宕墩在哪?我想在晃s时她大概已成竹在胸了。庄家假如c上还有输张，当务之急就是要消灭明手已建立的d赢张，但怎样通知同伴，庄家也无d，尽量用大将牌将吃d呢?虽然这是如此明显，但对面坐的是纯瓜一郎，头顶大师称号，大师分也高高在上，但打牌时随心所欲，胡打乱叫，开叫1nt经常只有12点不算，还偶有单张或阴着6张套，更让人恶心的是，借探讨牌技之名却惦记着偷摸小手，如此不堪之大师当今牌圈躲不胜躲。我黄妹妹走遍天涯海角，也闪不开这帮坏蛋。(哈哈)

    在出d之前，黄妹妹先提了sa！拉响警笛，那纯瓜想了想，不情愿的用hj将吃d，庄家眼一亮，用k超将吃，所余牌如下：

                 s   k
                 h   x
                 d   q
                 c   akxx
          s x        n      s  xx
          h a7x             h  8x
          d xx  W       E   d    
          c jx              c  109xx
                       S
                  s   j
                  h   q109xx
                  d   
                  c   qxx

    庄家出h9，企图偷渡，但被我黄妹妹用a打死，又一张d，纯瓜献出最后一滴鲜奶h8！终于女孩的h7笑到了最后。现在回过头来再看前面，假如不先晃庄家s，他会在你将牌去顶的时候，扔掉s而腾挪！

    桥牌是如此让人上瘾，这副小小的2h的防御，分明就是一首诗，一首朦胧的小诗，只有读懂她的人才会体验到其中那壮丽的美。就是因为要享受这一次次刹那的陶醉，一个倔强的女孩从南京到北京，从火车头到上海富诚，固执地漂泊在不会有个人前途的牌场上。但她--

    黄艳无悔！

    所有热爱桥牌的朋友们--

    我们无悔！
