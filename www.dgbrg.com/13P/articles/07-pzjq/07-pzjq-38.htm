作者：累神（_wushangjie)

第7篇：神鸟传奇之桥海无涯

    累神有一次向于国明老师讨教牌技，于老师教诲之余，曾叹道：桥牌是魔啊，进来了就别想出去：的确如此，只要你迷上了桥牌，，她不仅仅会让你上瘾，他会让你爱，让你痛，让你的付出一无所获却舍得其所：有时一付牌局一张牌竟能让你感悟生活的全部：大喜大悲。希望。失望。绝望。以及新生的期待。。。。。。

    这不有一个叫石淼的家伙，刚进队时我们叫他笨鸟。傻鸟。他总是笑咪咪的不说话，默默无言中他的牌技征服了我们。好几次我这边被对手打飞，但见石淼轻飘飘的出来，我说：对不起，后手很多。

    石淼看看记分表，拍拍我的肩膀：打的不错吗，比敌人得的分多：

    一次又一次，石淼拖着我们这些油瓶向冠军爬去，我们改叫他小鸟。鸟罗素(葛罗素之意)。有一天我们终于叫他神鸟了，是发自内心崇拜的叫。他拿了这样一手牌： 

                            S  xxxx
                            H  AQ
                            D  xxx
                            C  xxxx

      他的上家开叫，叫牌过程：  对手1   鸟   对手2   鸟伴
                              1H     /     2C      /
                              2H     /     2S      /
                              2NT    /     3H      /
                              4H     /      ?

    对手2陷入了沉思。你想，石淼拿着HAQ等在那个叫了2嘴H的后面，他多么希望对手继续叫下去。我们人性的弱点，敌人的不幸就是我们的盛宴：外表一脸书卷气的石淼在生活中是那么纯真，他同时搞了3个对象，但每一个连手都舍不得摸，他说：别毁了人家，等选准了再消灭不迟：

    可在牌桌上，他那双乱转的小眼躲在眼镜后面，满脑子都是阴家培的招术，你看，他又要狂赚一副了，对手2终于拿出了4NT，对手1表示1个关健张，最后6H成为最终定约：石淼心里乐呀，但他可是位又有风度又谨小慎微的牌手，面无表情的不叫。该他首功了，推推眼镜故作沉思状。其实他脑子里哪有思想啊。抽出一张小D轻轻的放下，他看见了明手牌，也看见了晴空中的乌云：

                            S  AQx
                            H  K8xx
                            D  Q10
                            C  AKxx

    HK赫然亮在桌上，一瞬间，石淼的心凉到了底。

    有个累神在叹息：桥海茫茫。牌如人生：

    希望永远伴随着失望。

    这可恨的桥牌真的值得我们付出青舂的代价，去为之献身吗?无论我们怎样的努力奋斗，那些封闭的牌圈，一年又一年的兄弟轮回，让多少才华空付流水。但坚强的石淼是我们的榜样，他横空出世就要惊天动地，这世间的不如意才是他成长的阶石： 

    此刻，在石淼的脸上，绝对看不出他刚经历了片刻的大喜大悲，天赐一副小白脸，痛苦的煞白掩护的很好，毕竟还有一线希望，你看庄家想了半天上DQ，同伴大K斯特，假如同伴是DKJ呢?

    庄家A得，马上一张小H，不能多想，要快，石淼当然知道该怎么做，只要有一线生机，就决不投降。跟HQ：屏住呼吸，等待奇迹。

    现在又轮到庄家去想了，四手牌如下：

                           S  AQx
                           H  K8xx
                           D  Q10
                           C  AKxx
              S  xxxx                  S  Jxx
              H  AQ                    H  J
              D  xxx                   D  Kxxxxx
              C  xxxx        庄家      C  xxx
                           S  Kxx
                           H  109xxxx
                           D  AJ
                           C  QJ

    庄家很鬼，他先假装在D上长考，又特意出一张小H来投石问路，但外面的黑夜静悄悄，他狼狠瞟了石淼一眼，咬着后槽牙明手放小企图放下HA成为统治命运的煞神。

    于是一只神鸟飞向天空，今天是她诞生的日子：

    她的传奇也由此开篇。
