舒服系列

作者：舒服斯基(_wushangjie)

第2篇：男人轻易不流泪


    用同生共死比喻深厚的友情，文学中到处都是，却很难在现实的人群中看到。我比较有眼福，在我们院里，真有一对哥们，相差大约3小时来到人间(不是一个娘)。我们权且称他俩为大点小点，哥俩一块摸爬滚打的长大，两人好的整日形影不离，甚至娶老婆都约在了同一天。婚后两人的遭遇有些不同，大点的女人漂亮能干，俩人相亲相爱小日子滋润。小点呢没那么幸运，媳妇虽长的更美，但脾气大，又不会家务，俩人那个热闹啊，是院里著名的吵架专业户，我们都认为他们长不了。但一晃10多年过去了，情况照旧，大点这两口还是那样恩恩爱爱，小点这一对有永远打不完的架，但也没分开的意思。哥俩的媳妇虽不怎么来往，可在附近的小洒馆里却常看到那哥俩的身影。。。。。。

    就这样，平凡的岁月一日又一日的流淌，只不过累神改成了舒服斯基，可他的系列仍然是换汤没换药的毒害我们的眼睛，等着吧，早晚祸从天降。你看，就在去年的同一日，那大点在广西被人一刀致命，小点则在广州车毁人亡。实现了同生共死，却没有影视中戏剧的悲壮，只让我们感到恐惧和宿命。更让人意外的是，那夫妻恩爱的大点之媳服丧没过一月就扔了孩子重穿新衣，而在我们眼中刁蛮的小点女人则至今还在娘家教子孝婆，真让一帮平日里闲话连连的老太太们呆若木鸡，也让我等俗人大跌眼镜！

    把桥牌中的搭档关系类比夫妻相处非常恰当，不算逮谁是谁的票友，在相对固定的伙伴中，一般都是一起成名。他们大多数给人的印象是关系良好，就象大点夫妇，也有象小点之家的，俩人一上桌从第一副会争到最后一副，打完牌在外面更是红着眼互瞪，看那架势，边上有块扳砖会有一位被开瓢的。干万别替他们担心，一有比赛俩个人还是面对面的坐着，大眼对大眼，来吧，朋友！

    想当年，我还在东拼西凑的业余队里瞎混，我们队里有两个老大哥叫冯京。康宁，早先在北京业余圈中也是有头有脸的人物，俩人爱吵架也众所周知，见怪不怪，我记得有一次，康宁开叫一阶高花，冯京加一副，康宁邀局，冯京不去，康宁想了想还是干拨一局，被人咔嚓一刀，输了5颗。这下冯京急了：你有病啊，我不去你还叫，先前你邀我干吗，你直接去死不就得了。

    康宁自知理亏，但不能服：你说什么废话，上次你邀我，我没去，你不也楞上来着！

    冯京话更冲：我会摸呀，我有命啊，你面像这么苦，别学我呀！

       hehehehehehe

    老夫我也是有名的大嗓门，骂同伴是我桥牌生涯的一部分，当然得这样，要不怎么让人知道这个队里谁是牛货谁是瓜，得冠军靠谁?或着凉了赖谁?所以岁月只让人增添白发，那与生俱来的性格冲动只有在写完了本文才有所顿悟！

    光阴似箭，已经好多年没打业余比赛了，那天闲的去玩了把八喜绿茵俱乐部搞的双人赛，除幸会了12岁阳光少年，也碰到几位当年游戏的老桥友，赛后喝酒时问到冯京康宁两位大哥，他们讲已经有小两年看不见了，大概是这一对吵包子终于闹翻，离牌各走其路了吧。。。。。。

    这一天，我那平日里勤俭持家，脾气温柔的熄妇突然暴跳如雷，把一张电话收据摔在我脸上，我当然明白是网费太高，赶紧道谦不迭，但还是被飞起一脚，可怜我这在外经常对桥伴吹胡子瞪眼在家则老实如猫的汉子只好流浪街头，也遭遇了本篇略带伤感的结局。

    我到附近一家新开的网吧去消磨时光，一进门，就看见了一位身材魁梧的大汉，面貌如此熟悉，竞是冯京！原来他是这家网吧的老板，老友相见，少不得一通客套，他不让我上网，非拉我去喝小酒，我肚子饥饿，正中下怀，席间问起康宁，冯京叹息连声：你那位老哥哪都挺好，就是命苦，刚发了点财，却突然得了脑瘫，已经在医院躺了一年多了，哎！

    原来如彼，我们又共同感怀了一阵人生无常后，我问：那你没事不玩牌了?

    冯京道：也和别人打过几次，配合不好，没意思，也就不想玩了。噢，对对，你现在混成大师了，老哥我今天考你一副牌，看你是否真的牛x！

    他拿出笔，在餐巾纸上写了如下一手牌：

                   s  ax
                   h  akxx2
                   d  xxx
                   c  qxx

      假设你是西且首开叫，局况是己方单方无局：

         w         n         e        s
         1h        x         /       1s
          /        2s       全不叫

    冯京问：对付这很一般的2s定约，你首功什么?

    我稍一琢磨，疑惑的看着他：这牌有何好想，拨h大插呀，大师和业余没什么区别，没特殊理由，有ak就先摘一个看看，假如错了，就如同年轻人的过失，上帝都会原谅的！

     冯京嘿嘿一乐：好，就先拨h插，这是实战中的牌，没什么邪的，明手牌是这样：

                   s  kq10x
                   h  qxx
                   d  aq
                   c  k10xx

    同伴跟h3，你得墩，怎样继续?

    一看明手，这是如此简单，我甚至连一秒钟都没停顿，我讲：同伴出3，我手中有2，他不是3张就是单张，假如他是3张，在单无局且他大概还有点实力的情况下，会在敌方加倍1h后叫2h，他pass，这几乎标明是单张h。所以我继续出k，让同伴将吃第3轮h，下次sa进手，再出h，希望同伴有sj提升或能消灭暗手的h赢墩！这样我方已拿4或5墩，怎么宕就听天由命吧！

    假如在实战中，我一定会这么防，但我知道这个答案一定是错的，否则他还考个屁。笑咪咪的望着他等他自揭迷底。

    果然，只看冯京长吁一声，分明在说：大师=大湿。他写下了全手牌：

                    s  kq10x
                    h  qxx
                    d  aq
                    c  k10xx
            s ax       n      s xxxx
            h akxx2           h 103
            d xxx  w       e  d xxxx
            c qxx      s      c aj9
                    s  j9x
                    h  jxx
                    d  kj10x
                    c  xxx

    冯京道：实战中我拿的是东家的牌。没错，我不会有3张h，否则在这种局况下我就是白皮也要加一付！假如我在ha上跟10，同伴一定会让我将吃第三轮，可我是多么希望他穿阿梅呀！于是我跟了一张3！

    多少次，我们在牌桌上，清楚的发现了敌人的命门，可你的同伴还在苦苦的思索，你明白他回牌有难度，你急啊，假如他真的亮出了你的期待。。。。。。

    冯京小饮一口酒，继续道：我的对面是康宁，你知道吗?他出的是什么?他出的是阿梅娘娘cq！非常轻松的将敌人宕2！你看好，假如我先将吃h，他就缺一次桥穿2次阿梅。敌人就会用d垫贼。这副牌真它桌都只下1，我问了许多人，防守思路都和你差别不大，看来，我的信号只有康宁能懂！可惜他大概在也打不了牌了！

    话到这，这位1米89的大汉语中竟有些哽咽，我在牌场苦战半生，当然能体会到当冯京看见cq时欣喜的内心，在那一刻，他知道他没选错搭档，虽然吵架仍然是必须的。

    其实我们只和自己佩服的或彼此熟悉且心心相印的人争论，像小点夫妇，象冯京康宁。。。。。

    这就是生活！
